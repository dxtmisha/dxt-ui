This library provides a high-level TypeScript abstraction for the Figma Plugin API, specifically optimized for node manipulation, data persistence, and AI-driven content analysis. Its core purpose is to facilitate the extraction of structured data—including CSS styles, node hierarchies, and text content—from Figma frames while providing managed wrappers for both node-specific (PluginData) and global (clientStorage) persistence with TTL-based caching. An AI assistant should study this library when implementing workflows that require automated text localization, selection state synchronization between plugin backend and UI, or programmatic asset export in JPG and JSON formats. The library integrates directly with Figma’s runtime environment and depends on external AI abstracts to process design context (via screenshots and text lists) into structured key-value maps. Key components include FigmaPluginMessenger for bidirectional UI communication and FigmaFrame for recursive node filtering and coordinate-based navigation.
