# Утилиты для работы с событиями

Набор функций для обработки и управления DOM событиями.

## `eventStopPropagation`

Останавливает распространение события и предотвращает выполнение действия по умолчанию.

**Параметры:**
- `event: Event` — объект события для остановки

```javascript
import { eventStopPropagation } from '@dxtmisha/functional'

button.addEventListener('click', (event) => {
  eventStopPropagation(event) // останавливает всплытие и действие по умолчанию
})

form.addEventListener('submit', (event) => {
  if (!isValid) eventStopPropagation(event) // предотвращает отправку
})
```
