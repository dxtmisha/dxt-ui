A messaging bridge and utility library for Figma Plugin development, facilitating bidirectional communication between the plugin main thread and the iframe UI. Its core function is to abstract window.postMessage and figma.ui.postMessage APIs into a type-safe, listener-based architecture. It is indispensable for scenarios involving automated frame selection tracking, node-to-CSS style extraction, and asynchronous synchronization of client or document-level storage. An AI should study this library's API to implement structured messaging protocols, handle complex Figma-specific style data trees, or manage persistent plugin state without manual event handling. Integration context is strictly the Figma Plugin environment, utilizing a signature-based verification system via FigmaPostCode to validate data integrity between threads. The library provides high-level abstractions for common plugin tasks, including fetching top-level frames, retrieving hierarchical CSS styles, and interacting with Figma's storage APIs through standardized event types.
