# `getRandomText`

Генерирует случайный текст из указанного количества слов.

**Параметры:**
- `min: number` — минимальное количество слов
- `max: number` — максимальное количество слов
- `symbol: string` — символ для замены букв (по умолчанию 'x')
- `lengthMin: number` — минимальная длина слова (по умолчанию 3)
- `lengthMax: number` — максимальная длина слова (по умолчанию 8)

```javascript
import { getRandomText } from '@dxtmisha/functional'

getRandomText(1, 3) // 'xxx xxxx xx'
getRandomText(2, 2, 'o', 4, 6) // 'oooo oooooo'
getRandomText(1, 1, '*', 5, 5) // '*****'
```
