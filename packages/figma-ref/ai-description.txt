This Vue 3 composable library provides reactive abstractions for the Figma Plugin API. Its core purpose is to bridge the Figma plugin backend and the UI layer by providing reactive state management for Figma's storage systems, selection state, and node styles. It is indispensable for developers building Figma plugin interfaces with Vue who need to synchronize the UI with canvas selections, persist data via clientStorage or document-level storage, and manage top-level frame metadata. An AI should study this API to implement reactive data persistence (useFigmaStorage, useFigmaClientStorage), observe or manipulate canvas frame selections (useFigmaFrameSelection, useFigmaUiSelected), and handle style-related messaging (useFigmaFrameStyles). The library integrates Vue's reactivity system (ShallowRef, ComputedRef) with Figma-specific data types and requires the @dxtmisha/figma package for type definitions. It abstracts the asynchronous nature of Figma's postMessage and storage APIs into synchronous-like reactive references and loading states.
