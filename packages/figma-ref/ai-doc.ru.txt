РЕАКТИВНЫЕ ОБЕРТКИ ДЛЯ FIGMA (VUE 3):
Пакет предоставляет Vue-композиции (composables) для удобной и реактивной работы с данными Figma в интерфейсе плагина.

1. ПРЕДНАЗНАЧЕНИЕ: Превращение асинхронных запросов к Figma в реактивные переменные Vue. Это позволяет интерфейсу мгновенно обновляться при изменении выделения (selection) или данных в Figma.
2. ЗАВИСИМОСТИ: Библиотека базируется на `@dxtmisha/figma` для обмена данными и использует реактивность Vue 3.
3. ОСНОВНЫЕ COMPOSABLES:
   - `useFigmaFrameSelection`: Реактивное отслеживание выделенных объектов.
   - `useFigmaStorage`, `useFigmaClientStorage`: Работа с постоянным хранилищем плагина.
   - `useFigmaUiFrames`: Список доступных фреймов для отображения в UI.

Используй этот пакет при разработке Vue-интерфейсов для плагинов Figma, чтобы обеспечить бесшовную синхронизацию состояния между Figma и UI.
