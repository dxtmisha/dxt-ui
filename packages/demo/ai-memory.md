## Component Demo Layout & Styling
1. **Minimal Styling Constraint**: Always use minimal custom styling for demo pages. Best practice is to have no `<style>` block at all.
2. **Direct Layout structure**: Place component variants directly inside layout containers (like `<DemoFlex>` or `<D1Grid>`) under `<D1Group>`. Avoid wrapping each variant in unnecessary nested divs or custom sub-labels.
