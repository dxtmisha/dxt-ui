# `debounce`

Позволяет ограничить частоту вызова функции. Если функция вызывается несколько раз подряд с коротким интервалом, выполнение произойдет только один раз после истечения указанной задержки. Это полезно для оптимизации производительности при обработке событий, которые срабатывают слишком часто, таких как изменение размера окна или ввод текста.

**Parameters:**
- `func: T` — функция, выполнение которой нужно ограничить
- `wait: number` — время задержки в миллисекундах

**Returns:**
Новая функция, которая при вызове сбрасывает предыдущий таймер и устанавливает новый, обеспечивая выполнение исходной функции только после паузы.

```javascript
import { debounce } from '@package'

const handleSearch = (query) => {
  console.log('Поиск:', query);
};

const debouncedSearch = debounce(handleSearch, 300);

// Если вызвать несколько раз подряд, сработает только последний вызов через 300 мс
debouncedSearch('a');
debouncedSearch('ab');
debouncedSearch('abc'); 
```
