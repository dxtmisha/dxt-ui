# `formatDate`

Функция `formatDate` предназначена для преобразования объекта `Date` в человекочитаемую строку с использованием стандарта `Intl.DateTimeFormat`. Она обеспечивает гибкую локализацию вывода, позволяя отображать дату и время в формате, соответствующем культурным особенностям выбранного региона.

**Параметры:**
- `date: Date` — объект даты, который необходимо отформатировать
- `locale: string` — локаль форматирования (по умолчанию 'en-US')

**Возвращает:**
Отформатированная строка даты.

```javascript
import { formatDate } from '@package'

// Пример форматирования даты для русской локали
const date = new Date(2023, 10, 15, 14, 30)
formatDate(date, 'ru-RU')
// Результат: "15 ноября 2023 г., 14:30"

// Пример с локалью по умолчанию (en-US)
formatDate(date)
// Результат: "November 15, 2023, 02:30 PM"
```
