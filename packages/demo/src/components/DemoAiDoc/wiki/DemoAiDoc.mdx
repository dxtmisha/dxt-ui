Компонент `DemoAiDoc` представляет собой контейнер (аккордеон), предназначенный для оформления документации. Он позволяет группировать информацию под заголовком, добавлять описания и управлять видимостью контента.

## Key Features

- **Сворачиваемый контент** — возможность экономить место на странице, скрывая детали.
- **Гибкая настройка через слоты** — кастомизация заголовка, основного содержимого и подвала.
- **Синхронизация состояния** — реактивное управление открытием через пропсы и события.
- **Интуитивное управление** — переключение состояния кликом по шапке с визуальной индикацией.

## Typical Use Cases

- Оформление примеров использования компонентов.
- Группировка свойств и методов в документации API.
- Создание интерактивных FAQ-секций.
- Размещение второстепенной справочной информации.

## Basic Usage

```html
<script setup>
import { DemoAiDoc } from '@dxtmisha/demo'
</script>

<template>
  <DemoAiDoc 
    title="Базовый компонент" 
    description="Пример использования компонента с заголовком и описанием."
  >
    <div class="content">
      Здесь находится основной контент документации.
    </div>
  </DemoAiDoc>
</template>
```

## Состояние и контент

Эта группа свойств управляет текстовым наполнением и начальным состоянием отображения блока.

- `title` — текст заголовока, отображаемый в шапке блока.
- `description` — дополнительный текст, отображаемый над основным контентом.
- `isOpen` — определяет, развернут ли блок при инициализации.

```html
<template>
  <DemoAiDoc 
    title="Настройки интерфейса"
    description="Этот блок открыт по умолчанию и содержит описание."
    :isOpen="true"
  />
</template>
```

## Events

### `click`

Срабатывает при клике на любую область компонента.

**Parameters:**
- `event: MouseEvent` — стандартное событие мыши.

### `toggle`

Срабатывает при переключении состояния развернутости блока (клик по шапке).

**Parameters:**
- `isOpen: boolean` — новое состояние видимости (true — открыто, false — закрыто).

```html
<script setup>
const onToggle = (state) => {
  console.log('Блок теперь:', state ? 'открыт' : 'закрыт')
}
</script>

<template>
  <DemoAiDoc title="Логируемый блок" @toggle="onToggle" />
</template>
```

## Slots

### `header`

Позволяет полностью заменить стандартный текст заголовка в шапке.

**Parameters (Scoped Props):**
- `title: string` — значение пропса title.

```html
<template>
  <DemoAiDoc title="Важный раздел">
    <template #header="{ title }">
      <div style="display: flex; align-items: center; gap: 8px;">
        <span class="icon">ℹ️</span>
        <strong>{{ title }}</strong>
      </div>
    </template>
  </DemoAiDoc>
</template>
```

### `default`

Основной слот для размещения содержимого, которое будет скрываться и отображаться при переключении.

### `footer`

Слот для размещения контента в нижней части блока. Секция футера отображается только если этот слот заполнен.

```html
<template>
  <DemoAiDoc title="Блок с действиями">
    <p>Контент блока...</p>
    <template #footer>
      <div class="actions">
        <button>Скопировать код</button>
      </div>
    </template>
  </DemoAiDoc>
</template>
```
