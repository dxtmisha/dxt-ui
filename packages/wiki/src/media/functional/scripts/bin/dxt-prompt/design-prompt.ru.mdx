# dxt-prompt

Команда `dxt-prompt` генерирует консолидированный файл системных инструкций ИИ `ai-prompt.md` в корне проекта. Этот файл объединяет все руководства по коду, стандарты BEM-классов, правила SCSS и архитектуру Vue-компонентов для тонкой настройки ИИ-ассистентов.

## Ключевые возможности

- **Централизованные руководства**: Объединяет все глобальные и специфические для компонентов инструкции в один файл.
- **Автоматическое сканирование**: Рекурсивно сканирует настроенные директории модулей для сбора локальных правил и определений.
- **Стандартизация правил**: Склеивает шаблоны промптов для Vue и общих принципов разработки, гарантируя строгое следование стандартам проекта.

## Как это работает?

1. **Сканирование папок**: Скрипт обходит директории, указанные в конфигурации проекта (например, модули в `src/`).
2. **Сбор промптов**: Извлекает локальные файлы правил и системные инструкции.
3. **Сборка контекста**: Объединяет глобальные требования, правила разработки Vue и другие высокоприоритетные инструкции в структурированный шаблон.
4. **Сохранение файла**: Записывает итоговый результат в файл `ai-prompt.md` в корне репозитория.

## Использование

Запустите генератор промптов через `npx`:
```bash
npx dxt-prompt
```
Команда автоматически обновит или создаст файл `ai-prompt.md`.
