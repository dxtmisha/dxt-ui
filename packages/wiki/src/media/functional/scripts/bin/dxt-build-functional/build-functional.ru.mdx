# dxt-build-functional

Команда `dxt-build-functional` автоматически собирает и повторно экспортирует все реактивные утилиты, классы и композитные функции из пакета `@dxtmisha/functional` в единый файл точки входа `functional.ts`.

## Ключевые возможности

- **Автоматическое создание точки входа**: Динамически формирует единый список экспорта для всех реактивных API.
- **Алфавитная сортировка**: Сортирует экспортируемые сущности по алфавиту для чистоты кода и предотвращения конфликтов слияния.
- **Типизированные экспорты**: Автоматически перенаправляет TypeScript-типы (`export type *`) из базового пакета.

## Как это работает?

1. **Импорт модуля**: Скрипт загружает базовый модуль `@dxtmisha/functional`.
2. **Сбор ключей**: Динамически извлекает все экспортируемые ключи из объекта модуля.
3. **Сортировка и генерация**: Сортирует ключи по алфавиту и генерирует строки экспорта для каждого члена.
4. **Запись файла**: Объединяет строки и сохраняет готовый файл под именем `functional.ts` в настроенную директорию библиотеки.

## Использование

Запустите утилиту сборки через `npx` или скрипты npm:
```bash
npx dxt-build-functional
```
