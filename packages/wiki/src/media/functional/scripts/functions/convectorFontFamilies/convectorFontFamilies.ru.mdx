# `convectorFontFamilies`

Функция преобразует необработанные строки семейств шрифтов в стандартные списки шрифтов CSS, добавляя системные резервные шрифты по умолчанию.

**Параметры:**
- `item: PropertyItemInput` — элемент свойства токена дизайна для преобразования.

**Возвращает:**
`void` — функция напрямую изменяет свойство `value` переданного объекта `item`.

## Принцип работы

Функция проверяет, является ли `item.value` строкой и не содержит ли она символов интерполяции в фигурных скобках `{` или `}`. Если оба условия выполняются, она оборачивает значение в одинарные кавычки и добавляет `, sans-serif` для обеспечения стандартного резервного поведения браузера.

## Пример

```typescript
import { convectorFontFamilies } from '@dxtmisha/scripts'

const item = { value: 'Inter' }
convectorFontFamilies(item)

console.log(item.value) // "'Inter', sans-serif"
```
