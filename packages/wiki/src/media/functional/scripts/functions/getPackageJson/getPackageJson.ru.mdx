# `getPackageJson`

Функция возвращает содержимое файла `package.json`.

**Возвращает:**
`Record<string, any> | undefined` — содержимое файла `package.json` или `undefined`, если файл не удалось прочитать.

## Как это работает

Функция использует метод `PropertiesFile.readFile` для загрузки содержимого файла `package.json`, указанного в глобальной конфигурации.

## Пример

```typescript
import { getPackageJson } from '@dxtmisha/scripts'

const packageJson = getPackageJson()
if (packageJson) {
  console.log(`Package Name: ${packageJson.name}`)
}
```
