# `hasNativeDirname`

Hàm kiểm tra xem biến toàn cục `__dirname` có sẵn trong môi trường hiện tại hay không.

**Trả về:**
`boolean` — `true` nếu `__dirname` được xác định, ngược lại là `false`.

## Cơ chế hoạt động

Hàm sử dụng `typeof __dirname !== 'undefined'` để xác định xem môi trường có hỗ trợ việc xác định tên thư mục gốc hay không (thường là môi trường CommonJS).

## Ví dụ

```typescript
import { hasNativeDirname } from '@dxtmisha/scripts'

if (hasNativeDirname()) {
  console.log('Môi trường hỗ trợ __dirname gốc')
}
```
