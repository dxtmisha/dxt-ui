# `toPathStandardSep`

Функция преобразует прямые слеши в строке пути в стандартный разделитель пути текущей операционной системы.

**Параметры:**
- `path: string` — входная строка пути.

**Возвращает:**
`string` — путь со стандартными разделителями.

## Как это работает

Функция использует регулярное выражение для поиска всех прямых слешей (`/`) во входной строке и заменяет их платформенно-зависимым разделителем (`path.sep`).

## Пример

```typescript
import { toPathStandardSep } from '@dxtmisha/scripts'

// В Windows:
toPathStandardSep('src/components/button') // 'src\\components\\button'

// В Unix/macOS:
toPathStandardSep('src/components/button') // 'src/components/button'
```
