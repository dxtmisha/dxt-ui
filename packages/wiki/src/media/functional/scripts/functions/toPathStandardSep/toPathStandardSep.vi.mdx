# `toPathStandardSep`

Hàm chuyển đổi các dấu gạch chéo xuôi trong một chuỗi đường dẫn thành dấu phân cách đường dẫn tiêu chuẩn của hệ điều hành hiện tại.

**Tham số:**
- `path: string` — chuỗi đường dẫn đầu vào.

**Trả về:**
`string` — đường dẫn với các dấu phân cách tiêu chuẩn.

## Cơ chế hoạt động

Hàm sử dụng biểu thức chính quy để tìm tất cả các dấu gạch chéo xuôi (`/`) trong chuỗi đầu vào và thay thế chúng bằng dấu phân cách dành riêng cho nền tảng (`path.sep`).

## Ví dụ

```typescript
import { toPathStandardSep } from '@dxtmisha/scripts'

// Trên Windows:
toPathStandardSep('src/components/button') // 'src\\components\\button'

// Trên Unix:
toPathStandardSep('src/components/button') // 'src/components/button'
```
