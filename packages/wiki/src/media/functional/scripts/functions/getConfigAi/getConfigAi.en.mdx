# `getConfigAi`

The function returns the AI configuration, including the API key and model.

**Returns:**
`[string, string]` — a tuple with the API key and the model name.

## How it works

The function retrieves the configuration values from the `PropertiesConfig` class, which manages project-level settings.

## Example

```typescript
import { getConfigAi } from '@dxtmisha/scripts'

const [apiKey, model] = getConfigAi()
console.log(`API Key: ${apiKey}, Model: ${model}`)
```
