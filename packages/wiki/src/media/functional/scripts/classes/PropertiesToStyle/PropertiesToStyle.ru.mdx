# Класс PropertiesToStyle

Класс `PropertiesToStyle` обрабатывает токены дизайна, поддерживающие дополнительные пользовательские значения, оборачивая их в выделенные узлы структур стилей. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Синтез пользовательских селекторов** — Создает переопределения пользовательских состояний под единым ключом `custom` внутри карт значений свойств.
- **Динамическое преобразование свойств** — Переписывает свойства с директивой `style` для их поведения в качестве переменных состояний.
- **Кэширование результатов** — Сохраняет обработанные переопределения стилей в файл кэша `016-style`.

## Инициализация

```typescript
import { PropertiesToStyle, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToStyle(items)
transformer.to()
```

## Методы

### Шаги жизненного цикла

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для обхода свойств, проверки маркеров `style` и внедрения записей стилей `custom`.
