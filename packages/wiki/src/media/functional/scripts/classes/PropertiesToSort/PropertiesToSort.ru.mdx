# Класс PropertiesToSort

Класс `PropertiesToSort` упорядочивает и сортирует элементы в дереве токенов дизайна в соответствии с предопределенными правилами, категориями, переменными и именами. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Сортировка на основе правил** — Сортирует свойства дизайна с использованием правил порядка, определенных в `sortList` на основе переменных и категорий.
- **Иерархическое упорядочивание** — Рекурсивно сортирует вложенные значения и дочерние ветки внутри контейнерных узлов.
- **Синтез индексов** — Назначает вычисленный числовой приоритет каждому токену для выполнения сортировки.
- **Кэширование результатов** — Сохраняет отсортированное дерево свойств в файл кэша `048-sort`.

## Инициализация

```typescript
import { PropertiesToSort, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToSort(items)
transformer.to()
```

## Методы

### Операции сортировки

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для запуска сортировки структуры токенов дизайна.
- `read(properties?: PropertyItemsItem): PropertyList` — Рекурсивно обходит свойства и сортирует их дочерние массивы и значения свойств.
- `getCategoryName(item: PropertyItem): string` — Возвращает форматированную строку категории.
- `getKeys(item: PropertyItem): Item<number>` — Возвращает строку индекса сортировки и вычисляет числовое значение веса порядка на основе правил `sortList`.
- `join(data: PropertiesSortList): PropertyList` — Выполняет окончательную сортировку массива по весу/порядку и сопоставляет элементы обратно в отсортированный объект.
