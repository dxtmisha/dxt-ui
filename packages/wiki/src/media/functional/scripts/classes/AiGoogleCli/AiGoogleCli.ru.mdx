# Класс AiGoogleCli

Настроенная реализация Google AI через CLI, расширяющая [AiGoogleCliLite](?path=/docs/scripts-classes-aigoogleclilite--docs). Этот класс автоматически получает необходимые учетные данные из конфигурации проекта.

## Назначение

Класс упрощает использование CLI-интерфейса Gemini, автоматически подключая API-ключ и параметры модели через функцию `getConfigAi()`. Он предназначен для выполнения скриптов генерации в консольном окружении.

## Конструктор

При инициализации класс автоматически:
1. Вызывает `getConfigAi()` для получения актуального ключа.
2. Инициализирует обертку `AiGoogleCliLite` с полученными данными.

## Наследование

Класс наследует механизмы выполнения команд через оболочку из [AiGoogleCliLite](?path=/docs/scripts-classes-aigoogleclilite--docs).

## Пример использования

```typescript
import { AiGoogleCli } from '@dxtmisha/scripts'

// Конфигурация подтягивается автоматически
const ai = new AiGoogleCli();
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Напиши документацию для этого метода');
```
