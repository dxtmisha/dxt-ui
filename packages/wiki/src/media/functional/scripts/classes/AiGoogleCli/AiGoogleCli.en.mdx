# AiGoogleCli Class

A configured implementation of Google AI via CLI that extends [AiGoogleCliLite](?path=/docs/scripts-classes-aigoogleclilite--docs). This class automatically retrieves the necessary credentials from the project configuration.

## Purpose

The class simplifies the usage of the Gemini CLI interface by automatically connecting the API key and model parameters via the `getConfigAi()` function. It is intended for running generation scripts within a console environment.

## Constructor

Upon initialization, the class automatically:
1. Calls `getConfigAi()` to retrieve the current API key.
2. Initializes the `AiGoogleCliLite` wrapper with the retrieved data.

## Inheritance

The class inherits shell command execution mechanisms from [AiGoogleCliLite](?path=/docs/scripts-classes-aigoogleclilite--docs).

## Usage Example

```typescript
import { AiGoogleCli } from '@dxtmisha/scripts'

// Configuration is pulled automatically
const ai = new AiGoogleCli();
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Write documentation for this method');
```
