# Класс PropertiesToSub

Класс `PropertiesToSub` рекурсивно разрешает вложенные выражения интерполяции в фигурных скобках (такие как `calc({ref.width} * 2)`) внутри значений токенов. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Разрешение вложенных значений** — Рекурсивно разрешает вложенные фигурные скобки, содержащие ссылки на переменные, внутри более крупных строковых значений.
- **Защита от циклов** — Ограничивает операции поиска до 24 итераций во избежание бесконечных циклов в циклических определениях.
- **Очистка фигурных скобок** — Автоматически удаляет внешние фигурные скобки, если для токена установлено значение `fullValue` равное true.
- **Кэширование результатов** — Сохраняет разрешенные подзначения в файл кэша `006-sub`.

## Инициализация

```typescript
import { PropertiesToSub, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToSub(items)
transformer.to()
```

## Методы

### Операции разрешения

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для поиска и разрешения всех свойств с подзначениями.
- `is(value: PropertyItem['value']): boolean` — Возвращает true, если значение токена содержит вложенный шаблон фигурных скобок.
- `getValue(design: string, component: string, value: string): string` — Итеративно разрешает маркеры интерполяции фигурных скобок, используя глобальный реестр элементов свойств.
- `read({ design, component, item }: PropertyItemsItem): void` — Разрешает подзначения для одного элемента свойства и удаляет фигурные скобки, если установлен флаг `fullValue`.
