# DesignWikiStorm
Класс `DesignWikiStorm` координирует генерацию файла `web-types.json`, который включает расширенные функции IDE, такие как автодополнение и всплывающие подсказки с документацией для компонентов в редакторах JetBrains (например, WebStorm). Он обходит библиотеку компонентов, собирает определения из соответствующих директорий и структурирует их в соответствии со схемой JSON Web-Types.

### Основные методы
- `make()`: Основная точка входа, которая сканирует директории дизайн-системы, агрегирует данные компонентов с помощью `DesignWikiStormItem` и записывает финальный файл `web-types.json`.

### Методы конфигурации
- `getList()`: Сканирует указанную директорию на наличие папок компонентов, содержащих файл `index.ts`.
- `getPath()`: Возвращает абсолютный путь, по которому будет сохранен файл `web-types.json`.
