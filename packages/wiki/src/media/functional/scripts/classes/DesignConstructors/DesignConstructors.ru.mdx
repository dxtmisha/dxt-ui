# DesignConstructors
Класс `DesignConstructors` управляет общим жизненным циклом и координацией конструкторов дизайна. Он отвечает за идентификацию компонентов, требующих генерации на основе конструкторов, координацию создания их соответствующих файлов и обновление основного файла стилей, чтобы включить все сгенерированные стили компонентов.

### Основные методы
- `make()`: Основная точка входа, которая выполняет обход библиотеки компонентов, инициализирует `DesignConstructor` для каждого компонента и запускает полный процесс генерации.
- `makeStyles()`: Обновляет основной файл стилей дизайн-системы, агрегируя импорты всех отдельных стилей компонентов.

### Методы конфигурации
- `getList()`: Сканирует директорию дизайн-системы для поиска компонентов, содержащих файл `constructors.json`.
- `getPath()`: Возвращает абсолютный путь к корневой директории дизайн-системы.
- `getMainStylesPath()`: Возвращает путь к основному файлу стилей SCSS.
