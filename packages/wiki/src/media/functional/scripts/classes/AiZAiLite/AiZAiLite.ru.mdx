# Класс AiZAiLite

Класс `AiZAiLite` представляет собой легковесную реализацию `AiOpenAiLite`, сконфигурированную для Zhipu AI (Z.ai). Он переопределяет метод инициализации клиента для перенаправления всех запросов на OpenAI-совместимый эндпоинт Z.ai.

## Ключевые особенности

- **Совместимость с OpenAI** — наследует все возможности `AiOpenAiLite`, включая генерацию текста, буферы промптов и мультимодальные структуры.
- **Выделенный эндпоинт** — автоматически настраивает клиент OpenAI с базовым URL-адресом Z.ai (`https://api.z.ai/api/paas/v4`).

## Инициализация

`AiZAiLite` требует ручной передачи учетных данных через конструктор.

**Параметры конструктора:**
- `key: string` — API-ключ для сервиса Z.ai.
- `model: string` — начальный идентификатор модели (например, 'glm-4').

```typescript
import { AiZAiLite } from '@dxtmisha/scripts'

const ai = new AiZAiLite(process.env.ZAI_API_KEY, 'glm-4')
```

## Методы

### Внутренние хуки реализации

Этот класс переопределяет хук инициализации `AiOpenAiLite`:

- `init(): void` — Создает экземпляр клиента `OpenAI` с базовым URL-адресом Z.ai и предоставленным ключом.
