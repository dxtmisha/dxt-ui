# Lớp AiZAiLite

Lớp `AiZAiLite` là một triển khai gọn nhẹ của `AiOpenAiLite` được định cấu hình riêng cho Zhipu AI (Z.ai). Nó ghi đè phương thức khởi tạo client để điều hướng tất cả các yêu cầu dịch vụ đến endpoint tương thích với OpenAI của Z.ai.

## Tính năng chính

- **Khả năng Tương thích OpenAI** — kế thừa tất cả các khả năng của `AiOpenAiLite` bao gồm tạo văn bản, bộ đệm lời nhắc và cấu trúc đa phương thức.
- **Endpoint Chuyên dụng** — tự động định cấu hình client OpenAI với URL cơ sở của Z.ai (`https://api.z.ai/api/paas/v4`).

## Khởi tạo

`AiZAiLite` yêu cầu truyền thông tin xác thực thủ công thông qua constructor.

**Tham số khởi tạo:**
- `key: string` — API key cho dịch vụ Z.ai.
- `model: string` — Định danh mô hình ban đầu (ví dụ: 'glm-4').

```typescript
import { AiZAiLite } from '@dxtmisha/scripts'

const ai = new AiZAiLite(process.env.ZAI_API_KEY, 'glm-4')
```

## Phương thức

### Các Hook Triển khai Nội bộ

Lớp này ghi đè hook khởi tạo của `AiOpenAiLite`:

- `init(): void` — Khởi tạo client `OpenAI` với URL cơ sở của Z.ai và key được cung cấp.
