# Класс StylesToContainer

Класс `StylesToContainer` отвечает за создание CSS-запросов `@container` на основе свойств дизайна. Он реализует логику для обертывания вложенных правил стилей в блок container query.

## Ключевые возможности

- **Обертка Container Query** — Автоматически оборачивает созданный контент стилей в правило `@container`.
- **Динамическое разрешение запроса** — Использует значение свойства дизайна для определения условия container query.
- **Управление отступами** — Корректно обрабатывает отступы блоков и необязательные начальные пустые строки.

## Инициализация

Инициализируйте преобразователь `StylesToContainer` с помощью необходимого свойства и функции контента для вложенных правил.

**Параметры:**
- `property: PropertyItemsItem` — текущая ветка дерева свойств, которая обрабатывается.
- `space: string` — строка отступа (например, `"  "` или `"\t"`).
- `content?: () => string[]` — вызываемая функция, которая возвращает массив строк, представляющих внутренние правила стиля.
- `first?: boolean` — указывает, является ли этот элемент первым на текущем уровне (необязательно, по умолчанию: `true`).

```typescript
import { StylesToContainer } from '@dxtmisha/scripts'

// Использование в логике преобразования
const converter = new StylesToContainer(property, '  ', contentCallable)
const result = converter.make()
```

## Методы

### Управление

- `make(): string[]` — Основная точка входа. Организует преобразование и добавляет необходимый отступ к каждой строке.
