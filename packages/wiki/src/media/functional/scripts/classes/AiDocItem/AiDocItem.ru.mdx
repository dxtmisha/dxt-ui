# Класс AiDocItem

Специализированная реализация класса `AiDocItemAbstract`, предназначенная для автоматизированной генерации документации и обновления кода функциональных утилит (functions).

## Назначение

Класс расширяет базовую логику `AiDocItemAbstract`, подключая специфичные для функций шаблоны промптов и примеры демонстрационного кода. Это позволяет ИИ более точно анализировать структуру функций и генерировать документацию в едином стиле библиотеки.

## Переопределенные свойства

Класс устанавливает пути к ресурсам, используемым в процессе генерации:

- `pathDemo` — путь к файлу `demoDocEn.mdx`, который служит эталонным примером оформления документации для функций.
- `pathPrompt` — путь к файлу `aiDocFunctionPrompt.en.txt`, содержащему системные инструкции для ИИ по анализу кода функций.

## Наследование

Класс наследует все методы управления процессом из [AiDocItemAbstract](?path=/docs/scripts-classes-aidocitemabstract--docs), включая основной метод `make()`.

## Пример использования

```typescript
import { AiDocItem } from '@dxtmisha/scripts'

// Создание экземпляра для конкретной функции
const generator = new AiDocItem(sourcePath, gitItem);

// Запуск процесса обновления документации
await generator.make();
```
