# AiGoogleLite Class

An implementation of the `AiAbstract` class that uses the official `@google/genai` SDK to interact with Google AI (Gemini) models. This class provides the core logic for client initialization and executing requests to Google's cloud infrastructure.

## Key Features

- **SDK Integration** — direct integration with the Google Generative AI library.
- **Multimodal Support** — capability to send both text content and Base64-formatted images.
- **Flexible Request Structure** — converts accumulated data (text, images) into the format required by the Gemini API.

## Primary Method

### `response`

A protected method that performs the actual network request to the Google AI API. It aggregates all prepared content parts (text and images) and sends them to the specified Gemini model.

**Main workflow steps:**
1. Calling the helper methods `toImages()` and `toContents()` to format the data.
2. Sending the request via the `GoogleGenAI` client (using the `generateContent` method).
3. Safely extracting the textual result from the server response.

**Parameters:**
- `model: string` — the model identifier (e.g., `'gemini-1.5-flash'`).
- `contents: string` — the primary text content of the prompt.

**Returns:** `Promise<string>` — the generated text or an empty string if an error occurs.

## Data Processing Methods

- `init(): void` — Initializes the `GoogleGenAI` instance using the provided API key.
- `toImages(): any` — Converts the image array from `AiAbstract` into the `inlineData` structure required by the SDK.
- `toContents(): any` — Formats accumulated text blocks into an array of `{ text: ... }` objects.

## Inheritance

The class extends [AiAbstract](?path=/docs/scripts-classes-aiabstract--docs), providing a concrete implementation for working with Google AI.

## Usage Example

```typescript
import { AiGoogleLite } from '@dxtmisha/scripts'

const ai = new AiGoogleLite('YOUR_API_KEY');
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Tell me about Vue.js');
console.log(result);
```
