# Класс AiGoogleLite

Реализация абстрактного класса `AiAbstract`, использующая официальный SDK `@google/genai` для взаимодействия с моделями Google AI (Gemini). Этот класс обеспечивает базовую логику инициализации клиента и выполнения запросов к облачной инфраструктуре Google.

## Ключевые особенности

- **Интеграция с SDK** — прямая работа с библиотекой Google Generative AI.
- **Поддержка мультимодальности** — возможность передачи текстового контента и изображений в формате Base64.
- **Гибкая структура запросов** — преобразование накопленных данных (текст, изображения) в формат, требуемый API Gemini.

## Основной метод

### `response`

Защищенный метод, выполняющий непосредственный сетевой запрос к API Google AI. Он собирает все подготовленные части контента (тексты и изображения) и отправляет их указанной модели Gemini.

**Основные этапы работы:**
1. Вызов вспомогательных методов `toImages()` и `toContents()` для форматирования данных.
2. Отправка запроса через клиент `GoogleGenAI` (метод `generateContent`).
3. Безопасное извлечение текстового результата из ответа сервера.

**Параметры:**
- `model: string` — идентификатор модели (например, `'gemini-1.5-flash'`).
- `contents: string` — основное текстовое содержимое промпта.

**Возвращает:** `Promise<string>` — сгенерированный текст или пустую строку при ошибке.

## Методы обработки данных

- `init(): void` — Инициализирует инстанс `GoogleGenAI` с использованием предоставленного API-ключа.
- `toImages(): any` — Преобразует массив изображений из `AiAbstract` в структуру `inlineData`, необходимую для SDK.
- `toContents(): any` — Форматирует накопленные текстовые блоки в массив объектов `{ text: ... }`.

## Наследование

Класс расширяет [AiAbstract](?path=/docs/scripts-classes-aiabstract--docs), предоставляя конкретную реализацию для работы с Google AI.

## Пример использования

```typescript
import { AiGoogleLite } from '@dxtmisha/scripts'

const ai = new AiGoogleLite('YOUR_API_KEY');
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Расскажи о Vue.js');
console.log(result);
```
