# Класс PropertiesToMulti

Класс `PropertiesToMulti` преобразует токены дизайна, содержащие несколько пользовательских свойств или состояний, в стандартизированные структуры, где значения сгруппированы под соответствующими именами ключей. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Преобразование множественных значений** — Находит свойства с типом `property`, содержащие сложные вложенные значения, и реструктурирует их.
- **Продвижение состояний** — Разрешает базовые значения (такие как переменные или состояния) и сопоставляет их со структурами модификаторов состояний.
- **Кэширование результатов** — Сохраняет преобразованные структуры в файл кэша `012-multi`.

## Инициализация

```typescript
import { PropertiesToMulti, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToMulti(items)
transformer.to()
```

## Методы

### Операции преобразования

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для получения свойств с несколькими значениями и перезаписи их дочерних элементов.
- `getList(): PropertyItemsItem[]` — Фильтрует все активные элементы, возвращая только переменные свойств, которые обладают непустыми сложными объектами значений.
- `read(name: string, properties: PropertyList, isVar: boolean): void` — Реструктурирует дочерние значения, переводя их свойства в структуры состояний и корректируя тип выходного свойства на основе `isVar`.
