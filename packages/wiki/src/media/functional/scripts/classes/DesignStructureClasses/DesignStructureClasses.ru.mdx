# DesignStructureClasses
Класс `DesignStructureClasses` предназначен для извлечения списка специфических CSS-классов из структурных определений компонента. Он рекурсивно обходит иерархию свойств для идентификации элементов, отмеченных как подклассы, и форматирует их в стандартизированные имена классов.

### Основные методы
- `make()`: Основной рекурсивный метод, который обрабатывает значения свойств для создания списка определений CSS-классов, включая индексные ключи и отформатированные имена классов.

### Вторичные методы
- `isClasses()`: Определяет, помечен ли данный элемент свойства явно как CSS-подкласс.
- `getNames()`: Разрешает иерархическое имя класса, учитывая, должно ли использоваться полное имя или относительное имя на основе родителя.
