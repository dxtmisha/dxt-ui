# Класс AiZAi

Класс `AiZAi` представляет собой конкретную реализацию `AiZAiLite`, которая интегрируется с настройками окружения проекта. Он автоматически загружает учетные данные с помощью `getConfigAi()` для инициализации клиента Zhipu AI (Z.ai).

## Ключевые особенности

- **Автоматическая конфигурация** — автоматически загружает API-ключи и модели по умолчанию с помощью `getConfigAi()`.
- **Наследуемый функционал Z.ai** — поддерживает все возможности `AiZAiLite` и `AiOpenAiLite`, используя OpenAI-совместимые эндпоинты, настроенные с базовым URL-адресом Z.ai.

## Инициализация

Класс `AiZAi` может быть инициализирован напрямую без аргументов конструктора, так как он автоматически извлекает учетные данные из окружения.

```typescript
import { AiZAi } from '@dxtmisha/scripts'

const ai = new AiZAi()

// Установка модели (опционально, если модель по умолчанию не задана или требует переопределения)
ai.setModel('glm-4')

// Накопление контекста
ai.addPrompt('Вы — полезный помощник.')

// Генерация текста
const response = await ai.generate('Объясните концепцию монад.')
```

## Методы

Этот класс не определяет дополнительных публичных методов. Он наследует все методы настройки и выполнения из `AiZAiLite`, `AiOpenAiLite` и `AiAbstract`.
