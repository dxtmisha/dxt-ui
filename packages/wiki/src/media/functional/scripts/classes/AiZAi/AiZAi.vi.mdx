# Lớp AiZAi

Lớp `AiZAi` là một triển khai cụ thể của `AiZAiLite` tích hợp với các cấu hình môi trường của dự án. Nó tự động tải thông tin xác thực bằng cách sử dụng `getConfigAi()` để khởi tạo client Zhipu AI (Z.ai).

## Tính năng chính

- **Cấu hình Tự động** — tự động tải API key và mô hình mặc định bằng cách sử dụng `getConfigAi()`.
- **Kế thừa các Tính năng của Z.ai** — hỗ trợ tất cả các tính năng của `AiZAiLite` và `AiOpenAiLite` thông qua các endpoint tương thích với OpenAI được định cấu hình bằng URL cơ sở của Z.ai.

## Khởi tạo

Lớp `AiZAi` có thể được khởi tạo trực tiếp mà không cần tham số trong constructor vì nó tự động lấy thông tin xác thực từ môi trường.

```typescript
import { AiZAi } from '@dxtmisha/scripts'

const ai = new AiZAi()

// Thiết lập mô hình (tùy chọn, nếu mô hình mặc định chưa được đặt hoặc cần ghi đè)
ai.setModel('glm-4')

// Tích lũy ngữ cảnh
ai.addPrompt('Bạn là một trợ lý hữu ích.')

// Tạo văn bản
const response = await ai.generate('Giải thích khái niệm về monad.')
```

## Phương thức

Lớp này không định nghĩa thêm phương thức public nào khác. Nó kế thừa tất cả các phương thức cấu hình và thực thi từ `AiZAiLite`, `AiOpenAiLite` và `AiAbstract`.
