# Класс AiClaude

Класс `AiClaude` представляет собой конкретную реализацию `AiClaudeLite`, которая автоматически интегрируется с настройками окружения проекта. Он инициализирует клиент Anthropic AI, используя учетные данные, загруженные через функцию `getConfigAi()`.

## Ключевые особенности

- **Автоматическая конфигурация** — автоматически загружает API-ключи и модели по умолчанию с помощью `getConfigAi()`.
- **Наследуемый функционал Claude** — поддерживает все возможности `AiClaudeLite`, включая мультимодальный ввод (изображения), накопление промптов и генерацию текста.

## Инициализация

Класс `AiClaude` может быть инициализирован напрямую без аргументов конструктора, так как он автоматически извлекает учетные данные из окружения.

```typescript
import { AiClaude } from '@dxtmisha/scripts'

const ai = new AiClaude()

// Установка модели (опционально, если модель по умолчанию не задана или требует переопределения)
ai.setModel('claude-3-5-sonnet-20241022')

// Накопление контекста
ai.addPrompt('Вы — полезный помощник.')

// Генерация текста
const response = await ai.generate('Объясните квантовые вычисления.')
```

## Методы

Этот класс не определяет дополнительных публичных методов. Он наследует все методы настройки и выполнения из `AiClaudeLite` и `AiAbstract`.
