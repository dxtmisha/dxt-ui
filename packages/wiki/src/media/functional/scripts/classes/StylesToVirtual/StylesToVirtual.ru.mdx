# Класс StylesToVirtual

Класс `StylesToVirtual` отвечает за создание CSS-псевдоэлементов (таких как `::before`, `::after`) и связанных с ними стилей.

## Ключевые возможности

- **Обертка псевдоэлементов** — Автоматически оборачивает контент в селектор `&::[name]`.
- **Автоматическое свойство content** — Для псевдоэлементов, таких как `before` и `after`, автоматически внедряет `content: ' ';`, если иное не указано.
- **Управление отступами** — Правильно обрабатывает отступы блоков и необязательные начальные пустые строки.

## Инициализация

Инициализируйте преобразователь `StylesToVirtual` с помощью необходимого свойства и функции контента.

**Параметры:**
- `property: PropertyItemsItem` — текущая обрабатываемая ветвь дерева свойств.
- `space: string` — строка отступа.
- `content?: () => string[]` — вызываемая функция, которая возвращает массив строк, представляющих стили для псевдоэлемента.
- `first?: boolean` — указывает, является ли элемент первым (необязательно).

```typescript
import { StylesToVirtual } from '@dxtmisha/scripts'

const converter = new StylesToVirtual(property, '  ', contentCallable)
const result = converter.make()
```

## Методы

### Управление

- `make(): string[]` — Основная точка входа. Организует процесс обертывания вложенных стилей в блок псевдоэлемента.
