# Класс AiDocItemClasses

Специализированная реализация класса `AiDocItemAbstract`, предназначенная для автоматизированной генерации документации и описаний для классовых структур (classes).

## Назначение

Класс адаптирует базовую логику `AiDocItemAbstract` под специфику объектно-ориентированного кода. Он использует специализированные шаблоны промптов, которые настраивают ИИ на глубокий анализ методов, свойств, конструкторов и иерархии наследования классов.

## Переопределенные свойства

Класс переопределяет пути к ресурсам, необходимым для анализа классовых структур:

- `pathDemo` — путь к файлу `demoClassEn.mdx`, содержащему примеры корректного оформления документации для классов.
- `pathPrompt` — путь к файлу `aiDocClassPrompt.en.txt`, содержащему системные инструкции для ИИ по анализу кода классов.

## Наследование

Класс наследует все механизмы управления процессом генерации из [AiDocItemAbstract](?path=/docs/scripts-classes-aidocitemabstract--docs), включая основной метод `make()`.

## Пример использования

```typescript
import { AiDocItemClasses } from '@dxtmisha/scripts'

// Создание экземпляра для конкретного класса
const generator = new AiDocItemClasses(sourcePath, gitItem);

// Запуск генерации документации
await generator.make();
```
