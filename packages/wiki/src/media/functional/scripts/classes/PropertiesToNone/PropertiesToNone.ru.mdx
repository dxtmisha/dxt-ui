# Класс PropertiesToNone

Класс `PropertiesToNone` очищает дерево токенов дизайна, удаляя пустые и неиспользуемые конфигурации, тем самым оптимизируя память и размер выходных данных. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Удаление избыточных узлов** — Идентифицирует узлы свойств и переменных, содержащие пустые объекты значений.
- **Журналирование операций** — Выводит предупреждения об удалении пустых путей в консоль для аудита разработчиками.
- **Кэширование результатов** — Сохраняет очищенное дерево свойств в файл кэша `900-none`.

## Инициализация

```typescript
import { PropertiesToNone, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToNone(items)
transformer.to()
```

## Методы

### Шаги жизненного цикла

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для поиска и удаления пустых записей в ветках переменных и свойств.
