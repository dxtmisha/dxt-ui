# StylesToClassFull Class

The `StylesToClassFull` class is designed to generate CSS rules where the selector is provided as a full string. Unlike `StylesToClass`, this class always generates the CSS block, even if it contains no internal properties.

## Key Features

- **Mandatory Generation** — Always produces a CSS block with the specified selector.
- **Full Selector Support** — Directly uses the provided string as the CSS selector without additional processing.
- **Automatic Block Wrapping** — Correctly wraps content within standard CSS braces `{}`.

## Initialization

Initialize the `StylesToClassFull` converter with the required property and a content callable for the style rules.

**Parameters:**
- `property: PropertyItemsItem` — the current branch of the property tree being processed.
- `space: string` — the indentation string (e.g., `"  "` or `"\t"`).
- `content?: () => string[]` — a callable function that returns an array of strings representing the internal style rules.
- `first?: boolean` — indicates if this is the first element in the current level (optional, default: `true`).

```typescript
import { StylesToClassFull } from '@dxtmisha/scripts'

// Usage in conversion logic
const converter = new StylesToClassFull(property, '  ', contentCallable)
const result = converter.make()
```

## Methods

### Control

- `make(): string[]` — The main entry point. Orchestrates the transformation and adds the required indentation to each line.
