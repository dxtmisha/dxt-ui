# DesignConstructor
Lớp `DesignConstructor` là một trình tạo cấp cao chịu trách nhiệm tạo và cập nhật các file liên quan đến component dựa trên các template đã xác định trước và logic constructor. Nó điều phối việc tạo các file khác nhau bao gồm các thuộc tính của component, props, kiểu dữ liệu, style và file index chính, đảm bảo tính nhất quán và tuân thủ kiến trúc component.

### Các phương thức chính
- `make()`: Điểm đầu vào chính kích hoạt việc tạo tất cả các file component cần thiết (Properties, Props, Types, Styles, Index) bằng cách sử dụng các template và dữ liệu `DesignStructure`.

### Các phương thức phụ
- `makeProperties()`: Tạo file quản lý thuộc tính nội bộ của component.
- `makeProps()`: Tạo giao diện thuộc tính công khai (public props) cho component.
- `makeTypes()`: Tạo file khai báo TypeScript cho các prop của component.
- `makeStyles()`: Tạo file style của component dựa trên các design token.
- `makeIndex()`: Tạo file entry point chính cho component.
- `toReplace()`: Cấu hình và kích hoạt các thay thế template cụ thể cho mỗi loại file.
- `toWrite()`: Xử lý quy trình ghi file thực tế cho nội dung đã tạo.
