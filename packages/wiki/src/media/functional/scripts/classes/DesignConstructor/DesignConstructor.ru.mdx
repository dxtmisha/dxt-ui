# DesignConstructor
Класс `DesignConstructor` — это высокоуровневый генератор, отвечающий за создание и обновление файлов, связанных с компонентами, на основе предопределенных шаблонов и логики конструктора. Он координирует генерацию различных файлов, включая свойства компонентов, props, типы, стили и основной индексный файл, обеспечивая согласованность и соблюдение архитектуры компонентов.

### Основные методы
- `make()`: Основная точка входа, которая запускает генерацию всех необходимых файлов компонента (Properties, Props, Types, Styles, Index) с использованием шаблонов и данных `DesignStructure`.

### Вторичные методы
- `makeProperties()`: Генерирует файл внутреннего управления свойствами компонента.
- `makeProps()`: Генерирует публичный интерфейс свойств для компонента.
- `makeTypes()`: Генерирует файл объявлений TypeScript для props компонента.
- `makeStyles()`: Генерирует файл стилей компонента на основе дизайн-токенов.
- `makeIndex()`: Генерирует основной файл входа для компонента.
- `toReplace()`: Настраивает и запускает специфические замены в шаблонах для каждого типа файла.
- `toWrite()`: Управляет фактическим процессом записи файлов для сгенерированного контента.
