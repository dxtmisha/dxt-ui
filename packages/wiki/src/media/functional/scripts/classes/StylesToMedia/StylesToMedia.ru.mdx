# Класс StylesToMedia

Класс `StylesToMedia` отвечает за создание CSS-запросов `@media` на основе свойств дизайна. Он реализует логику для обертывания вложенных правил стилей в блок медиа-запроса.

## Ключевые возможности

- **Обертка Медиа-запроса** — Автоматически оборачивает созданный контент стилей в правило `@media screen and ...`.
- **Динамическое разрешение запроса** — Использует имя свойства дизайна для определения условия медиа-запроса.
- **Управление отступами** — Корректно обрабатывает отступы блоков и необязательные начальные пустые строки.

## Инициализация

Инициализируйте преобразователь `StylesToMedia` с помощью необходимого свойства и функции контента для вложенных правил.

**Параметры:**
- `property: PropertyItemsItem` — текущая ветка дерева свойств, которая обрабатывается.
- `space: string` — строка отступа (например, `"  "` или `"\t"`).
- `content?: () => string[]` — вызываемая функция, которая возвращает массив строк, представляющих внутренние правила стиля.
- `first?: boolean` — указывает, является ли этот элемент первым на текущем уровне (необязательно, по умолчанию: `true`).

```typescript
import { StylesToMedia } from '@dxtmisha/scripts'

// Использование в логике преобразования
const converter = new StylesToMedia(property, '  ', contentCallable)
const result = converter.make()
```

## Методы

### Управление

- `make(): string[]` — Основная точка входа. Организует преобразование и добавляет необходимый отступ к каждой строке.
