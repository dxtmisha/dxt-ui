# DesignFlags
Lớp `DesignFlags` là một tiện ích chuyên dụng để tạo sprite sheet và các style CSS tương ứng cho quốc kỳ. Nó tự động hóa quá trình tải các icon SVG, sắp xếp chúng trên một canvas và tạo các lớp CSS dựa trên vị trí để sử dụng hiệu quả trong giao diện người dùng.

### Các phương thức chính
- `make()`: Điểm đầu vào chính thực hiện lặp qua thư mục flag, tạo sprite sheet và lưu cả file hình ảnh và file CSS.
- `getStyle()`: Tạo toàn bộ nội dung CSS với các vị trí background cho mỗi lớp flag.

### Các phương thức cấu hình & Canvas
- `getCanvasWidth()`, `getCanvasHeight()`: Tính toán kích thước canvas cần thiết dựa trên số lượng flag và số cột.
- `getDx()`, `getDy()`: Tính toán tọa độ vẽ cho mục flag hiện tại.
- `initCanvas()`, `initContext()`: Thiết lập môi trường vẽ.
- `initList()`: Quét thư mục được chỉ định để tìm các file `.svg` đưa vào sprite sheet.

### Lưu & Điều hướng
- `addImage()`: Tải và vẽ một flag SVG riêng lẻ lên canvas.
- `addData()`: Ghi lại tọa độ và metadata cho mỗi flag.
- `next()`: Di chuyển con trỏ đến vị trí lưới tiếp theo trên sprite sheet.
- `save()`, `saveStyle()`: Xử lý đầu ra file cho sprite kết quả (WebP) và stylesheet (CSS).
