# DesignFlags
Класс `DesignFlags` — это специализированная утилита для генерации спрайт-листа и соответствующих CSS-стилей для флагов стран. Он автоматизирует процесс загрузки SVG-иконок, их размещения на холсте и генерации CSS-классов на основе позиций для эффективного использования в интерфейсе.

### Основные методы
- `make()`: Основная точка входа, которая проходит по директории флагов, генерирует спрайт-лист и сохраняет как изображение, так и CSS-файл.
- `getStyle()`: Генерирует полное содержимое CSS с позициями фона для каждого класса флага.

### Методы конфигурации и холста
- `getCanvasWidth()`, `getCanvasHeight()`: Рассчитывают необходимые размеры холста на основе количества флагов и колонок.
- `getDx()`, `getDy()`: Рассчитывают координаты рисования для текущего элемента флага.
- `initCanvas()`, `initContext()`: Настраивают среду рисования.
- `initList()`: Сканирует указанную директорию на наличие `.svg` файлов для включения в спрайт-лист.

### Сохранение и навигация
- `addImage()`: Загружает и рисует отдельный SVG-флаг на холсте.
- `addData()`: Записывает координаты и метаданные для каждого флага.
- `next()`: Продвигает курсор на следующую позицию сетки в спрайт-листе.
- `save()`, `saveStyle()`: Управляют выводом файлов для результирующего спрайта (WebP) и таблицы стилей (CSS).
