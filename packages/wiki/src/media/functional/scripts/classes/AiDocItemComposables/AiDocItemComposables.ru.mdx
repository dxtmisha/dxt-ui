# Класс AiDocItemComposables

Специализированная реализация класса `AiDocItemAbstract`, предназначенная для автоматизированной генерации документации и обновления кода для Vue-композитных функций (composables).

## Назначение

Класс расширяет базовую логику `AiDocItemAbstract`, подключая специфичные для композитных функций шаблоны промптов и примеры демонстрационного кода. Это позволяет ИИ более точно анализировать структуру реактивных функций (использующих `ref`, `computed`, `watch` и др.) и генерировать документацию, соответствующую стандартам композитных функций библиотеки.

## Переопределенные свойства

Класс устанавливает пути к ресурсам, используемым в процессе генерации для композитных функций:

- `pathDemo` — путь к файлу `demoComposableEn.mdx`, который служит эталонным примером оформления документации для композитных функций.
- `pathPrompt` — путь к файлу `aiDocComposablePrompt.en.txt`, содержащему системные инструкции для ИИ по анализу кода композитных функций.

## Наследование

Класс наследует все методы управления процессом из [AiDocItemAbstract](?path=/docs/scripts-classes-aidocitemabstract--docs), включая основной метод `make()`.

## Пример использования

```typescript
import { AiDocItemComposables } from '@dxtmisha/scripts'

// Создание экземпляра для конкретной композитной функции
const generator = new AiDocItemComposables(sourcePath, gitItem);

// Запуск процесса обновления документации
await generator.make();
```
