# DesignTypes
The `DesignTypes` class is a specialized tool for optimizing and refining TypeScript type definitions using AI. It processes `.d.ts` files generated by the design system, sending them to an AI model to remove redundancies, improve readability, and ensure consistency across the component library.

### Primary Methods
- `make()`: The main entry point that reads the target `.d.ts` file, sends its content to the AI, and saves the optimized result back to the file.
- `getPrompt()`: Construct the detailed instruction prompt for the AI, explaining the desired optimizations for the type definitions.

### Properties
- `ai`: An instance of an AI service (e.g., `AiGoogle`) used for processing the type definitions.
- `file`: The path to the TypeScript declaration file to be optimized.
