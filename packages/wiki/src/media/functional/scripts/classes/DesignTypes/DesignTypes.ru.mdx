# DesignTypes
Класс `DesignTypes` — это специализированный инструмент для оптимизации и уточнения определений типов TypeScript с использованием ИИ. Он обрабатывает файлы `.d.ts`, сгенерированные дизайн-системой, отправляя их модели ИИ для удаления избыточности, улучшения читаемости и обеспечения согласованности во всей библиотеке компонентов.

### Основные методы
- `make()`: Основная точка входа, которая считывает целевой файл `.d.ts`, отправляет его содержимое ИИ и сохраняет оптимизированный результат обратно в файл.
- `getPrompt()`: Формирует подробную инструкцию (промпт) для ИИ, объясняющую желаемые оптимизации для определений типов.

### Свойства
- `ai`: Экземпляр сервиса ИИ (например, `AiGoogle`), используемый для обработки определений типов.
- `file`: Путь к файлу объявлений TypeScript, который необходимо оптимизировать.
