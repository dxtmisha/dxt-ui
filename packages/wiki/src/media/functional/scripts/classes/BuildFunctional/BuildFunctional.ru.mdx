# Класс BuildFunctional

Класс `BuildFunctional` — это утилита, отвечающая за генерацию центрального файла экспорта для функциональной библиотеки. Он сканирует пакет `@dxtmisha/functional` и создает файл `functional.ts` в директории библиотеки, повторно экспортируя все обнаруженные элементы.

## Ключевые особенности

- **Автоматический реэкспорт** — автоматически обнаруживает и повторно экспортирует все элементы из функционального пакета.
- **Поддержка типов** — включает `export type *` для обеспечения полной доступности типов TypeScript.
- **Интеграция с библиотекой** — интегрируется со структурой библиотеки проекта, записывая данные в настроенную директорию библиотеки.

## Инициализация

Вы можете создать экземпляр `BuildFunctional` без параметров.

```typescript
import { BuildFunctional } from './BuildFunctional'

const builder = new BuildFunctional()
builder.make()
```

## Методы

### make

- `make(): void` — запускает процесс сборки. Он получает список элементов, форматирует код экспорта и записывает его в `src/library/functional.ts`.

### getList

- `getList(): string[]` — возвращает массив ключей (экспортируемых элементов) из пакета `@dxtmisha/functional`.
