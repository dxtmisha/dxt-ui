# Класс AiGoogleCliLite

Реализация абстрактного класса `AiAbstract`, использующая системную оболочку (shell) для выполнения команд через утилиту `gemini` CLI. Этот подход позволяет взаимодействовать с Google AI там, где использование SDK невозможно или нежелательно.

## Ключевые особенности

- **CLI-интеграция** — выполнение запросов через запуск внешнего процесса в операционной системе.
- **Работа через временные файлы** — промпты сохраняются во временной директории для обхода ограничений на длину аргументов командной строки.
- **Автоматическая очистка** — временные файлы и директории удаляются сразу после выполнения запроса.

## Основной метод

### `response`

Защищенный метод, формирующий и выполняющий команду в терминале. Он подготавливает полный текст промпта, экранирует спецсимволы и вызывает нативную утилиту `gemini`.

**Основные этапы работы:**
1. Подготовка содержимого промпта и сохранение его во временный файл (`createFile`).
2. Формирование строки команды с флагами модели и инструкциями по выводу.
3. Вызов `exec` с передачей API-ключа через переменные окружения.
4. Парсинг `stdout` и возвращение результата.
5. Очистка временных ресурсов в блоке завершения.

**Параметры:**
- `model: string` — идентификатор модели.
- `contents: string` — основное содержимое запроса.

**Возвращает:** `Promise<string>`

## Управление контекстом

- `createFile(content: string): string` — Записывает контент во временный файл и возвращает путь в формате `@путь/к/файлу.txt`, понятном для CLI.
- `getFileName(): string` — Генерирует уникальное имя для временного файла в папке `./ai-tmp`.
- `removeFile(): void` — Удаляет временную директорию с промптами.

## Наследование

Класс расширяет [AiAbstract](?path=/docs/scripts-classes-aiabstract--docs). В данной реализации поддержка изображений (`toImages`) в настоящее время не предусмотрена.

## Пример использования

```typescript
import { AiGoogleCliLite } from '@dxtmisha/scripts'

const ai = new AiGoogleCliLite('YOUR_API_KEY');
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Напиши юнит-тест для функции сложения');
```
