# DesignWikiStormItem
Lớp `DesignWikiStormItem` chịu trách nhiệm tạo các định nghĩa Web-Type chi tiết cho một component duy nhất. Nó trích xuất và chuyển đổi các metadata của component thành định dạng tương thích với schema JSON của Web-Types, bao gồm tên thuộc tính, kiểu dữ liệu, mô tả, giá trị mặc định và các slot khả dụng.

### Các phương thức chính
- `get()`: Điểm đầu vào chính tổng hợp tất cả thông tin cụ thể của component thành một đối tượng cấu trúc phù hợp để đưa vào file `web-types.json`.

### Các phương thức phụ
- `getAttributes()`: Trích xuất các thuộc tính của component (props) và các metadata liên quan.
- `getSlots()`: Xác định và định dạng các slot của component.
- `getEvents()`: (Placeholder) Để tích hợp các định nghĩa sự kiện của component trong tương lai.
- `getDescription()`: Lấy mô tả JSDoc hoặc wiki của component và các thuộc tính của nó.
- `getType()`: Chuẩn hóa các kiểu TypeScript thành các chuỗi tương thích với Web-Types.
- `getDefault()`: Phân giải các giá trị mặc định cho các thuộc tính của component.
- `is()`: Kiểm tra xem thư mục component có hợp lệ và chứa các file index cần thiết hay không.
