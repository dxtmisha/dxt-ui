# DesignWikiStormItem
Класс `DesignWikiStormItem` отвечает за генерацию подробных определений Web-Type для одного компонента. Он извлекает и преобразует метаданные компонента в формат, совместимый со схемой JSON Web-Types, включая имена свойств, типы, описания, значения по умолчанию и доступные слоты.

### Основные методы
- `get()`: Основная точка входа, которая агрегирует всю специфичную для компонента информацию в структурированный объект, подходящий для включения в файл `web-types.json`.

### Вторичные методы
- `getAttributes()`: Извлекает свойства компонента (props) и связанные с ними метаданные.
- `getSlots()`: Идентифицирует и форматирует слоты компонента.
- `getEvents()`: (Заполнитель) Для будущей интеграции определений событий компонента.
- `getDescription()`: Извлекает JSDoc или вики-описание для компонента и его свойств.
- `getType()`: Нормализует типы TypeScript в строки, совместимые с Web-Types.
- `getDefault()`: Разрешает значения по умолчанию для свойств компонента.
- `is()`: Проверяет, является ли директория компонента валидной и содержит ли необходимые индексные файлы.
