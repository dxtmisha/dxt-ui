# Класс PropertiesToReplace

Класс `PropertiesToReplace` преобразует строковые значения токенов дизайна с помощью регулярных выражений, заданных конфигурацией `replace`. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Регулярные выражения** — Оценивает критерии соответствия регулярных выражений и переопределяет значения токенов обработанным выводом.
- **Гибкие варианты конфигурации** — Поддерживает как короткие строки шаблонов, так и подробные объекты конфигурации `PropertyReplace` (определяющие шаблон, флаги и строки замены).
- **Безопасная очистка** — Удаляет временный флаг директивы `replace` из свойств токена после успешного преобразования целевого значения.
- **Кэширование результатов** — Сохраняет обновленные сопоставления токенов в файл кэша `001-replace`.

## Инициализация

```typescript
import { PropertiesToReplace, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToReplace(items)
transformer.to()
```

## Методы

### Операции преобразования и утилиты

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для прохода по всем свойствам, обнаружения строковых значений с директивой `replace` и выполнения преобразований.
- `getInfo(info: PropertiesReplaceValue): PropertyReplace` — Нормализует входящие короткие шаблоны строк или подробные конфигурации в структурированные объекты `PropertyReplace`.
- `getValue(info: PropertyReplace, value: string): string` — Выполняет замену по регулярному выражению, возвращая преобразованную строку.
