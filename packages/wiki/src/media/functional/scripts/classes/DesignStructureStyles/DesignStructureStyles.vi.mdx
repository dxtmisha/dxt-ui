# DesignStructureStyles
Lớp `DesignStructureStyles` chịu trách nhiệm chuyển đổi các design token thành các định nghĩa style CSS thực tế. Nó sử dụng tiện ích `StylesProperties` để tạo ra các quy tắc CSS đã được định dạng dựa trên các mục cấu trúc của component.

### Các phương thức chính
- `make()`: Thực hiện việc chuyển đổi các token của component thành một mảng các chuỗi CSS, bao bọc chúng trong một bộ chọn tiêu chuẩn (ví dụ: `& { ... }`).

### Thuộc tính
- `data`: Lưu trữ mảng kết quả của các chuỗi CSS style.
