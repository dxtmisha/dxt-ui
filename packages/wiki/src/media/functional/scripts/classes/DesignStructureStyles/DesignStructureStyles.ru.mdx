# DesignStructureStyles
Класс `DesignStructureStyles` отвечает за преобразование дизайн-токенов в фактические определения CSS-стилей. Он использует утилиту `StylesProperties` для генерации отформатированных CSS-правил на основе структурных элементов компонента.

### Основные методы
- `make()`: Выполняет преобразование токенов компонента в массив CSS-строк, оборачивая их в стандартный селектор (например, `& { ... }`).

### Свойства
- `data`: Хранит результирующий массив строк CSS-стилей.
