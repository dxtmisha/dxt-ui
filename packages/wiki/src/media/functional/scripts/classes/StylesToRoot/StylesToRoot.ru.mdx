# Класс StylesToRoot

Класс `StylesToRoot` используется для обертывания блока вложенных правил в директиву CSS `@at-root`. Это полезно для выноса правил из их текущей области видимости в корень таблицы стилей.

## Ключевые возможности

- **Обертка At-Root** — Автоматически оборачивает созданный контент в блок `@at-root`.
- **Динамическое включение селектора** — Добавляет имя свойства (выступающее как расширение селектора) к директиве `@at-root`.
- **Управление отступами** — Обрабатывает стандартные отступы блоков и начальные пустые строки.

## Инициализация

Инициализируйте преобразователь `StylesToRoot` с помощью необходимого свойства и функции контента.

**Параметры:**
- `property: PropertyItemsItem` — текущая обрабатываемая ветвь дерева свойств.
- `space: string` — строка отступа.
- `content?: () => string[]` — вызываемая функция, которая возвращает массив строк, представляющих внутренние правила стиля.
- `first?: boolean` — указывает, является ли элемент первым (необязательно).

```typescript
import { StylesToRoot } from '@dxtmisha/scripts'

const converter = new StylesToRoot(property, '  ', contentCallable)
const result = converter.make()
```

## Методы

### Управление

- `make(): string[]` — Основная точка входа. Организует процесс обертывания контента в структуру `@at-root`.
