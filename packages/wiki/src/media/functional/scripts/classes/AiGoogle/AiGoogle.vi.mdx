# Lớp AiGoogle

Một bản triển khai đã được cấu hình của Google AI (Gemini), kế thừa từ [AiGoogleLite](?path=/docs/scripts-classes-aigooglelite--docs). Lớp này tự động lấy các thông tin xác thực cần thiết từ cấu hình toàn cục của dự án, giúp loại bỏ việc phải truyền mã API thủ công.

## Mục đích

Lớp này đóng vai trò như một lớp bọc (wrapper) tiện lợi cho việc sử dụng Gemini tiêu chuẩn trong hạ tầng của thư viện. Nó tự động tích hợp với hàm `getConfigAi()`, đảm bảo sẵn sàng sử dụng ngay sau khi khởi tạo.

## Hàm khởi tạo (Constructor)

Khi được khởi tạo, lớp sẽ tự động:
1. Gọi hàm `getConfigAi()` để lấy mã API và các tham số khác từ môi trường hoặc các tệp cấu hình.
2. Truyền dữ liệu đã lấy được vào hàm khởi tạo của lớp cha `AiGoogleLite`.

## Kế thừa

Lớp này kế thừa tất cả logic tương tác với SDK từ [AiGoogleLite](?path=/docs/scripts-classes-aigooglelite--docs) và các cơ chế tích lũy prompt chung từ [AiAbstract](?path=/docs/scripts-classes-aiabstract--docs).

## Ví dụ sử dụng

```typescript
import { AiGoogle } from '@dxtmisha/scripts'

// Mã API và các cài đặt được tự động lấy từ cấu hình dự án
const ai = new AiGoogle();
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Explain how reactivity works in Vue');
```
