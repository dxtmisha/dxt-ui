# Класс AiGoogle

Настроенная реализация Google AI (Gemini), расширяющая [AiGoogleLite](?path=/docs/scripts-classes-aigooglelite--docs). Этот класс автоматически получает необходимые учетные данные из глобальной конфигурации проекта, избавляя от необходимости ручной передачи ключей.

## Назначение

Класс служит удобной оберткой для стандартного использования Gemini в рамках инфраструктуры библиотеки. Он автоматически интегрируется с функцией `getConfigAi()`, обеспечивая готовность к работе сразу после инициализации.

## Конструктор

При вызове конструктора класс автоматически:
1. Вызывает функцию `getConfigAi()` для получения API-ключа и других параметров из окружения или файлов настроек.
2. Передает полученные данные в суперконструктор `AiGoogleLite`.

## Наследование

Класс наследует всю логику взаимодействия с SDK из [AiGoogleLite](?path=/docs/scripts-classes-aigooglelite--docs) и общие механизмы накопления промптов из [AiAbstract](?path=/docs/scripts-classes-aiabstract--docs).

## Пример использования

```typescript
import { AiGoogle } from '@dxtmisha/scripts'

// Ключ и настройки берутся автоматически из конфигурации проекта
const ai = new AiGoogle();
ai.setModel('gemini-1.5-flash');

const result = await ai.generate('Объясни принцип работы реактивности в Vue');
```
