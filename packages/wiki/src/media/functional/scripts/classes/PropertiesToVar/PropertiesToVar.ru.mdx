# Класс PropertiesToVar

Класс `PropertiesToVar` преобразует токены дизайна, представляющие переменные, в стандартные пользовательские свойства CSS (`--var-name: values`). Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Синтез переменных CSS** — Преобразует имена токенов переменных в пользовательские свойства с префиксом из двух дефисов (`--prefix-name`).
- **Математические выражения** — Автоматически оборачивает стандартные математические выражения (`+`, `-`, `*`, `/`) внутри блоков `calc(...)`.
- **Развертывание ссылок** — Разрешает ссылки в фигурных скобках в вложенные форматы CSS `var(...)` с поддержкой резервных значений по умолчанию.
- **Разрешение прозрачности** — Безопасно обнаруживает переменные цветов, содержащие атрибуты прозрачности, помечая их метаданными прозрачности.
- **Кэширование результатов** — Сохраняет разрешенные определения переменных в файл кэша `020-var`.

## Инициализация

```typescript
import { PropertiesToVar, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToVar(items)
transformer.to()
```

## Методы

### Операции разрешения

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для получения узлов типов переменных, синтезирования имен переменных, форматирования их значений CSS и разрешения прозрачности цветов.
- `getName(property: PropertyItemsItem): string` — Создает стандартные ключи именования свойств CSS (например, `--sys-color-primary`), ограниченные родительскими пространствами имен.
- `toCalculator(value: string, full?: boolean): string` — Оборачивает математические выражения в инструкции `calc()`, если они еще не отформатированы.
- `toLink(value: string): string` — Нормализует ключи индексов ссылок в стандартные сегменты имен переменных.
- `toValue(value: string, defaultValue?: PropertyItem['_default']): string` — Рекурсивно разрешает вложенные ссылки в фигурных скобках в стандартные значения CSS `var(--name, fallback)`.
- `isNone(parents: PropertyItemsParent[]): boolean` — Возвращает true, если какой-либо родительский элемент имеет тип `PropertyType.none`, представляющий скрытое свойство.
- `isColorWithOpacity(property: PropertyItemsItem): boolean` — Безопасно обходит ссылки на цвета для проверки активности свойств прозрачности.
