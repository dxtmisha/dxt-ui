# Lớp AiClaudeCli

Lớp `AiClaudeCli` là một triển khai cụ thể của `AiClaudeCliLite` tích hợp với các cấu hình môi trường của dự án. Nó tự động tải thông tin xác thực bằng cách sử dụng `getConfigAi()` để gọi Claude thông qua terminal shell.

## Tính năng chính

- **Cấu hình Tự động** — tự động tải API key và mô hình từ môi trường dự án bằng cách sử dụng `getConfigAi()`.
- **Tạo Phản hồi qua Shell** — kế thừa cơ chế CLI runner của `AiClaudeCliLite` để thực thi các lệnh terminal.

## Khởi tạo

Lớp `AiClaudeCli` có thể được khởi tạo trực tiếp mà không cần tham số trong constructor vì nó tự động lấy thông tin xác thực từ môi trường.

```typescript
import { AiClaudeCli } from '@dxtmisha/scripts'

const ai = new AiClaudeCli()

// Tạo văn bản thông qua CLI hệ thống
const response = await ai.generate('Viết thuật toán sắp xếp nhanh (quick sort) bằng TypeScript.')
```

## Phương thức

Lớp này không định nghĩa thêm phương thức public nào khác. Nó kế thừa tất cả các phương thức cấu hình và thực thi từ `AiClaudeCliLite` và `AiAbstract`.
