# Класс PropertiesToClone

Класс `PropertiesToClone` дублирует параметры и структуры конфигурации из родительских элементов в соответствующие дочерние свойства. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Каскадная конфигурация** — Клонирует настройки, помеченные флагом `settingClone`, вниз по дереву свойств.
- **Глубокий обход** — Рекурсивно сканирует и наследует атрибуты для похожих свойств у потомков.
- **Игнорирование типов** — Автоматически игнорирует типы переменных и свойств при клонировании во избежание перезаписи значений.
- **Кэширование результатов** — Сохраняет состояние обработанных свойств в файл кэша `007-01-clone`.

## Инициализация

```typescript
import { PropertiesToClone, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToClone(items)
transformer.to()
```

## Методы

### Операции клонирования

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для обхода всех свойств и применения клонирования.
- `findItem(parent: PropertyItem, name: string): PropertyItem[]` — Рекурсивно ищет похожие записи свойств на уровнях потомков родительского элемента.
- `update(original: PropertyItem, properties: PropertyItem[]): void` — Выполняет внедрение атрибутов, объединяя свойства оригинального элемента без перезаписи существующих данных.
