# Класс PropertiesToFull

Класс `PropertiesToFull` разрешает имена токенов дизайна и строковые значения в их полные, глобально уникальные переменные CSS и ссылки. Он наследует класс `PropertiesToAbstract`.

## Ключевые особенности

- **Синтез полных переменных** — Преобразует относительные имена токенов в полные объявления пользовательских свойств CSS на основе пространств имен дизайна и компонентов.
- **Разрешение ссылок на значения** — Разрешает простые справочные значения в полные форматы глобальных указателей на переменные.
- **Кэширование результатов** — Сохраняет разрешенные переменные в файл кэша `018-full`.

## Инициализация

```typescript
import { PropertiesToFull, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToFull(items)
transformer.to()
```

## Методы

### Шаги жизненного цикла

- `init(): void` — Переопределяет абстрактный шаг жизненного цикла для обхода всех активных свойств, проверки области видимости компонентов и разрешения полных имен/значений CSS.
