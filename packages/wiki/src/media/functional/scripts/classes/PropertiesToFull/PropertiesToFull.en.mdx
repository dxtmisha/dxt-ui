# PropertiesToFull Class

The `PropertiesToFull` class resolves design token names and string values into their full, globally unique CSS variables and links. It extends `PropertiesToAbstract`.

## Key Features

- **Full Variable Synthesis** — Converts relative token names into full CSS custom property declarations based on design and component namespace.
- **Value Link Resolution** — Resolves simple reference values into full global variable pointer formats.
- **Cache Persistence** — Saves the resolved variables to the `018-full` cache file.

## Initialization

```typescript
import { PropertiesToFull, PropertiesItems } from '@dxtmisha/scripts'

const transformer = new PropertiesToFull(items)
transformer.to()
```

## Methods

### Lifecycle Hooks

- `init(): void` — Overrides the abstract lifecycle hook to iterate through all active properties, checking for component scope, and resolving full names/CSS values.
