# `useFigmaStorage`

Composable để quản lý dữ liệu trong `PluginData` của Figma. Nó sử dụng mô hình singleton để đảm bảo rằng nhiều lần gọi với cùng một khóa và node sẽ trả về cùng một thực thể `FigmaStorage`.

**Tham số:**
- `name: string` — tên khóa duy nhất cho mục lưu trữ.
- `item?: PluginDataMixin` — (Tùy chọn) node Figma nơi dữ liệu được lưu trữ. Mặc định là `figma.root`.
- `cache?: number` — (Tùy chọn) thời gian lưu bộ nhớ đệm tính bằng giây (TTL).

**Trả về:**
- `FigmaStorage<T>` — một thực thể của lớp `FigmaStorage` để quản lý dữ liệu.

## Ví dụ sử dụng

```typescript
import { useFigmaStorage } from '@dxtmisha/figma'

// 1. Khởi tạo lưu trữ cho cài đặt plugin
const settings = useFigmaStorage('settings')

// 2. Truy xuất dữ liệu với giá trị mặc định
const theme = settings.get('dark')

// 3. Cập nhật dữ liệu
settings.set('light')
```
