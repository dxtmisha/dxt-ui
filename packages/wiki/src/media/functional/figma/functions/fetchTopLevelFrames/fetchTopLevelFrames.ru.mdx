# `fetchTopLevelFrames`

Утилита для получения списка всех фреймов верхнего уровня, присутствующих в документе Figma. Обеспечивает связь между интерфейсом (UI) и основной частью плагина для доступа к структуре документа.

**Параметры:**
- `callback: (frames: UiFigmaFramesList) => void` — Функция, которая будет вызвана после получения списка фреймов верхнего уровня.

**Возвращает:**
`void`

## Типы

### UiFigmaFramesList

Массив элементов `UiFigmaFramesItem`.

### UiFigmaFramesItem

- `name: string` — Название фрейма.
- `id: string` — Уникальный идентификатор элемента.
- `image?: Uint8Array | string` — (Опционально) Изображение фрейма в виде байтового массива или строки Base64.

## Примеры

<Source
  language='typescript'
  code={`
import { fetchTopLevelFrames } from '@dxtmisha/figma'

// Получить все фреймы верхнего уровня и вывести информацию о них
fetchTopLevelFrames((frames) => {
  console.log('Найдено фреймов верхнего уровня:', frames.length)
  
  frames.forEach(frame => {
    console.log(\`Фрейм: \${frame.name} (ID: \${frame.id})\`)
  })
})
  `}
/>
