# `ensureMaxSize`

Tiện ích đảm bảo hình ảnh không vượt quá kích thước tối đa bằng cách thay đổi kích thước tỷ lệ nếu cần thiết. Nó chuyển đổi dữ liệu hình ảnh đầu vào thành chuỗi base64 sau khi xử lý.

**Tham số:**
- `file: Uint8Array` — Dữ liệu tệp hình ảnh dưới dạng mảng byte.
- `compress: number = 0.56` — Hệ số kích thước tối đa dưới dạng một phần của kích thước gốc.

**Trả về:**
`Promise<string>` — Một promise sẽ trả về chuỗi base64 của hình ảnh đã được xử lý.

```javascript
import { ensureMaxSize } from '@dxtmisha/figma'

const resizedBase64 = await ensureMaxSize(imageBytes, 0.5)
console.log('Base64 của hình ảnh đã xử lý:', resizedBase64)
```
