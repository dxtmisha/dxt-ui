# `ensureMaxSize`

Утилита, которая гарантирует, что изображение не превышает максимальный размер, при необходимости пропорционально изменяя его. После обработки она преобразует данные изображения в строку base64.

**Параметры:**
- `file: Uint8Array` — Данные файла изображения в виде массива байтов.
- `compress: number = 0.56` — Коэффициент максимального размера как доля от оригинального размера.

**Возвращает:**
`Promise<string>` — Промис, который разрешается в строку обработанного изображения в кодировке base64.

```javascript
import { ensureMaxSize } from '@dxtmisha/figma'

const resizedBase64 = await ensureMaxSize(imageBytes, 0.5)
console.log('Base64 обработанного изображения:', resizedBase64)
```
