# `ensureMaxSize`

Utility that ensures an image does not exceed a maximum size by resizing it proportionally if necessary. It converts the input image data into a base64 string after processing.

**Parameters:**
- `file: Uint8Array` — The image file data as a byte array.
- `compress: number = 0.56` — Maximum size factor as a fraction of the original size.

**Returns:**
`Promise<string>` — A promise that resolves to the base64-encoded string of the processed image.

```javascript
import { ensureMaxSize } from '@dxtmisha/figma'

const resizedBase64 = await ensureMaxSize(imageBytes, 0.5)
console.log('Processed image base64:', resizedBase64)
```
