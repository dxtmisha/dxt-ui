# `addFramesSelected`

Утилита, которая отправляет состояние выбора конкретного фрейма в плагин Figma.

**Параметры:**
- `id: string` — Идентификатор фрейма для обновления.
- `selected: boolean` — Состояние выбора, которое необходимо применить к фрейму.

**Возвращает:**
`void`

```javascript
import { addFramesSelected } from '@dxtmisha/figma'

addFramesSelected('123:456', true)
```
