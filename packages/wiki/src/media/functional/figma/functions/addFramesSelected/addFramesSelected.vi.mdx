# `addFramesSelected`

Một tiện ích giúp gửi trạng thái lựa chọn của một frame cụ thể đến plugin Figma.

**Tham số:**
- `id: string` — Mã định danh (ID) của frame cần cập nhật.
- `selected: boolean` — Trạng thái lựa chọn sẽ áp dụng cho frame.

**Trả về:**
`void`

```javascript
import { addFramesSelected } from '@dxtmisha/figma'

addFramesSelected('123:456', true)
```
