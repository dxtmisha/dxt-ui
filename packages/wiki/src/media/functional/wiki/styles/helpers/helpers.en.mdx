# Helper Utilities

These classes are specifically designed for documentation examples, providing quick ways to create mock cards, buttons, and placeholder blocks to visualize layouts and component contexts.

## Classes

### Mock Card
- `.wiki-storybook-card`: The main card container (`320px` width, `12px` border-radius).
  - `.wiki-storybook-card__image`: A fixed-height placeholder for images.
  - `.wiki-storybook-card__content`: Padded vertical container for text and actions.
  - `.wiki-storybook-card__label`: Larger title text (`20px`).
  - `.wiki-storybook-card__information`: Smaller, muted info text (`14px`).
  - `.wiki-storybook-card__actions`: Horizontal flex container for action buttons.

### Button
- `.wiki-storybook-button`: A minimalist button with a subtle border and interactive states (hover/active).

### Dummy Placeholder
- `.wiki-storybook-dummy`: A simple rectangular placeholder.
  - **Colors**: `--color--blue`, `--color--red`, `--color--green`.
  - **Sizes**: `--size--sm` (`64px`), `--size--md` (`128px`), `--size--lg` (`256px`).
