# Sizing Utilities

These classes provide width constraints for documentation examples, which is useful for testing responsive behavior or simply demonstrating components in narrower contexts.

## Classes

- `.wiki-storybook-decreased`: Sets the container width to `72%`.
- `.wiki-storybook-decreasedX2`: A responsive utility that progressively decreases width as the screen gets larger:
  - Default: `100%`
  - MD breakpoint: `64%`
  - LG breakpoint: `48%`

## Examples

### Fixed Decrease
Use `.wiki-storybook-decreased` to consistently occupy about 3/4 of the available width.
