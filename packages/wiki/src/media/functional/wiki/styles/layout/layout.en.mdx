# Layout Utilities

These classes provide structural layout options for documentation examples, ensuring consistent spacing and alignment across different Storybook stories.

## Classes

- `.wiki-storybook-container`: Sets `container-type: inline-size`, enabling container queries for children.
- `.wiki-storybook-group`: A 12-column grid container with an `8px` gap.
- `.wiki-storybook-flex`: A flexbox container with `flex-wrap: wrap` and an `8px` gap.
- `.wiki-storybook-flex-center`: A flexbox container that centers its content both horizontally and vertically.
- `.wiki-storybook-flex-column`: A vertical flexbox container (`flex-direction: column`) with a `16px` gap.

### Width Utilities
These classes allow for controlling the width of an element relative to its container.

- `.wiki-storybook-decreased`: Sets width to `72%`.
- `.wiki-storybook-decreasedX2`: Sets width to `64%` on medium screens and `48%` on large screens.
