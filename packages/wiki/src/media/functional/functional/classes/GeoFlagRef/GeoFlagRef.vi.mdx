# Lớp GeoFlagRef

Một lớp để làm việc với cờ quốc gia theo cách phản ứng. Đảm bảo dữ liệu cờ luôn được đồng bộ với mã quốc gia phản ứng.

## Các tính năng chính

- **Tính phản ứng** — tự động cập nhật dữ liệu khi mã quốc gia thay đổi.
- **Hỗ trợ nhiều quốc gia** — lấy thông tin cho nhiều quốc gia cùng một lúc.
- **Tên quốc gia bản ngữ** — lấy tên quốc gia bằng ngôn ngữ bản địa của họ.

## Khởi tạo

```typescript
import { GeoFlagRef } from '@dxtmisha/functional'
import { ref } from 'vue'

const code = ref('VN')
const geoFlagRef = new GeoFlagRef(code)
```

## Trạng thái hiện tại

- `getCode(): Ref<string>` — trả về một đối tượng phản ứng chứa mã quốc gia.

## Truy xuất thông tin

- `get(code?: string): ComputedRef<GeoFlagItem | undefined>` — trả về thông tin chi tiết về một quốc gia và cờ của quốc gia đó.
- `getFlag(code?: string): ComputedRef<string | undefined>` — trả về định danh biểu tượng cờ.
- `getList(codes?: string[]): ComputedRef<GeoFlagItem[]>` — trả về một mảng dữ liệu cờ cho danh sách các mã đã chỉ định.
- `getNational(codes?: string[]): ComputedRef<GeoFlagNational[]>` — trả về danh sách các quốc gia với tên bằng ngôn ngữ bản địa của họ.
