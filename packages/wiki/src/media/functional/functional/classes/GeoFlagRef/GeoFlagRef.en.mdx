# GeoFlagRef Class

A class for working with country flags in a reactive way. Ensures flag data stays synchronized with a reactive country code.

## Key Features

- **Reactivity** — automatic data updates when the country code changes.
- **Multiple Country Support** — retrieve information for multiple countries at once.
- **National Names** — get country names in their native language.

## Initialization

```typescript
import { GeoFlagRef } from '@dxtmisha/functional'
import { ref } from 'vue'

const code = ref('US')
const geoFlagRef = new GeoFlagRef(code)
```

## Current State

- `getCode(): Ref<string>` — returns a reactive object containing the country code.

## Information Retrieval

- `get(code?: string): ComputedRef<GeoFlagItem | undefined>` — returns comprehensive information about a country and its flag.
- `getFlag(code?: string): ComputedRef<string | undefined>` — returns the flag icon identifier.
- `getList(codes?: string[]): ComputedRef<GeoFlagItem[]>` — returns an array of flag data for the specified list of codes.
- `getNational(codes?: string[]): ComputedRef<GeoFlagNational[]>` — returns a list of countries with names in their national language.
