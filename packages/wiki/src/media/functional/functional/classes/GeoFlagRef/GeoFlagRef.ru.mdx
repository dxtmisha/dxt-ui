# Класс GeoFlagRef

Класс для реактивной работы с флагами стран. Обеспечивает синхронизацию данных о флагах с реактивным кодом страны.

## Ключевые особенности

- **Реактивность** — автоматическое обновление данных при изменении кода страны.
- **Поддержка списка стран** — получение информации о нескольких странах одновременно.
- **Национальные названия** — получение названий стран на их родном языке.

## Инициализация

```typescript
import { GeoFlagRef } from '@dxtmisha/functional'
import { ref } from 'vue'

const code = ref('RU')
const geoFlagRef = new GeoFlagRef(code)
```

## Текущее состояние

- `getCode(): Ref<string>` — возвращает реактивный объект с кодом страны.

## Получение данных

- `get(code?: string): ComputedRef<GeoFlagItem | undefined>` — возвращает полную информацию о стране и её флаге.
- `getFlag(code?: string): ComputedRef<string | undefined>` — возвращает идентификатор иконки флага.
- `getList(codes?: string[]): ComputedRef<GeoFlagItem[]>` — возвращает массив данных о флагах для списка кодов.
- `getNational(codes?: string[]): ComputedRef<GeoFlagNational[]>` — возвращает список стран с названиями на национальном языке.
