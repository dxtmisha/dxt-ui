# Класс GeoFlagRef

Реактивный менеджер для работы с географическими флагами и информацией о странах. Предоставляет удобный способ получения иконок стран и их данных, обеспечивая реактивность при изменении выбранного кода страны.

## Ключевые особенности

- **Реактивность** — автоматически обновляет информацию о флаге при изменении `Ref` кода страны.
- **Удобное получение** — предоставляет простые методы для получения данных о стране и её иконке.
- **Международная поддержка** — поддерживает получение списков стран на национальных языках.

## Инициализация

```typescript
import { GeoFlagRef } from '@dxtmisha/functional'
import { ref } from 'vue'

const countryCode = ref('RU')
const flagManager = new GeoFlagRef(countryCode)

console.log(flagManager.getFlag().value) // Ссылка на иконку флага РФ
```

## Получение флагов

### `get`

Возвращает реактивную информацию о конкретной стране, включая её название и иконку флага.

**Параметры:**
- `code?: string` — код страны (по умолчанию используется текущий код экземпляра).

**Возвращает:** `ComputedRef<GeoFlagItem | undefined>`

### `getFlag`

Возвращает реактивную ссылку на иконку флага страны.

**Параметры:**
- `code?: string` — код страны (по умолчанию используется текущий код экземпляра).

**Возвращает:** `ComputedRef<string | undefined>`

## Списки и информация

### `getCode`

Возвращает реактивный объект с кодом страны, используемым в экземпляре.

**Возвращает:** `Ref<string>`

### `getList`

Возвращает реактивный список объектов стран на основе предоставленного массива кодов.

**Параметры:**
- `codes?: string[]` — необязательный список кодов стран для фильтрации.

**Возвращает:** `ComputedRef<GeoFlagItem[]>`

### `getNational`

Возвращает реактивный список стран с названиями на их национальном языке.

**Параметры:**
- `codes?: string[]` — необязательный список кодов стран для фильтрации.

**Возвращает:** `ComputedRef<GeoFlagNational[]>`

## Объект GeoFlagItem

Основной объект, содержащий информацию о стране и её флаге:

- `language: string` — название языка.
- `country: string` — название страны.
- `standard: string` — стандартный код локали.
- `icon?: string` — идентификатор или ссылка на иконку флага.
- `label: string` — отображаемая метка.
- `value: string` — значение для выбора.

## Объект GeoFlagNational

Расширенная информация о стране, включая названия на национальном языке:

- `description: string` — описание на национальном языке.
- `nationalLanguage: string` — название языка на национальном языке.
- `nationalCountry: string` — название страны на национальном языке.
- (Плюс все поля из `GeoFlagItem`)
