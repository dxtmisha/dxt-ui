# Класс ScrollbarWidthRef

Реактивный утилитарный класс для определения и управления шириной полосы прокрутки браузера. Он использует механизм обнаружения и предоставляет реактивные свойства для проверки того, была ли рассчитана ширина скролла.

## Ключевые особенности

- **Реактивность** — использует Vue `ref` и `computed` для предоставления состояния скролла.
- **Асинхронное обнаружение** — автоматически рассчитывает ширину при инициализации.
- **Легкость** — минимальный размер, полезен для расчетов макета, зависящих от наличия скролла.

## Использование

```typescript
import { ScrollbarWidthRef } from '@dxtmisha/functional'

const scrollbar = new ScrollbarWidthRef()

// Проверка, рассчитана ли ширина
console.log(scrollbar.is.value)

// Получение состояния (boolean)
console.log(scrollbar.item.value)
```

## Свойства

- `item`: `Ref<boolean | undefined>` — Реактивный элемент, содержащий состояние наличия скролла.
- `is`: `ComputedRef<boolean>` — Проверяет, завершен ли расчет ширины полосы прокрутки.
