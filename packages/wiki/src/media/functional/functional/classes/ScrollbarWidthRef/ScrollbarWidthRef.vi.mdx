# Lớp ScrollbarWidthRef

Một lớp tiện ích phản ứng để phát hiện và quản lý chiều rộng thanh cuộn (scrollbar) của trình duyệt. Nó sử dụng cơ chế phát hiện tự động và cung cấp các thuộc tính phản ứng để kiểm tra xem chiều rộng thanh cuộn đã được tính toán hay chưa.

## Các tính năng chính

- **Tính phản ứng** — sử dụng Vue `ref` và `computed` để cung cấp trạng thái thanh cuộn.
- **Phát hiện bất đồng bộ** — tự động tính toán chiều rộng khi khởi tạo.
- **Nhẹ nhàng** — dấu vết tối thiểu, hữu ích cho các tính toán bố cục phụ thuộc vào sự hiện diện của thanh cuộn.

## Cách sử dụng

```typescript
import { ScrollbarWidthRef } from '@dxtmisha/functional'

const scrollbar = new ScrollbarWidthRef()

// Kiểm tra xem chiều rộng đã được tính toán chưa
console.log(scrollbar.is.value)

// Lấy trạng thái (boolean)
console.log(scrollbar.item.value)
```

## Thuộc tính

- `item`: `Ref<boolean | undefined>` — Mục phản ứng chứa trạng thái hiện diện của thanh cuộn.
- `is`: `ComputedRef<boolean>` — Kiểm tra xem việc tính toán chiều rộng thanh cuộn đã hoàn tất hay chưa.
