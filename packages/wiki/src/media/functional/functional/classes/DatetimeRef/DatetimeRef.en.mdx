# DatetimeRef Class

A class for working with dates and times in a reactive way. It provides a wrapper around the `Datetime` utility, ensuring that date-related data stays synchronized with Vue properties and provides easy access to various date components.

## Key Features

- **Reactivity** — automatically updates the underlying date object when input properties change.
- **Localization support** — provides methods for language-sensitive date and time formatting.
- **Convenient accessors** — offers computed properties for getting year, month, day, hour, etc.
- **Standardized output** — includes methods for generating standardized date strings.

## Initialization

```typescript
import { DatetimeRef } from '@dxtmisha/functional'
import { ref } from 'vue'

const date = ref('2023-10-27')
const datetimeRef = new DatetimeRef(date, 'date', 'en-US')

console.log(datetimeRef.getYear().value) // 2023
```

## Date Information

- `getItem(): RefOrNormal<NumberOrStringOrDate>` — Returns the original input data for the date.
- `getDate(): Ref<Date>` — Returns the reactive `Date` object.
- `getDatetime(): Datetime` — Returns the underlying `Datetime` class instance.
- `getHoursType(): ComputedRef<GeoHours>` — Returns the hours format (12h or 24h).
- `getFirstDayCode(): ComputedRef<GeoFirstDay>` — Returns the code for the first day of the week.
- `getMaxDay(): ComputedRef<number>` — Returns the maximum number of days in the current month.

## Date Components

- `getYear(): ComputedRef<number>` — Returns the year.
- `getMonth(): ComputedRef<number>` — Returns the month (0-11).
- `getDay(): ComputedRef<number>` — Returns the day of the month.
- `getHour(): ComputedRef<number>` — Returns the hour.
- `getMinute(): ComputedRef<number>` — Returns the minutes.
- `getSecond(): ComputedRef<number>` — Returns the seconds.

## Formatting

- `locale(type?: GeoDate, styleOptions?: object): ComputedRef<string>` — Returns a language-sensitive formatted date string.
- `standard(timeZone?: boolean): ComputedRef<string>` — Returns a standardized date string (ISO-like).

## Internal Structure

`DatetimeRef` uses Vue's `watch` and `computed` systems to maintain synchronization between the input `Ref` and the internal `Datetime` instance. It triggers reactive updates whenever the date, locale code, or format type changes.
