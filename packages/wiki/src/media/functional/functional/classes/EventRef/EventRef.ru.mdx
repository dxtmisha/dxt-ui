# Класс EventRef

Реактивный менеджер прослушивания событий, который упрощает процесс прикрепления и открепления событий к элементам DOM или объекту window. Он автоматически обрабатывает перепривязку слушателей, когда целевой элемент (переданный как Vue `Ref`) изменяется.

## Ключевые особенности

- **Реактивность** — автоматически обновляет подписки на события при изменении `Ref` элемента или контрольного элемента.
- **Поддержка нескольких типов** — может прослушивать несколько типов событий одновременно.
- **Автоматическая очистка** — интегрируется с системой реактивности Vue для эффективного управления слушателями.
- **Контрольный элемент** — поддерживает дополнительный контрольный элемент для специальных проверок (например, проверки, находится ли элемент все еще в DOM).

## Инициализация

```typescript
import { EventRef } from '@dxtmisha/functional'
import { ref, onMounted } from 'vue'

const buttonRef = ref<HTMLButtonElement>()

const event = new EventRef(
  buttonRef,
  undefined,
  'click',
  (e) => console.log('Кнопка нажата!', e)
)

onMounted(() => event.start())
```

## Управление состоянием (Основные методы)

### `start`

Активирует прослушивание событий. Этот метод определяет все указанные типы событий и прикрепляет слушателей к целевому элементу. Если элемент является `Window` или `HTMLElement`, он также поддерживает специализированные события, такие как `resize` или `scroll-sync`.

**Возвращает:** `this`

### `stop`

Отключает прослушивание событий. Удаляет всех активных слушателей и отключает любые наблюдатели (например, `ResizeObserver`), связанные с экземпляром.

**Возвращает:** `this`

### `toggle`

Переключает активное состояние слушателя событий в зависимости от переданного логического значения.

**Параметры:**
- `activity: boolean` — `true` для начала прослушивания, `false` для остановки.

**Возвращает:** `this`

### `isActive`

Проверяет, активно ли в данный момент прослушивание событий и прикреплено ли оно к элементу.

**Возвращает:** `boolean`

## Настройка

- `setElement(elementSelector: ElementOrString): this` — Изменяет целевой элемент для слушателя событий.
- `setType(type: string | string[]): this` — Обновляет типы событий для прослушивания.
- `setListener(listener: function): this` — Обновляет функцию обратного вызова обработчика событий.
- `setOptions(options: EventOptions): this` — Изменение объекта options, который определяет характеристики объекта.
- `setDetail(detail?: D): this` — Изменение зависимого значения для метода dispatch.

## Утилитарные методы

- `dispatch(detail?: object): this` — Вручную вызывает события, связанные с экземпляром.
- `getElement(): E | undefined` — Возвращает текущий назначенный целевой элемент.
- `reset(): this` — Перезагружает события прослушивания.

## Внутреннее устройство

`EventRef` расширяет базовый класс `EventItem`. Он устанавливает наблюдатели (`watch`) Vue за предоставленными `elementSelector` и `elementSelectorControl`, чтобы гарантировать, что даже при динамическом изменении DOM-элемента слушатели событий будут правильно остановлены на старом элементе и запущены на новом.
