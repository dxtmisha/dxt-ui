# `useRouterList`

Managing a list of links for the router. Allows creating navigation menus based on data lists, automatically handling active states and transitions.

**Parameters:**
- `list: RefType<ConstrBind<T>[]>` — list of navigation items. Each item must contain `value` and `label`.
- `selected?: Ref<string> | string` — initially selected item or its value. If not specified, the first item in the list is selected.

**Returns:**
An object with reactive properties and control methods:
- `item` — current active item (computed property).
- `selected` — reference (`ref`) to the selected item's value.
- `label` — label of the current item.
- `list` — list of items, augmented with a `to` property for use with `router-link` or similar components.
- `to(name: string)` — method to navigate to an item by its value.
- `toMain()` — method to navigate to the first item in the list.

```typescript
import { ref } from 'vue'
import { useRouterList } from '@dxtmisha/functional'

const menuItems = ref([
  { value: 'dashboard', label: 'Dashboard' },
  { value: 'settings', label: 'Settings' }
])

// Initialize with 'dashboard' selected
const { list, selected, to } = useRouterList(menuItems, 'dashboard')

// The list will contain objects with a 'to' field, for example:
// { value: 'dashboard', label: 'Dashboard', to: { name: 'dashboard' } }

// Programmatic navigation
to('settings')
```
