# `useLoadingRef`

Composable для получения глобального статуса загрузки.

**Возвращает:**
- `ShallowRef<boolean>` — реактивная ссылка на текущий статус загрузки.

## Пример использования

```typescript
import { useLoadingRef } from '@dxtmisha/functional'

const isLoading = useLoadingRef()

// Используйте isLoading.value в вашем шаблоне или логике
```
