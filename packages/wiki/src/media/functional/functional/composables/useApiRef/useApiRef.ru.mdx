# `useApiRef`

Возвращает данные и методы для удобной работы с API запросами. Предоставляет реактивные состояния загрузки, кэширования и перезапуска.

**Параметры:**
- `path?: RefOrNormal<string | undefined>` — путь к запросу (может быть реактивным).
- `options?: ApiOptions` — данные для запроса (объект `ApiFetch` или строка с именем метода).
- `reactivity?: boolean` — включить ли реактивность: автоматический перезапуск запроса при изменении `path`, `options` или глобальных `conditions` (по умолчанию `true`).
- `conditions?: RefType<boolean>` — локальные условия выполнения запроса (если `false`, запрос не выполняется).
- `transformation?: (data: T) => ApiData<R>` — функция преобразования ответа от сервера перед записью в `data`.
- `unmounted?: boolean` — очищать ли данные из кэша (обнулять `data`) при размонтировании компонента.

**Возвращает (`UseApiRef<R>`):**
Объект с реактивными свойствами и методами управления:
- `data` — ссылка (`ref`) на загруженные данные (объект `ApiData<R>`) или `undefined`, если загрузка еще не завершена.
- `isStarting` — флаг начала запроса (`true` если еще нет данных).
- `loading` — текущее состояние загрузки запроса (`true` пока идет ожидание ответа).
- `reading` — флаг активного чтения.
- `reset()` — метод для перезагрузки данных.
- `stop()` — метод для остановки слежения за запросом.
- `abort()` — метод для быстрой отмены выполняемого запроса через `AbortController`.

**Глобальные настройки:**
Вы можете определить глобальные условия для всех `useApiRef` с помощью функции `setApiRefGlobalConditions(conditions)`.

### Когда выполняется запрос? (Lazy Loading)
Особенностью `useApiRef` является ленивая загрузка (Lazy Loading). При вызове функции сетевой запрос **не отправляется мгновенно**. 

Запрос будет отправлен только в тот момент, когда вы **впервые обратитесь к свойству `data`** (например, запросите `data.value` или выведете его в Vue-шаблон). Это позволяет использовать композицию на верхнем уровне сетапа без ненужных обращений к сети, пока данные реально не понадобятся для рендеринга.

```typescript
import { ref } from 'vue'
import { useApiRef } from '@dxtmisha/functional'

const userId = ref('123')

const {
  data: user,
  loading,
  reset,
  stop
} = useApiRef(
  () => `/api/users/${userId.value}`,
  { method: 'GET' },
  true // Реактивно обновлять данные при изменении userId
)

// Принудительная перезагрузка
const refreshData = async () => {
  await reset()
}

const cancelRequest = () => {
  abort() // Быстро отменит запрос по сети
}

// Остановка слежения (например, при уходе со страницы)
const abortRequest = () => {
  stop() // Остановит watcher и очистит данные
}
```
