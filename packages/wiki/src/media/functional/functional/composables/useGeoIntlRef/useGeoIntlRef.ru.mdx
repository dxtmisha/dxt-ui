# `useGeoIntlRef`

Composable, возвращающий экземпляр класса `GeoIntlRef` для работы с интернационализацией и форматированием данных (чисел, валют, дат и т.д.) на основе текущей локали.

**Возвращает (`GeoIntlRef`):**
- Объект класса `GeoIntlRef`, предоставляющий методы для реактивного форматирования.

## Методы `GeoIntlRef`

Класс предоставляет реактивные методы для форматирования. Все доступные методы этого класса можно найти в описании GeoIntlRef.

## Пример

```typescript
import { useGeoIntlRef } from '@dxtmisha/functional'

const intl = useGeoIntlRef()

// Реактивное форматирование числа
const formattedNumber = intl.number(1234.56) // "1 234,56" (для ru-RU)

// Форматирование валюты
const price = intl.currency(1000, 'RUB') // "1 000 ₽"
```
