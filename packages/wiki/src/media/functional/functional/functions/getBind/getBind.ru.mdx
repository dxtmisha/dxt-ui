# `getBind`

Утилитарная функция для генерации объекта свойств для подкомпонента на основе обычных (нереактивных) значений. Она обрабатывает различные типы входных данных и обеспечивает согласованную структуру для привязки свойств. Это нереактивный аналог `getBindRef`.

**Параметры:**
- `value: T | R | undefined | null` — Входное значение для привязки. Может быть примитивом или объектом.
- `nameExtra: ItemList | string = {}` — Дополнительные свойства для объединения или имя свойства, если `value` не является объектом.
- `name: string = 'value'` — Имя свойства по умолчанию (по умолчанию `'value'`).
- `except: boolean = false` — Если true, пропускает проверку на наличие первичного ключа в объекте `value`.

**Возвращает:**
`ConstrBind<R>` — Объект, содержащий свойства, готовые к привязке.

```typescript
import { getBind } from '@dxtmisha/functional'

// Вариант 1: Простое значение
const bind1 = getBind('Привет', {}, 'label') 
// Результат: { label: 'Привет' }

// Вариант 2: Объект с дополнительными свойствами
const bind2 = getBind({ id: 1 }, { class: 'active' }, 'data')
// Результат: { id: 1, class: 'active' }
```
