# `render`

Обертка над функцией Vue `h` (hyperscript), которая обеспечивает автоматическую генерацию ключей для компонентов. Она гарантирует, что компоненты имеют уникальные ключи для эффективного рендеринга и обработки переходов, особенно при работе со списками или динамическими элементами.

**Параметры:**
- `name: string` — Имя или тег компонента для рендеринга.
- `props?: T` — Свойства, передаваемые компоненту.
- `children?: RawChildren | RawSlots` — Дочерние элементы или слоты компонента.
- `index?: string` — Необязательный индекс или идентификатор, используемый для генерации ключа.

**Возвращает:**
`VNode` — Отрендеренный виртуальный узел.

```typescript
import { render } from '@dxtmisha/functional'

// Рендерит div с автоматически сгенерированным ключом
const vnode = render('div', { class: 'container' }, 'Привет, мир')

// Рендерит компонент с определенным индексом для генерации ключа
const item = render('my-component', { data: itemData }, null, 'item-1')
```
