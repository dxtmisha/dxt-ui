# `toRefItem`

Утилитарная функция, которая гарантирует, что значение является реактивной переменной `Ref`. Если входное значение уже является `Ref`, оно возвращается как есть. Если входное значение является обычным значением, оно оборачивается в новую переменную `Ref`. Это полезно для функций, которые ожидают реактивную ссылку в качестве аргумента.

**Параметры:**
- `item: RefOrNormal<T>` — Реактивная переменная (`Ref`) или обычное значение.

**Возвращает:**
`Ref<T>` — Реактивная переменная `Ref`, содержащая значение.

```typescript
import { ref } from 'vue'
import { toRefItem } from '@dxtmisha/functional'

const existingRef = ref(10)
const ordinaryValue = 20

const normalized1 = toRefItem(existingRef)   // Возвращает исходный ref
const normalized2 = toRefItem(ordinaryValue) // Возвращает новый ref(20)

console.log(normalized1.value) // 10
console.log(normalized2.value) // 20
```
