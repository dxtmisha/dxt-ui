# `getComputedAsync`

Оборачивает асинхронную функцию обратного вызова в реактивное вычисляемое свойство. Оно автоматически обрабатывает разрешение промиса и позволяет использовать необязательное значение `ignore` для фильтрации нежелательных результатов.

**Параметры:**
- `callback: () => Promise<Ref<T | I> | T | I>` — Асинхронная функция, возвращающая значение для вычисления.
- `ignore?: I` — Значение, которое, если оно возвращено функцией обратного вызова, будет проигнорировано, и вычисляемое свойство не будет обновлено.

**Возвращает:**
`ComputedRef<T | undefined>` — Реактивное вычисляемое свойство, содержащее разрешенное значение из функции обратного вызова, или `undefined` до тех пор, пока не будет разрешено первое допустимое значение.

```typescript
import { getComputedAsync } from '@dxtmisha/functional'

// Асинхронно получает данные и обновляется при их получении
const userData = getComputedAsync(async () => {
  const response = await fetch('/api/user')
  return response.json()
})

// Использование параметра ignore
const data = getComputedAsync(async () => {
  const result = await someAsyncAction()
  return result === null ? 'ignored' : result
}, 'ignored')
```
