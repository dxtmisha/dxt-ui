# `getRandomText`

Генерирует строку, состоящую из случайного количества «слов». Под словом понимается последовательность одинаковых символов заданной длины. Функция полезна для заполнения интерфейса заглушками (placeholder), похожими на текст, но не имеющими смысла или языковой привязки.

Она работает на основе утилит `random` (для выбора случайных чисел) и `strFill` (для заполнения слова символами).

**Параметры:**
- `min: number` — Минимальное количество слов в итоговой строке.
- `max: number` — Максимальное количество слов в итоговой строке.
- `symbol: string` — (Необязательно) Символ, из которого состоят сгенерированные слова. По умолчанию `#`.
- `lengthMin: number` — (Необязательно) Минимальная длина одного слова. По умолчанию `2`.
- `lengthMax: number` — (Необязательно) Максимальная длина одного слова. По умолчанию `12`.

**Возвращает:**
`string` — Строку, состоящую из слов, разделённых пробелами.

```typescript
import { getRandomText } from '@dxtmisha/functional-basic'

// Сгенерирует от 2 до 5 слов, состоящих из символа '#'
// Длина каждого слова от 2 до 12 символов
console.log(getRandomText(2, 5)) // '#### ## #######'

// Генерация точного числа слов (3) из определенного символа ('*')
// Длина слов строго по 5 символов
console.log(getRandomText(3, 3, '*', 5, 5)) // '***** ***** *****'
```
