# `toDate`

Chuyển đổi giá trị đầu vào (ngày, số hoặc chuỗi) thành đối tượng `Date`. Hỗ trợ nhiều định dạng chuỗi khác nhau, bao gồm các chuỗi giống ISO không có dấu phân cách, cũng như tự động thay thế múi giờ.

**Tham số:**
- `value?: T` — Giá trị đầu vào (đối tượng `Date`, dấu thời gian tính bằng mili giây hoặc chuỗi ngày tháng).

**Trả về:**
`Date` — Đối tượng ngày tháng kết quả. Nếu giá trị không được cung cấp hoặc là `null`, ngày hiện tại sẽ được trả về.

```typescript
import { toDate } from '@dxtmisha/functional-basic'

console.log(toDate('20231027')) // '2023-10-27T00:00:00.000Z'
console.log(toDate(1672531200000)) // '2023-01-01T00:00:00.000Z'
console.log(toDate('12:00')) // '2000-01-01T12:00:00.000Z'
```
