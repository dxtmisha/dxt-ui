# `toDate`

Преобразует входное значение (дату, число или строку) в объект `Date`. Поддерживает различные строковые форматы, включая ISO-подобные строки без разделителей, а также автоматическую подстановку часового пояса.

**Параметры:**
- `value?: T` — Входное значение (объект `Date`, метка времени в миллисекундах или строка даты).

**Возвращает:**
`Date` — Полученный объект даты. Если значение не передано или равно `null`, возвращается текущая дата.

```typescript
import { toDate } from '@dxtmisha/functional-basic'

console.log(toDate('20231027')) // '2023-10-27T00:00:00.000Z'
console.log(toDate(1672531200000)) // '2023-01-01T00:00:00.000Z'
console.log(toDate('12:00')) // '2000-01-01T12:00:00.000Z'
```
