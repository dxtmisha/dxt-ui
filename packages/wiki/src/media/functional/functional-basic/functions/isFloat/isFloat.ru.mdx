# `isFloat`

Проверяет, является ли значение числом или дробным числом.

**Параметры:**
- `value: any` — значения для проверки.

**Возвращает:**
`boolean` — `true`, если значение является числом или строковым представлением числа, иначе `false`.

```typescript
import { isFloat } from '@dxtmisha/functional-basic'

console.log(isFloat(123)) // true
console.log(isFloat('123.45')) // true
console.log(isFloat('abc')) // false
```
