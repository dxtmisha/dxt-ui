# `removeCommonPrefix`

Xóa tiền tố chung khỏi phần đầu của chuỗi chính. Nếu chuỗi chính bắt đầu bằng tiền tố được chỉ định, hàm sẽ cắt bỏ nó và trả về chuỗi đã được loại bỏ khoảng trắng dư thừa ở hai đầu.

**Tham số:**
- `mainStr: string` — Chuỗi chính.
- `prefix: string` — Chuỗi tiền tố cần xóa.

**Trả về:**
`string` — Một chuỗi mới không có tiền tố và đã được loại bỏ khoảng trắng thừa.

```typescript
import { removeCommonPrefix } from '@dxtmisha/functional-basic'

console.log(removeCommonPrefix('Xin chào, thế giới!', 'Xin chào, ')) // 'thế giới!'
console.log(removeCommonPrefix('user_name', 'user_')) // 'name'
```
