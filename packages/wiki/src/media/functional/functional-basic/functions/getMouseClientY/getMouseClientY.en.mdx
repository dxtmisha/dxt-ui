# `getMouseClientY`

Extracts the Y coordinate (vertical position) of the pointer (mouse or touch) relative to the visible area of the browser window (viewport) from the event object.

The function processes both standard mouse events (`MouseEvent`) and touch events (`TouchEvent`), giving priority to the `clientY` property. If `clientY` is not found directly on the event, it searches within the `targetTouches` and `touches` arrays.

**Parameters:**
- `event: MouseEvent & TouchEvent` — The event object (`mousemove`, `touchmove`, `click`, etc.).

**Returns:**
`number` — The Y coordinate of the cursor/touch. If not found, it returns `0`.

```typescript
import { getMouseClientY } from '@dxtmisha/functional-basic'

document.addEventListener('click', (event) => {
  const y = getMouseClientY(event)
  console.log(`The user clicked ${y}px away from the top edge of the window`)
})
```
