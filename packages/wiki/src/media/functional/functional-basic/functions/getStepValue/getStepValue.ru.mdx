# `getStepValue`

Вычисляет абсолютное значение одного процента (`1%`) от общего диапазона (разницы между минимальным и максимальным значениями). Функция полезна для определения, насколько изменить значение слайдера или прогресс-бара при сдвиге на 1%.

**Параметры:**
- `min: number | undefined` — Минимальное значение диапазона. Если передано `undefined`, будет использован `0`.
- `max: number` — Максимальное значение диапазона.

**Возвращает:**
`number` — Значение, равное 1% от длины диапазона. Если максимальное значение меньше или равно минимальному, функция вернет `0`.

```typescript
import { getStepValue } from '@dxtmisha/functional-basic'

// Диапазон от 0 до 500, значит 1% это 5 ед.
console.log(getStepValue(0, 500)) // 5

// Диапазон от 50 до 150 (разница 100), значит 1% это 1 ед.
console.log(getStepValue(50, 150)) // 1
```
