# `getCurrentTime`

Returns the current time in milliseconds. The function is a wrapper around `new Date().getTime()` and allows you to quickly get a timestamp.

**Returns:**
Current time in milliseconds (number).

```typescript
import { getCurrentTime } from '@dxtmisha/functional-basic'

const time = getCurrentTime()
// Result: 1710498600000 (example)
```
