# `getCurrentTime`

Trả về thời gian hiện tại tính bằng mili giây. Hàm này là một wrapper bao quanh `new Date().getTime()` và cho phép bạn nhanh chóng lấy dấu thời gian (timestamp).

**Trả về:**
Thời gian hiện tại tính bằng mili giây (number).

```typescript
import { getCurrentTime } from '@dxtmisha/functional-basic'

const time = getCurrentTime()
// Kết quả: 1710498600000 (ví dụ)
```
