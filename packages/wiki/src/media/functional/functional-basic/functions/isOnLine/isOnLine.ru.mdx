# `isOnLine`

Простая утилита для определения текущего статуса подключения устройства к сети.

Функция проверяет свойство `navigator.onLine`. Если среда выполнения не является браузером (например, во время серверного рендеринга — SSR), по умолчанию возвращается `true`, чтобы сетезависимая логика не блокировалась преждевременно.

**Возвращает:**
`boolean` — Возвращает `true`, если устройство в сети или если среда выполнения не браузер. Возвращает `false`, если устройство явно находится в автономном режиме (offline).

```typescript
import { isOnLine } from '@dxtmisha/functional-basic'

if (isOnLine()) {
  console.log('Устройство в сети!')
} else {
  console.log('Устройство не в сети.')
}
```
