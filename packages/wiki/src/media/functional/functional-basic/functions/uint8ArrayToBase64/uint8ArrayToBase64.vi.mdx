# `uint8ArrayToBase64`

Chuyển đổi một mảng dữ liệu nhị phân `Uint8Array` thành một chuỗi được mã hóa Base64. Hỗ trợ cả môi trường trình duyệt (thông qua `btoa`) và môi trường Node.js (thông qua `Buffer`).

**Tham số:**
- `bytes: Uint8Array` — Dữ liệu nhị phân.

**Trả về:**
`string` — Chuỗi định dạng Base64.

```typescript
import { uint8ArrayToBase64 } from '@dxtmisha/functional-basic'

const data = new Uint8Array([72, 101, 108, 108, 111])
console.log(uint8ArrayToBase64(data)) // 'SGVsbG8='
```
