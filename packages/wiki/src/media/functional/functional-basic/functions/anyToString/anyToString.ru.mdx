# `anyToString`

Утилита, которая безопасно приводит любые входящие данные к строковому типу. Это полезно при выводе данных в интерфейс, когда заранее неизвестен тип переменной (объект, массив, boolean или null).

**Параметры:**
- `value: V` — Значение для преобразования.
- `isArrayString: boolean = true` — Если `true`, массивы примитивов объединяются через запятую. Если `false`, преобразуются в JSON.
- `trim: boolean = true` — Если `true`, обрезает лишние пробелы (только для строк).

**Возвращает:**
`string` — Строковое представление значения.

<Source
  language='typescript'
  code={`
import { anyToString } from '@dxtmisha/functional-basic'

// Строки
anyToString('  Hello  ')               // 'Hello'
anyToString('  Hello  ', true, false)  // '  Hello  '

// Массивы
anyToString([1, 2, 3])                 // '1,2,3'
anyToString([1, 2, 3], false)          // '[1,2,3]'

// Объекты
anyToString({ a: 1 })                  // '{"a":1}'

// Булевы значения (преобразование в 1/0)
anyToString(true)                      // '1'
anyToString(false)                     // '0'

// Другие типы
anyToString(123)                       // '123'
anyToString(null)                      // ''
anyToString(undefined)                 // ''
anyToString(Symbol('test'))            // 'Symbol(test)'
  `}
/>
