# `anyToString`

Преобразует переданное значение в строку.
- Строки обрезаются по краям от пробелов.
- Массивы, не содержащие объектов внутри себя, объединяются через запятую.
- Объекты и массивы, содержащие объекты, преобразуются в JSON-строку.
- Истинное булево значение (`true`) преобразуется в `'1'`, а ложное (`false`) — в `'0'`.
- Для `null` или `undefined` возвращается пустая строка `''`.
- В остальных случаях возвращается результат выполнения метода `toString()`.

**Параметры:**
- `value: V` — Значения для преобразования.

**Возвращает:**
`string` — Строковое представление переданного значения.

```typescript
import { anyToString } from '@dxtmisha/functional-basic'

console.log(anyToString(' test ')) // 'test'
console.log(anyToString([1, 2, 3])) // '1,2,3'
console.log(anyToString({ a: 1 })) // '{"a":1}'
console.log(anyToString(true)) // '1'
console.log(anyToString(false)) // '0'
console.log(anyToString(null)) // ''
```
