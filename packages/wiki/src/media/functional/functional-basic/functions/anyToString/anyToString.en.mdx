# `anyToString`

A utility that safely converts any input data into a string type. This is useful when outputting data to the interface when the variable type (object, array, boolean, or null) is not known in advance.

**Parameters:**
- `value: V` — The value for conversion.
- `isArrayString: boolean = true` — If `true`, primitive arrays are joined by commas. If `false`, they are stringified as JSON.
- `trim: boolean = true` — If `true`, trims leading and trailing whitespace (only for strings).

**Returns:**
`string` — The string representation of the value.

<Source
  language='typescript'
  code={`
import { anyToString } from '@dxtmisha/functional-basic'

// Strings
anyToString('  Hello  ')               // 'Hello'
anyToString('  Hello  ', true, false)  // '  Hello  '

// Arrays
anyToString([1, 2, 3])                 // '1,2,3'
anyToString([1, 2, 3], false)          // '[1,2,3]'

// Objects
anyToString({ a: 1 })                  // '{"a":1}'

// Boolean values (legacy-compatible 1/0 conversion)
anyToString(true)                      // '1'
anyToString(false)                     // '0'

// Other types
anyToString(123)                       // '123'
anyToString(null)                      // ''
anyToString(undefined)                 // ''
anyToString(Symbol('test'))            // 'Symbol(test)'
  `}
/>
