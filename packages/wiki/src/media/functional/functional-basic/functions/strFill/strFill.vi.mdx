# `strFill`

Phương thức tạo một chuỗi có độ dài count, bao gồm các ký tự value.

**Tham số:**
- `value: string` — ký tự để điền.
- `count: number` — độ dài của chuỗi.

**Trả về:**
`string` — một chuỗi được điền với giá trị đã chỉ định.

```typescript
import { strFill } from '@dxtmisha/functional-basic'

const string = strFill('a', 3)
console.log(string) // 'aaa'
```
