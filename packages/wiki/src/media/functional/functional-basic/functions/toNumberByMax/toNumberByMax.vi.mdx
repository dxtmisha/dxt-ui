# `toNumberByMax`

Chuyển đổi giá trị đầu vào thành một số và nếu giá trị đó vượt quá mức tối đa được chỉ định, hàm sẽ trả về một chuỗi có ký hiệu `+` ở cuối. Hàm cũng hỗ trợ thực hiện định dạng cục bộ cho kết quả.

**Tham số:**
- `value: string | number` — Giá trị đầu vào.
- `max?: string | number` — Giá trị tối đa cho phép.
- `formatting?: boolean` — Cờ bật định dạng thông qua `GeoIntl`.
- `language?: string` — Mã ngôn ngữ để định dạng (ví dụ: `'vi-VN'`).

**Trả về:**
`string | number` — Một chuỗi đã được định dạng (ví dụ: `'99+'`) hoặc một số.

```typescript
import { toNumberByMax } from '@dxtmisha/functional-basic'

console.log(toNumberByMax(150, 99)) // '99+'
console.log(toNumberByMax(50, 99)) // 50
console.log(toNumberByMax(1000, 500, true, 'vi-VN')) // '500+'
```
