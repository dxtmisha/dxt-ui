# `isShare`

Проверяет, поддерживается ли Web Share API в текущей среде.

**Возвращает:**
`boolean` — `true`, если Web Share API поддерживается, иначе `false`.

```typescript
import { isShare } from '@dxtmisha/functional-basic'

if (isShare()) {
  console.log('Обмен данными поддерживается!')
}
```
