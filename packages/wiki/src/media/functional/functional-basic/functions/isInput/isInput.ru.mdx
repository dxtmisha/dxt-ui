# `isInput`

Проверяет, является ли элемент полем ввода (`input`, `textarea`, `select`) или элементом с атрибутом `contenteditable`.

**Параметры:**
- `element: HTMLElement | EventTarget | null` — проверяемый элемент.

**Возвращает:**
`boolean` — `true`, если элемент является полем ввода или редактируемым, иначе `false`.

```typescript
import { isInput } from '@dxtmisha/functional-basic'

const input = document.createElement('input')
isInput(input) // true

const div = document.createElement('div')
isInput(div) // false
```
