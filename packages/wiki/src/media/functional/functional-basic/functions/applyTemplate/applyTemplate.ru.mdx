# `applyTemplate`

Применяет шаблон к тексту, подставляя значения из переданного объекта или массива вместо специальных маркеров (ключей).

Функция поддерживает два вида шаблонов:
1. **Позиционные маркеры (`%d`, `%e`, `%f` и т.д.)** — заменяются на значения из переданного массива (или объекта) по порядку.
2. **Именованные маркеры в квадратных или фигурных скобках (`[key]` или `{key}`)** — заменяются на соответствующие значения по ключу из объекта.
   Также поддерживаются парные теги вида `[key]контент[/key]` или `{key}контент{/key}`. При их использовании функция подставит содержимое между тегами вместо слова `[content]` или `{content}` внутри переданного значения.

**Параметры:**
- `text: string` — Исходный текст с шаблоном.
- `replacement: Record<string, string | number> | string[]` — Объект или массив со значениями для подстановки. По умолчанию `{}`.

**Возвращает:**
`string` — Текст с подставленными значениями.

```typescript
import { applyTemplate } from '@dxtmisha/functional-basic'

// Позиционные маркеры
console.log(applyTemplate('Привет, %d %e!', ['Иван', 'Иванов'])) // 'Привет, Иван Иванов!'

// Именованные маркеры
console.log(applyTemplate('Привет, [name]!', { name: 'Иван' })) // 'Привет, Иван!'
console.log(applyTemplate('Привет, {name}!', { name: 'Иван' })) // 'Привет, Иван!'

// Обёртка с парными тегами и контентом
console.log(applyTemplate('Нажми [link]здесь[/link]', { link: '<a href="#">[content]</a>' }))
console.log(applyTemplate('Нажми {link}здесь{/link}', { link: '<a href="#">{content}</a>' }))
// 'Нажми <a href="#">здесь</a>'
```
