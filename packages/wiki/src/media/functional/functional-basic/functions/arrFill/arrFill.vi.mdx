# `arrFill`

Phương thức tạo một mảng gồm "count" phần tử với các giá trị bằng `value`.

**Tham số:**
- `value: T` — giá trị để điền vào mảng.
- `count: number` — số lượng phần tử trong mảng đó.

**Trả về:**
`T[]` — một mảng được điền với giá trị đã chỉ định.

```typescript
import { arrFill } from '@dxtmisha/functional-basic'

const array = arrFill('a', 3)
console.log(array) // ['a', 'a', 'a']
```
