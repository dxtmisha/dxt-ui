# `arrFill`

Формирует и возвращает массив заданного размера, заполненный одинаковыми значениями.

Это полезно для быстрого создания заглушек (скелетонов), инициализации начальных состояний или списков по умолчанию.

**Параметры:**
- `value: T` — значение, заполняющее массив.
- `count: number` — число элементов этого массива.

**Возвращает:**
`T[]` — массив, заполненный указанным значением.

```typescript
import { arrFill } from '@dxtmisha/functional-basic'

const array = arrFill('a', 3)
console.log(array) // ['a', 'a', 'a']
```
