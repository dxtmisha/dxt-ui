# `copyObject`

Создает глубокую копию простого объекта с использованием `JSON.parse(JSON.stringify(value))`.

> Обратите внимание, что этот метод подходит только для простых объектов и массивов, содержащих сериализуемые данные. Он не скопирует методы (функции), значения `undefined`, символы, и не сможет обработать циклические ссылки.

**Параметры:**
- `value: T` — Объект или значение, которое необходимо скопировать.

**Возвращает:**
`T` — Новая копия объекта.

```typescript
import { copyObject } from '@dxtmisha/functional-basic'

const original = { name: 'John', details: { age: 30 } }
const copy = copyObject(original)

copy.details.age = 31

console.log(original.details.age) // 30 (исходный объект не изменился)
console.log(copy.details.age) // 31
```
