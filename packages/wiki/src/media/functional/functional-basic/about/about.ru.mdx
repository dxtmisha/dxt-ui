# @dxtmisha/functional-basic

`@dxtmisha/functional-basic` — базовая библиотека утилит для современной веб-разработки. Она не привязана ни к одному фреймворку и может использоваться в любом TypeScript-проекте, будь то Vue, React, Nuxt или чистый браузерный код.

## Зачем нужна эта библиотека?

В любом веб-приложении неизбежно появляется один и тот же набор задач: HTTP-запросы, работа с локализацией, даты, cookie, кеш, состояние загрузки. Обычно каждый проект решает их по-своему, накапливая разрозненные утилиты, которые разбросаны по папкам, дублируются между проектами и со временем устаревают.

`functional-basic` собирает все эти решения в одном месте — продуманном, протестированном и готовом к использованию. Это не мега-фреймворк с жёсткими соглашениями, а именно библиотека: подключаете только то, что нужно, и используете так, как удобно вам.

## Что она умеет?

Библиотека охватывает широкий спектр задач, с которыми сталкивается разработчик ежедневно.

Для работы с **сетью** — есть полнофункциональный HTTP-клиент с поддержкой хуков подготовки и завершения запроса, кешированием ответов, управлением заголовками, эмуляцией ответов для разработки и интегрированным индикатором загрузки. Запросы можно прерывать, настраивать, расширять — без лишних зависимостей.

Для **геолокации и интернационализации** — автоматическое определение страны и языка пользователя, форматирование чисел, дат, валют и единиц измерения по локальным стандартам, склонения, флаги стран (200+ государств), телефонные коды и маски ввода. Всё строится поверх нативного `Intl` API, что гарантирует корректность на любом рынке.

Для **управления состоянием браузера** — удобные обёртки над `localStorage`, `sessionStorage`, `cookie` и URL hash. Всё с типизацией и возможностью подписки на изменения. Данные сохраняются между сессиями без лишнего кода с вашей стороны.

Для **SEO и мета-тегов** — единый класс, синхронизирующий `<title>`, Open Graph и Twitter Card одновременно. Один вызов `setTitle()` обновляет сразу заголовок вкладки, превью в соцсетях и карточку в Twitter. Подходит как для браузера, так и для SSR.

Для **работы с данными** — богатый набор функций: глубокое копирование объектов, работа с массивами, безопасное преобразование типов, валидация значений, форматирование строк и шаблоны. Всё написано с расчётом на реальные граничные случаи, а не только «счастливый путь».

Для **DOM** — создание элементов, работа с атрибутами, прокрутка, поиск, изображения. Небольшие, но незаменимые инструменты, которые экономят время при работе с браузерным окружением.

## Для кого это?

Библиотека ориентирована на разработчиков, которые хотят избежать дублирования кода и не изобретать велосипед. Она одинаково хорошо подходит как для небольших лендингов, так и для крупных корпоративных приложений с многоязычным интерфейсом.

Она комфортно встраивается в экосистему `@dxtmisha`, но абсолютно не требует её — подключить и использовать отдельные классы или функции можно в абсолютно любом проекте. Если вам нужен только форматировщик дат или HTTP-клиент — берёте именно его, всё остальное не тянется автоматически.

## Принципы

- **Без внешних зависимостей** — библиотека не тянет за собой сторонние пакеты. Меньше конфликтов, меньше уязвимостей, меньше неожиданностей при обновлениях.
- **Только TypeScript** — строгая типизация и автодополнение из коробки. Типы не являются «дополнением» — они неотъемлемая часть API.
- **Удобный API** — методы цепочек, предсказуемое поведение, понятные имена. Не нужно изучать документацию, чтобы понять, что делает `isFilled` или `setTitle`.
- **Покрытие тестами** — каждый класс и функция сопровождаются тестами. Изменения в библиотеке не сломают ваш проект незаметно.
