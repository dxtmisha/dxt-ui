# Класс TranslateFile

Статический служебный класс для управления данными переводов из внешних файлов. Поддерживает ленивую загрузку (lazy loading) пакетов перевода, автоматическое сопоставление локалей и внутреннее кэширование для оптимизации производительности.

## Ключевые особенности

- **Ленивая загрузка** — поддерживает асинхронную загрузку файлов перевода только тогда, когда они необходимы для текущей локали.
- **Умное кэширование** — автоматически кэширует содержимое загруженных файлов, предотвращая повторное чтение файлов или лишние сетевые запросы.
- **Приоритизация локалей** — интеллектуально выбирает наиболее подходящий файл на основе местоположения (Местоположение > Язык > Глобальное).
- **Бесшовная интеграция** — работает в связке с классом `Translate`, обеспечивая единую систему переводов.

## Инициализация

Класс является статическим и настраивается путем добавления загрузчиков файлов для различных локалей.

```javascript
import { TranslateFile } from '@dxtmisha/functional-basic'

// Регистрация загрузчиков файлов (обычно через динамические импорты)
TranslateFile.add({
  'en-US': () => import('./locales/en-US.json'),
  'en': () => import('./locales/en.json'),
  'global': () => import('./locales/global.json')
})
```

## Методы

### Получение данных

- `async getList(): Promise<TranslateDataFileList | undefined>` — Возвращает список переводов для текущего местоположения. Сначала проверяет внутренний кэш, затем пытается загрузить данные через зарегистрированные загрузчики.
- `isFile(): boolean` — Проверяет, есть ли зарегистрированные загрузчики файлов, применимые к текущему местоположению или языку.
- `add(data: TranslateDataFile): void` — Добавляет сопоставление локалей и соответствующих им функций-загрузчиков (функций, возвращающих Promise).

## Типы

### `TranslateDataFile`
Коллекция асинхронных функций-загрузчиков, индексированных строками локали/региона:
- **Ключ**: Код локали (например, `'en-US'`, `'ru'`, `'global'`).
- **Значение**: Функция, возвращающая `Promise` с `TranslateDataFileList`.

### `TranslateDataFileList`
Простой объект перевода (ключ-значение): `Record<string, string>`.

## Примеры

### Регистрация файлов с динамическим импортом

```javascript
import { TranslateFile, Translate } from '@dxtmisha/functional-basic'

TranslateFile.add({
  'en-US': () => import('./en-US.json'),
  'ru-RU': () => import('./ru-RU.json')
})

// Класс Translate будет автоматически использовать TranslateFile, если файлы доступны
const title = await Translate.get('app.title')
```
