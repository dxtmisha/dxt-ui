# Translate Class

A static utility class for managing translations and localized text. It provides powerful tools for retrieving, caching, and displaying content with template support, automatic locale detection, and batch request loading for optimized performance.

## Key Features

- **Batch Loading** — Intelligently groups translation requests, sending them as a single pack 160ms after being queued.
- **Flexible Templates** — Supports parameter replacement in text using both indices (`[0]`) and named keys (`{key}`).
- **Sync and Async Access** — Methods for instant text retrieval from cache (`getSync`) or guaranteed server loading (`get`).
- **Smart Cache** — Stores translations scoped to the current locale and supports fallbacks (language-country → language → global).
- **Flexible Data Sources** — Supports loading translations from both the server (API) and local files (via `TranslateFile`).
- **Local Development** — The `addNormalOrSync` mode automatically switches between mocks and real requests based on the environment.

## Initialization

The class is static and does not require instantiation. The primary configuration consists of setting the API URL.

```javascript
import { Translate } from '@dxtmisha/functional-basic'

// Configure API URL for requests
Translate.setUrl('/api/v1/translate')

// Preload necessary texts
await Translate.add(['welcome', 'auth.login', 'errors.required'])
```

## Methods

### Translation Retrieval

- `async get(name: string, replacement?: any): Promise<string>` — Asynchronously retrieves text. If not in cache and not on localhost, it attempts to load it from the server.
- `getSync(name: string, first?: boolean, replacement?: any): string` — Returns text from the cache. If `first` is `true`, returns a space ` ` if the text is missing; otherwise, returns the code `name` itself.
- `async getList(names: TranslateCode[]): Promise<TranslateList>` — Retrieves an object with a group of translations (asynchronous).
- `getListSync(names: TranslateCode[], first?: boolean): TranslateList` — Retrieves an object with a group of translations from the cache (synchronous).

### Data Management

- `async add(names: string | string[]): Promise<void>` — Adds codes to the loading queue. Requests are batched to minimize network load.
- `addSync(data: Record<string, string>): void` — Directly adds translated strings to the cache.
- `async addNormalOrSync(data: Record<string, string>): Promise<void>` — Intelligent addition: behaves as `addSync` on localhost, and as `add` otherwise.
- `addSyncByLocation(data: Record<string, Record<string, string>>): void` — Bulk-adds texts tied to specific locales.
- `addSyncByFile(data: TranslateDataFile): void` — Registers translation file loaders for lazy loading (see `TranslateFile`).

### Configuration

- `setUrl(url: string): typeof Translate` — Sets the API path for loading translations.
- `setPropsName(name: string): typeof Translate` — Sets the property name in the request body (default is `list`).
- `setReadApi(read: boolean): typeof Translate` — Enables or disables API requests for translations (default is `true`).

## Types

### `TranslateCode`
A string (code name) or an array of strings. Arrays are used for template replacement:
- `string` — e.g., `'welcome'`
- `string[]` — e.g., `['welcome', 'Guest']`

### `TranslateList`
A record where keys are translation codes and values are the resulting translated strings.

### `TranslateDataFile`
A mapping of locale strings to their respective loader functions. Used in `addSyncByFile`.

## Examples

### Working with Templates
```javascript
// In database: "Hello, [0]! You have {count} notifications."
const text = await Translate.get('user.notifications', ['John', { count: 5 }])
// Result: "Hello, John! You have 5 notifications."
```

### Synchronous Usage in Components
```javascript
// First, load everything needed (e.g., during app initialization)
await Translate.add(['btn.save', 'btn.cancel'])

// Inside a component, use synchronously to avoid excessive await usage
const label = Translate.getSync('btn.save')
```

### Bulk List Retrieval
```javascript
const labels = await Translate.getList([
  'home.title',
  ['home.welcome', 'Guest'], // with immediate template replacement
  'home.footer'
])

console.log(labels['home.welcome']) // "Welcome, Guest!"
```
