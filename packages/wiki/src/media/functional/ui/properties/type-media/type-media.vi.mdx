# Tính thích ứng

Type `media` và `container` sử dụng Responsive Breakpoints bằng thông số logic tùy chỉnh JSON.

## Tương tác Media

Thay vì ghi `@media` thủ công trên tệp CSS, bạn nên chèn giá trị thuộc `media-max` vào logic.
```json
{
  "grid": {
    "mobile": {
      "_type": "media-max",
      "value": "768px",
      "columns": { "value": "1fr" }
    }
  }
}
```
**SCSS Output:**
```scss
@media (max-width: 768px) {
  .grid {
    columns: 1fr;
  }
}
```

## Container Queries

Thay cho đo lường thiết bị gốc, Component Responsive sẽ xài chuẩn `container-max` bằng khung nội hàm component (`@container`) thích ứng chuẩn giao tiếp thông qua khung đo khu vực bọc bên ngoài.
