# Adaptability

The `media` and `container` types in `properties.json` seamlessly map breakpoints rendering component layout responsiveness.

## Media Queries

There is absolutely no need to write `@media` rules manually. Indicate `media-max` mapped specifically to JSON configurations.
```json
{
  "grid": {
    "mobile": {
      "_type": "media-max",
      "value": "768px",
      "columns": { "value": "1fr" }
    }
  }
}
```
**Outputs SCSS result:**
```scss
@media (max-width: 768px) {
  .grid {
    columns: 1fr;
  }
}
```

## Container Queries

Same structure goes for handling rules isolating containers directly through the `container-max` type mappings applying container-based boundaries logic natively instead of screen width breakpoints via `@container`.
