# Default Value (default)

The `_default` flag defines a fallback value for a property, which is used in `props.ts` and component `defaults` if no specific value is provided.

**Input Configuration (`properties.json`):**
```json
{
  "button": {
    "~size": {
      "sm": { "height": "32px" },
      "md": { "height": "40px" },
      "_default": "md"
    }
  }
}
```

**Output Result (TypeScript):**
```typescript
export const defaults = {
  size: 'md'
}
```
