# Thay thế (replace)

Cờ `_replace` cho phép sửa đổi giá trị hoặc tên cuối cùng bằng biểu thức chính quy. Nó yêu cầu một đối tượng với các trường `pattern`, `flags` và `replace`.

**Cấu hình đầu vào (`properties.json`):**
```json
{
  "button": {
    "padding": {
      "value": "12px",
      "_replace": {
        "pattern": "px",
        "replace": "rem",
        "flags": "g"
      }
    }
  }
}
```

**Kết quả (SCSS):**
```scss
.button {
  padding: 12rem;
}
```
