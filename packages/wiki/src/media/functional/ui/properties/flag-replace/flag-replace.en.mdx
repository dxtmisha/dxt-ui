# Replacement (replace)

The `_replace` flag enables modification of the final value or name using regular expressions. It expects an object with `pattern`, `flags`, and `replace` fields.

**Input Configuration (`properties.json`):**
```json
{
  "button": {
    "padding": {
      "value": "12px",
      "_replace": {
        "pattern": "px",
        "replace": "rem",
        "flags": "g"
      }
    }
  }
}
```

**Output Result (SCSS):**
```scss
.button {
  padding: 12rem;
}
```
