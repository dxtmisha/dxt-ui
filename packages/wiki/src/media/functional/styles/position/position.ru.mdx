# Позиционирование элементов

Библиотека предоставляет миксины для позиционирования элементов, создания абсолютных слоев, псевдоэлементов и отступов с поддержкой направления текста.

## Миксины

### Стандартные координаты положения

- `inset($value)` — Устанавливает свойство `inset`.
- `horizon($value)` — Устанавливает горизонтальное положение (задает и `left`, и `right`).
- `vertically($value)` — Устанавливает вертикальное положение (задает и `top`, и `bottom`).
- `top($value)` — Устанавливает свойство `top` и включает горизонтальный охват `horizon`.
- `bottom($value)` — Устанавливает свойство `bottom` и включает горизонтальный охват `horizon`.

### Позиционирование с поддержкой направления

- `left($value)` — Устанавливает `inset-inline-start` для совместимости с LTR и RTL.
- `right($value)` — Устанавливает `inset-inline-end` для совместимости с LTR и RTL.
- `__positionDir()` — Связывает физическое положение с системными переменными направления (`left: var(--d-dir-position-left)` и `right: var(--d-dir-position-right)`).

### Абсолютное позиционирование

- `absolute($value)` — Задает абсолютное позиционирование со сдвигом `inset`.
- `absoluteTop($value)` — Задает абсолютное позиционирование с выравниванием по верхнему краю.
- `absoluteBottom($value)` — Задает абсолютное позиционирование с выравниванием по нижнему краю.
- `absoluteAfter($value)` — Задает относительное позиционирование для родителя и абсолютное для псевдоэлемента `&::after`.
