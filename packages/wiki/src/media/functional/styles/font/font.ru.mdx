# Система типографики

Библиотека предоставляет миксины для управления свойствами текста, размерами шрифта, выравниванием, ограничением строк и отступами.

## Миксины

### Размер и высота текста

- `fontSize($value)` — Задает переменную `--sys-font-size` и размер шрифта.
- `lineHeight($value)` — Задает переменную `--sys-line-height` и высоту строки.

### Выравнивание и трансформация текста

- `textAlign($value)` — Выравнивает текст с поддержкой LTR и RTL (вызовы `'left'` и `'right'` заменяются на динамические).
- `textCase($value)` — Изменяет регистр символов с помощью `text-transform`.
- `textSelectNone()` — Отключает выделение текста на элементе в браузере.

### Переполнение текста и ограничение строк

- `textOverflow($value)` — Управляет переполнением текста или обрезает его.
- `truncate()` — Обрезает текст с троеточием в одну строку (`text-overflow: ellipsis`).
- `clamp($clamp)` — Ограничивает блок текста заданным количеством строк.

### Абзацные отступы

- `initParagraphSpacing()` — Инициализирует стили отступов для идущих подряд абзацев `p + p`.
- `paragraphSpacing($value)` — Задает переменную `--sys-paragraph-spacing` для абзацного отступа.
