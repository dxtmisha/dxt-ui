# Система флексбокс

Библиотека предоставляет миксины для управления флексбокс-сеткой (Flexbox), выравниванием, направлением потока и позиционированием элементов.

## Миксины

### Отображение и направление (Flow)

- `flex()` — Устанавливает свойство `display: flex`.
- `flexInline()` — Устанавливает свойство `display: inline-flex`.
- `flexX()` — Задает флекс-раскладку с выравниванием по центру (`display: flex`, `align-items: center`).
- `flexY()` — Задает вертикальную флекс-колонку с выравниванием по центру (`flex-flow: column`).
- `flexStretchX()` — Задает флекс-раскладку с растягиванием по высоте (`align-items: stretch`).
- `flexStretchY()` — Задает вертикальную колонку с растягиванием по ширине.

### Выравнивание и распределение (Alignment)

- `justifyContent($value)` — Устанавливает свойство `justify-content`.
- `flexCenter()` — Центрирует флекс-элементы по горизонтали и вертикали.
- `flexBetween()` — Раскладывает элементы с распределением свободного пространства (`justify-content: space-between`).
- `flexInlineX()` — Задает строчную флекс-раскладку с выравниванием по центру.
- `flexInlineY()` — Задает строчную вертикальную колонку с выравниванием по центру.
- `flexInlineCenter()` — Центрирует строчные флекс-элементы по горизонтали и вертикали.
- `flexInlineBetween()` — Раскладывает строчные элементы с распределением свободного пространства.

### Размеры и позиционирование

- `flexPosition($value)` — Динамически применяет один из предопределенных миксинов выравнивания на основе значения параметра (`'x'`, `'y'`, `'center'`, `'between'`, `'inline-x'`, `'inline-y'`, `'inline-center'`, `'inline-between'`).
- `flexDynamic()` — Задает правила гибких размеров (`flex-basis: 1px`, `flex-grow: 1`, `flex-shrink: 1`).
