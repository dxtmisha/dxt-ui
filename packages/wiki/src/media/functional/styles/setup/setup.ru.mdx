# Подготовка проекта

Библиотека `@dxtmisha/styles` предоставляет централизованный миксин подготовки (`initGlobal`) для настройки базовых сбросов (CSS Resets), системы направлений раскладки, абзацных отступов, трансформаций и цветовых палитр.

## Как это работает

Миксин `initGlobal` выступает единой точкой входа для подготовки всей системы стилей проекта. Он последовательно подключает и настраивает ключевые низкоуровневые утилиты:

1. **Базовый сброс параметров (Box Sizing & Borders)**: Для всех элементов (`*`, `*::after`, `*::before`) задается режим `box-sizing: border-box`, обнуляется ширина границ (`border-width: 0`), тип границ устанавливается в `solid`, а цвет границ по умолчанию делается прозрачным (`transparent`). Это предотвращает проблемы с нежелательным влиянием рамок на размеры элементов и их позиционирование.
2. **Инициализация направления текста (`initDir`)**: Подключает систему поддержки направлений LTR и RTL. Она настраивает необходимые CSS-переменные на элементе `html` и дочерних элементах, обеспечивая автоматическое изменение раскладки и отступов при переключении локалей.
3. **Абзацные отступы (`initParagraphSpacing`)**: Автоматически задает и форматирует отступы между идущими друг за другом абзацами (элементами `p`), обеспечивая визуальную согласованность текстовых блоков.
4. **Палитра цветов (`initPalette`)**: Регистрирует глобальные CSS-переменные для цветовой схемы проекта.
5. **Трансформации (`initTransform`)**: Подготавливает систему CSS-переменных для управления трехмерными и двумерными трансформациями элементов.
6. **Сброс внешних отступов (Margin Reset)**: Сбрасывает внешние отступы до нуля у элементов `body`, заголовков (`h1`-`h6`) и абзацев (`p`).

## Интеграция в проект

### 1. Создание главного файла стилей

Создайте файл стилей (например, `src/styles/main.scss`), который будет служить точкой входа для стилей вашего приложения.

### 2. Импорт библиотеки стилей

Импортируйте библиотеку `@dxtmisha/styles` в начале вашего SCSS-файла:

```scss
@use '@dxtmisha/styles';
```

### 3. Подготовка глобальных стилей

Вызовите миксин `initGlobal` на корневом уровне в вашем главном SCSS-файле:

```scss
@include styles.initGlobal;
```

### 4. Подключение стилей в точке входа приложения

Импортируйте ваш главный файл стилей в JavaScript или TypeScript точке входа приложения (например, `src/main.ts` или `src/entry-client.ts`):

```typescript
import './styles/main.scss'
```
