# `gradientOpacity`

Миксин устанавливает переменную `--sys-gradient-opacity`, обеспечивая независимый контроль прозрачности фонового градиента.

**Параметры:**
- `$opacity: Number | String` — значение прозрачности [0..1] или переменная CSS.
- `$important: Boolean` [false] — если `true`, добавляет `!important`.
- `$init: Boolean` [false] — если `true`, немедленно применяет свойство `background-image`.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-gradient-faded {
  @include styles.gradientOpacity(0.5);
}

.bg-gradient-faded-important {
  @include styles.gradientOpacity(var(--my-opacity), true, true);
}
```
