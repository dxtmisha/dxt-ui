# `gradientOpacity`

Mixin này thiết lập biến `--sys-gradient-opacity`, cho phép kiểm soát tính trong suốt của dải màu nền (gradient) một cách độc lập.

**Tham số:**
- `$opacity: Number | String` — giá trị trong suốt [0..1] hoặc biến CSS.
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.
- `$init: Boolean` [false] — nếu `true`, sẽ áp dụng ngay thuộc tính `background-image`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.bg-gradient-faded {
  @include styles.gradientOpacity(0.5);
}

.bg-gradient-faded-important {
  @include styles.gradientOpacity(var(--my-opacity), true, true);
}
```
