# `gradientOpacity`

The mixin sets the `--sys-gradient-opacity` variable, allowing for independent control of background gradient transparency.

**Parameters:**
- `$opacity: Number | String` — transparency value [0..1] or CSS variable.
- `$important: Boolean` [false] — if `true`, adds `!important`.
- `$init: Boolean` [false] — if `true`, immediately applies the `background-image` property.

## Example

```scss
@use '@dxtmisha/styles';

.bg-gradient-faded {
  @include styles.gradientOpacity(0.5);
}

.bg-gradient-faded-important {
  @include styles.gradientOpacity(var(--my-opacity), true, true);
}
```
