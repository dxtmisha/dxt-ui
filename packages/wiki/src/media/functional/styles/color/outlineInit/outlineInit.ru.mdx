# `outlineInit`

Миксин применяет текущий вычисленный системный цвет к свойству `outline-color`.

**Параметры:**
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.outline-update {
  @include styles.outlineInit;
}

.outline-force {
  @include styles.outlineInit(true);
}
```
