# `borderAsColor`

Миксин устанавливает `border-color` в соответствии с текущим системным цветом текста (`--sys-color`).

**Параметры:**
- Нет.

## Пример

```scss
@use '@dxtmisha/styles';

.border-follows-text {
  @include styles.borderAsColor;
}
```
