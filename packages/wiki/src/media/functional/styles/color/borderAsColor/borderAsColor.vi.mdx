# `borderAsColor`

Mixin thiết lập `border-color` để khớp với màu văn bản hệ thống hiện tại (`--sys-color`).

**Tham số:**
- Không.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.border-follows-text {
  @include styles.borderAsColor;
}
```
