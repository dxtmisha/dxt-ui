# `getPaletteItemOpacity`

Hàm tạo một chuỗi biến CSS lồng nhau (sử dụng tham số fallback của hàm `var()`) để tìm kiếm giá trị độ mờ trên nhiều nút bảng màu khác nhau.

**Tham số:**
- `$palette: List | String` — danh sách tên biến bảng màu hoặc một tên duy nhất.
- `$color: Color | String` — màu dự phòng nếu không có biến bảng màu nào được định nghĩa.

**Trả về:**
- `String` — một chuỗi `var(--name1-opacity, var(--name2-opacity, ...))` lồng nhau.

## Ví dụ

```scss
@use '@dxtmisha/styles';

$chain: styles.getPaletteItemOpacity(('theme-color', 'sys-color'), #ff0000); 
// var(--theme-color-opacity, var(--sys-color-opacity, 1))
```
