# `getPalette`

Hàm xác định đúng biến bảng màu cho một đầu vào màu cụ thể, xử lý cả tên bảng màu và giá trị màu trực tiếp.

**Tham số:**
- `$color: Color | String` — màu đầu vào hoặc tham chiếu biến CSS.

**Trả về:**
- `String` — kết quả tham chiếu biến CSS hoặc giá trị màu gốc.

## Ví dụ

```scss
@use '@dxtmisha/styles';

$p: styles.getPalette('--sys-primary'); 
// var(--sys-palette-primary, 255, 255, 255) (conceptual)

$direct: styles.getPalette(#ff0000);
// var(--sys-palette, 255, 0, 0)
```
