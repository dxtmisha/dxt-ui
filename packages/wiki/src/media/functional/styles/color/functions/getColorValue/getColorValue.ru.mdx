# `getColorValue`

Функция возвращает отформатированное значение CSS `rgba()` на основе заданного имени свойства, опционально используя резервный цвет по умолчанию.

**Параметры:**
- `$name: String` — имя переменной CSS (например, `--sys-color`).
- `$defaultValue: Color | String` [null] — необязательное резервное значение цвета.

**Возвращает:**
- `String` — отформатированная строка CSS `rgba()`.

## Пример

```scss
@use '@dxtmisha/styles';

$color: styles.getColorValue('--sys-primary'); 
// rgba(var(--sys-primary), var(--sys-primary-opacity, var(--sys-primary-opacity-custom, 1)))

$with-fallback: styles.getColorValue('--non-existent', #ff0000);
// rgba(var(--non-existent, 255, 0, 0), var(--non-existent-opacity, var(--non-existent-opacity-custom, 1)))
```
