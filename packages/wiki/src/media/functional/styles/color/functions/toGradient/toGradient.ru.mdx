# `toGradient`

Функция оборачивает значение цвета в функцию `linear-gradient` для обеспечения совместимости со свойством `background-image`.

**Параметры:**
- `$color: Color | String` — значение цвета для оборачивания.

**Возвращает:**
- `String` — строка в формате `linear-gradient(0deg, #color, #color)`.

## Пример

```scss
@use '@dxtmisha/styles';

$gradient: styles.toGradient(#ff0000); 
// linear-gradient(0deg, #ff0000, #ff0000)
```
