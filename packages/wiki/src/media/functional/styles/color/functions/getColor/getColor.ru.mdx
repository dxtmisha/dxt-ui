# `getColor`

Функция форматирует значение цвета с дополнительной прозрачностью и возвращает его либо как RGBA, либо обернутым в линейный градиент.

**Параметры:**
- `$color: Color | String` — базовый цвет для форматирования.
- `$opacity: Number` [1] — необязательный уровень прозрачности [0..1].
- `$gradient: Boolean` [false] — если `true`, возвращает значение в виде линейного градиента.

**Возвращает:**
- `String | Color` — отформатированное значение цвета.

## Пример

```scss
@use '@dxtmisha/styles';

$solid: styles.getColor(#ff0000, 0.5); 
// rgba(255, 0, 0, 0.5)

$gradient: styles.getColor(#ff0000, 1, true); 
// linear-gradient(0deg, #ff0000, #ff0000)
```
