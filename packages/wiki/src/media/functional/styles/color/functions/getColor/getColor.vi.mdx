# `getColor`

Hàm định dạng một giá trị màu sắc với tính trong suốt tùy chọn và trả về nó dưới dạng giá trị RGBA hoặc bọc trong một linear-gradient.

**Tham số:**
- `$color: Color | String` — màu cơ bản để định dạng.
- `$opacity: Number` [1] — mức độ mờ tùy chọn [0..1].
- `$gradient: Boolean` [false] — nếu `true`, trả về giá trị dưới dạng linear-gradient.

**Trả về:**
- `String | Color` — giá trị màu đã được định dạng.

## Ví dụ

```scss
@use '@dxtmisha/styles';

$solid: styles.getColor(#ff0000, 0.5); 
// rgba(255, 0, 0, 0.5)

$gradient: styles.getColor(#ff0000, 1, true); 
// linear-gradient(0deg, #ff0000, #ff0000)
```
