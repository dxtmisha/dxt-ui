# `isPalette`

Hàm kiểm tra xem một chuỗi có đại diện cho một biến CSS liên quan đến bảng màu hay không bằng cách tìm tiền tố `--`.

**Tham số:**
- `$color: String` — tên biến màu hoặc chuỗi cần kiểm tra.

**Trả về:**
- `Boolean` — `true` nếu chuỗi chứa tiền tố gạch nối kép (`--`).

## Ví dụ

```scss
@use '@dxtmisha/styles';

$is-palette: styles.isPalette('--sys-color'); // true
$is-not-palette: styles.isPalette('#ff0000'); // false
```
