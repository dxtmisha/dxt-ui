# `fill`

Миксин устанавливает свойство `fill` (для SVG-элементов), используя переменные дизайн-системы, что позволяет обеспечить динамическое обновление и контроль прозрачности.

**Параметры:**
- `$color: Color | String` — значение цвета, `'as-color'` или `'init'`.
- `$opacity: Number | null` [null] — необязательный уровень прозрачности [0..1].
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.icon-solid {
  @include styles.fill(styles.getColor('primary'), 1);
}

.icon-reset {
  @include styles.fill('init');
}
```
