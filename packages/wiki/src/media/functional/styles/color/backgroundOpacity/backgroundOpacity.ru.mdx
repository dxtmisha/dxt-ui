# `backgroundOpacity`

Миксин устанавливает переменную `--sys-background-opacity`, обеспечивая независимый контроль прозрачности фона.

**Параметры:**
- `$opacity: Number | String` — значение прозрачности [0..1] или переменная CSS.
- `$important: Boolean` [false] — если `true`, добавляет `!important`.
- `$init: Boolean` [false] — если `true`, немедленно применяет свойство `background-color`.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-faded {
  @include styles.backgroundOpacity(0.5);
}

.bg-faded-important {
  @include styles.backgroundOpacity(var(--my-opacity), true, true);
}
```
