# `backgroundOpacity`

The mixin sets the `--sys-background-opacity` variable, allowing for independent control of background transparency.

**Parameters:**
- `$opacity: Number | String` — transparency value [0..1] or CSS variable.
- `$important: Boolean` [false] — if `true`, adds `!important`.
- `$init: Boolean` [false] — if `true`, immediately applies the `background-color` property.

## Example

```scss
@use '@dxtmisha/styles';

.bg-faded {
  @include styles.backgroundOpacity(0.5);
}

.bg-faded-important {
  @include styles.backgroundOpacity(var(--my-opacity), true, true);
}
```
