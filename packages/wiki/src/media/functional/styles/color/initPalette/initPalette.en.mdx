# `initPalette`

The mixin initializes the `--sys-palette-color` variable, which serves as the core reference for palette-based styles.

**Parameters:**
- `$color: Color | String` — the color value to initialize.
- `$important: Boolean` [false] — if `true`, adds `!important`.

## Example

```scss
@use '@dxtmisha/styles';

.local-theme {
  @include styles.initPalette(#ff0000);
}
```
