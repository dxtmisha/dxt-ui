# `strokeOpacity`

Mixin này thiết lập biến `--sys-stroke-opacity`, cho phép kiểm soát tính trong suốt của nét vẽ SVG một cách độc lập.

**Tham số:**
- `$opacity: Number | String` — giá trị trong suốt [0..1] hoặc biến CSS.
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.
- `$init: Boolean` [false] — nếu `true`, sẽ áp dụng ngay thuộc tính `stroke`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.icon-outline-faded {
  @include styles.strokeOpacity(0.5);
}

.icon-outline-faded-important {
  @include styles.strokeOpacity(var(--my-opacity), true, true);
}
```
