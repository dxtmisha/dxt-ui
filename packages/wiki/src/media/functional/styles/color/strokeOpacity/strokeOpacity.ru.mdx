# `strokeOpacity`

Миксин устанавливает переменную `--sys-stroke-opacity`, обеспечивая независимый контроль прозрачности обводки SVG.

**Параметры:**
- `$opacity: Number | String` — значение прозрачности [0..1] или переменная CSS.
- `$important: Boolean` [false] — если `true`, добавляет `!important`.
- `$init: Boolean` [false] — если `true`, немедленно применяет свойство `stroke`.

## Пример

```scss
@use '@dxtmisha/styles';

.icon-outline-faded {
  @include styles.strokeOpacity(0.5);
}

.icon-outline-faded-important {
  @include styles.strokeOpacity(var(--my-opacity), true, true);
}
```
