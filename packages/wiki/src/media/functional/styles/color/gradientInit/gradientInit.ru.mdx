# `gradientInit`

Миксин применяет текущий вычисленный системный градиент к свойству `background-image`.

**Параметры:**
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-gradient-update {
  @include styles.gradientInit;
}

.bg-gradient-force {
  @include styles.gradientInit(true);
}
```
