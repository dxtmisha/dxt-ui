# `gradientInit`

Mixin áp dụng dải màu hệ thống (gradient) hiện tại đã được tính toán cho thuộc tính `background-image`.

**Tham số:**
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.bg-gradient-update {
  @include styles.gradientInit;
}

.bg-gradient-force {
  @include styles.gradientInit(true);
}
```
