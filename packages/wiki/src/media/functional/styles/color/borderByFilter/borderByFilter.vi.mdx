# `borderByFilter`

Mixin thiết lập `border-color` dựa trên bộ lọc (filter) nào hiện đang hoạt động trong hệ thống thiết kế.

**Tham số:**
- `$filter: String` — tên bộ lọc tìm kiếm.
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.border-filtered {
  @include styles.borderByFilter('search-active');
}
```
