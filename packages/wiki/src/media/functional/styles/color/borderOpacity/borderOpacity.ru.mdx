# `borderOpacity`

Миксин устанавливает переменную `--sys-border-opacity`, обеспечивая независимый контроль прозрачности границ.

**Параметры:**
- `$opacity: Number | String` — значение прозрачности [0..1] или переменная CSS.
- `$important: Boolean` [false] — если `true`, добавляет `!important`.
- `$init: Boolean` [false] — если `true`, немедленно применяет свойство `border-color`.

## Пример

```scss
@use '@dxtmisha/styles';

.border-faded {
  @include styles.borderOpacity(0.5);
}

.border-faded-important {
  @include styles.borderOpacity(var(--my-opacity), true, true);
}
```
