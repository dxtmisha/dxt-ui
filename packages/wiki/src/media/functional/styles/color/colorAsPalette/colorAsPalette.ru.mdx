# `colorAsPalette`

Миксин применяет свойство `color`, используя текущее состояние переменных палитры, с откатом к системному цвету, если палитра не определена.

**Параметры:**
- Нет.

## Пример

```scss
@use '@dxtmisha/styles';

.text-follow-palette {
  @include styles.colorAsPalette;
}
```
