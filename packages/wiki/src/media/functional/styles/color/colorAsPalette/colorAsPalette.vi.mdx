# `colorAsPalette`

Mixin áp dụng thuộc tính `color` bằng cách sử dụng trạng thái hiện tại của các biến bảng màu, dự phòng về màu hệ thống nếu bảng màu không được xác định.

**Tham số:**
- Không.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.text-follow-palette {
  @include styles.colorAsPalette;
}
```
