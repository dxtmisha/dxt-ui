# `backgroundSize`

Миксин устанавливает свойство `background-size`, обеспечивая правильное масштабирование фоновых изображений и градиентов.

**Параметры:**
- `$value: String | Number` — значение размера (например, 'cover', 'contain', '100%').

## Пример

```scss
@use '@dxtmisha/styles';

.bg-cover {
  @include styles.backgroundSize('cover');
}
```
