# `backgroundSize`

Mixin thiết lập thuộc tính `background-size`, đảm bảo tỷ lệ hình nền và dải màu (gradient) được hiển thị chính xác.

**Tham số:**
- `$value: String | Number` — giá trị kích thước (ví dụ: 'cover', 'contain', '100%').

## Ví dụ

```scss
@use '@dxtmisha/styles';

.bg-cover {
  @include styles.backgroundSize('cover');
}
```
