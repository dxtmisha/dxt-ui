# `backgroundInit`

Mixin áp dụng màu hệ thống hiện tại đã được tính toán cho thuộc tính `background-color`.

**Tham số:**
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.bg-update {
  @include styles.backgroundInit;
}

.bg-force {
  @include styles.backgroundInit(true);
}
```
