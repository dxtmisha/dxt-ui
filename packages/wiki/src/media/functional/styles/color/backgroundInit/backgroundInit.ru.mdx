# `backgroundInit`

Миксин применяет текущий вычисленный системный цвет к свойству `background-color`.

**Параметры:**
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-update {
  @include styles.backgroundInit;
}

.bg-force {
  @include styles.backgroundInit(true);
}
```
