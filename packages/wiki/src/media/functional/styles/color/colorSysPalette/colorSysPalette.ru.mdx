# `colorSysPalette`

Миксин устанавливает системную переменную цвета (например, `--sys-color`), генерируя цепочку поиска по указанным переменным палитры.

**Параметры:**
- `$prop: String` — имя свойства (например,, 'color', 'background').
- `$palette: List | String` — список имен переменных палитры для поиска.
- `$color: Color | String` — резервный цвет, если переменные палитры не определены.
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.sys-themed {
  @include styles.colorSysPalette('color', ('theme-primary', 'sys-primary'), #0000ff);
}
```
