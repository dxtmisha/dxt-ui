# `colorSysPalette`

Mixin thiết lập một biến màu hệ thống (ví dụ: `--sys-color`) bằng cách tạo chuỗi tìm kiếm qua các biến bảng màu đã chỉ định.

**Tham số:**
- `$prop: String` — tên thuộc tính (ví dụ: 'color', 'background').
- `$palette: List | String` — danh sách tên biến bảng màu để tìm kiếm.
- `$color: Color | String` — màu dự phòng nếu không có biến bảng màu nào được xác định.
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.sys-themed {
  @include styles.colorSysPalette('color', ('theme-primary', 'sys-primary'), #0000ff);
}
```
