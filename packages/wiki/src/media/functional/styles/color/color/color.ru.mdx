# `color`

Миксин устанавливает свойство `color` (цвет текста), используя переменные дизайн-системы, что позволяет обеспечить динамическое обновление и контроль прозрачности.

**Параметры:**
- `$color: Color | String` — значение цвета, `'as-color'` или `'init'`.
- `$opacity: Number | null` [null] — необязательный уровень прозрачности [0..1].
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.text-primary {
  @include styles.color(styles.getColor('primary'), 0.8);
}

.text-reset {
  @include styles.color('init');
}
```
