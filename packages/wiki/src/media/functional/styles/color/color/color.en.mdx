# `color`

The mixin sets the `color` property (text color) using the design system's variables, allowing for dynamic updates and opacity control.

**Parameters:**
- `$color: Color | String` — color value, `'as-color'`, or `'init'`.
- `$opacity: Number | null` [null] — optional opacity level [0..1].
- `$important: Boolean` [false] — if `true`, adds `!important`.

## Example

```scss
@use '@dxtmisha/styles';

.text-primary {
  @include styles.color(styles.getColor('primary'), 0.8);
}

.text-reset {
  @include styles.color('init');
}
```
