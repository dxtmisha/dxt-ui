# `strokeInit`

Mixin áp dụng màu hệ thống hiện tại đã được tính toán cho thuộc tính `stroke` của SVG.

**Tham số:**
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.icon-outline-update {
  @include styles.strokeInit;
}

.icon-outline-force {
  @include styles.strokeInit(true);
}
```
