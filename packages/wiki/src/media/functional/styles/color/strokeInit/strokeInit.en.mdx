# `strokeInit`

The mixin applies the current computed system color to the SVG `stroke` property.

**Parameters:**
- `$important: Boolean` [false] — if `true`, adds `!important`.

## Example

```scss
@use '@dxtmisha/styles';

.icon-outline-update {
  @include styles.strokeInit;
}

.icon-outline-force {
  @include styles.strokeInit(true);
}
```
