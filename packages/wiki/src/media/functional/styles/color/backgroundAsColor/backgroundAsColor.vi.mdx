# `backgroundAsColor`

Mixin thiết lập `background-color` để khớp với màu văn bản hệ thống hiện tại (`--sys-color`).

**Tham số:**
- Không.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.bg-follows-text {
  @include styles.backgroundAsColor;
}
```
