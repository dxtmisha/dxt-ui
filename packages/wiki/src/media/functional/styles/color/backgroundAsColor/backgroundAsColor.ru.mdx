# `backgroundAsColor`

Миксин устанавливает `background-color` в соответствии с текущим системным цветом текста (`--sys-color`).

**Параметры:**
- Нет.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-follows-text {
  @include styles.backgroundAsColor;
}
```
