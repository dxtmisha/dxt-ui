# `paletteVar`

Миксин устанавливает указанную переменну CSS, генерируя цепочку поиска по предоставленным переменным палитры.

**Параметры:**
- `$name: String` — имя переменной CSS для установки (например, '--local-color').
- `$palette: List | String` — список имен переменных палитры для поиска.
- `$color: Color | String` — резервный цвет, если переменные палитры не определены.
- `$opacity: Number | null` [null] — необязательный уровень прозрачности [0..1].
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.custom-component {
  @include styles.paletteVar('--my-color', ('theme-primary', 'sys-primary'), #0000ff);
}
```
