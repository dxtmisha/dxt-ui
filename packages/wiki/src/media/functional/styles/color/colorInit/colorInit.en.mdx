# `colorInit`

The mixin applies the current computed system color to the `color` property.

**Parameters:**
- `$important: Boolean` [false] — if `true`, adds `!important`.

## Example

```scss
@use '@dxtmisha/styles';

.text-update {
  @include styles.colorInit;
}

.text-force {
  @include styles.colorInit(true);
}
```
