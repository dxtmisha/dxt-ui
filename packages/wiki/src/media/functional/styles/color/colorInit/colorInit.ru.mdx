# `colorInit`

Миксин применяет текущий вычисленный системный цвет к свойству `color`.

**Параметры:**
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.text-update {
  @include styles.colorInit;
}

.text-force {
  @include styles.colorInit(true);
}
```
