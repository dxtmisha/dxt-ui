# `colorInit`

Mixin áp dụng màu hệ thống hiện tại đã được tính toán cho thuộc tính `color`.

**Tham số:**
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.text-update {
  @include styles.colorInit;
}

.text-force {
  @include styles.colorInit(true);
}
```
