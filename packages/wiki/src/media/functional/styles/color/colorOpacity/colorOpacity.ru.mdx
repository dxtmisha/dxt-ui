# `colorOpacity`

Миксин устанавливает переменную `--sys-color-opacity`, обеспечивая независимый контроль прозрачности текста.

**Параметры:**
- `$opacity: Number | String` — значение прозрачности [0..1] или переменная CSS.
- `$important: Boolean` [false] — если `true`, добавляет `!important`.
- `$init: Boolean` [false] — если `true`, немедленно применяет свойство `color`.

## Пример

```scss
@use '@dxtmisha/styles';

.text-faded {
  @include styles.colorOpacity(0.5);
}

.text-faded-important {
  @include styles.colorOpacity(var(--my-opacity), true, true);
}
```
