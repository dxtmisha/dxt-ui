# `borderColor`

The mixin sets the `border-color` property using the design system's variables, allowing for dynamic updates and opacity control.

**Parameters:**
- `$color: Color | String` — color value, `'as-color'`, or `'init'`.
- `$opacity: Number | null` [null] — optional opacity level [0..1].
- `$important: Boolean` [false] — if `true`, adds `!important`.

## Example

```scss
@use '@dxtmisha/styles';

.border-primary {
  @include styles.borderColor(styles.getColor('primary'), 1);
}

.border-reset {
  @include styles.borderColor('init');
}
```
