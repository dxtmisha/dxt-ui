# `borderColor`

Миксин устанавливает свойство `border-color`, используя переменные дизайн-системы, что позволяет обеспечить динамическое обновление и контроль прозрачности.

**Параметры:**
- `$color: Color | String` — значение цвета, `'as-color'` или `'init'`.
- `$opacity: Number | null` [null] — необязательный уровень прозрачности [0..1].
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.border-primary {
  @include styles.borderColor(styles.getColor('primary'), 1);
}

.border-reset {
  @include styles.borderColor('init');
}
```
