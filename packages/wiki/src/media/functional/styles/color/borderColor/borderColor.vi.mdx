# `borderColor`

Mixin thiết lập thuộc tính `border-color` bằng cách sử dụng các biến của hệ thống thiết kế, cho phép cập nhật động và kiểm soát độ mờ.

**Tham số:**
- `$color: Color | String` — giá trị màu, `'as-color'`, hoặc `'init'`.
- `$opacity: Number | null` [null] — mức độ mờ tùy chọn [0..1].
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.border-primary {
  @include styles.borderColor(styles.getColor('primary'), 1);
}

.border-reset {
  @include styles.borderColor('init');
}
```
