# `paletteGradient`

Mixin thiết lập `background-image` (dưới dạng dải màu tuyến tính rắn) bằng cách tạo chuỗi tìm kiếm qua các biến bảng màu đã chỉ định.

**Tham số:**
- `$palette: List | String` — danh sách tên biến bảng màu để tìm kiếm.
- `$color: Color | String` — màu dự phòng nếu không có biến bảng màu nào được xác định.
- `$opacity: Number | null` [null] — mức độ mờ tùy chọn [0..1].
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.bg-gradient-themed {
  @include styles.paletteGradient(('theme-primary', 'sys-primary'), #0000ff);
}
```
