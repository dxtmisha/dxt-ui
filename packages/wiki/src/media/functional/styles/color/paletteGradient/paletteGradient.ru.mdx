# `paletteGradient`

Миксин устанавливает `background-image` (как сплошной линейный градиент), генерируя цепочку поиска по указанным переменным палитры.

**Параметры:**
- `$palette: List | String` — список имен переменных палитры для поиска.
- `$color: Color | String` — резервный цвет, если переменные палитры не определены.
- `$opacity: Number | null` [null] — необязательный уровень прозрачности [0..1].
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-gradient-themed {
  @include styles.paletteGradient(('theme-primary', 'sys-primary'), #0000ff);
}
```
