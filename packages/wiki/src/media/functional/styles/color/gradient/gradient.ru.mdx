# `gradient`

Миксин устанавливает свойство `background-image`, используя переменные дизайн-системы, отформатированные как сплошной линейный градиент.

**Параметры:**
- `$color: Color | String` — значение цвета, `'as-color'` или `'init'`.
- `$opacity: Number | null` [null] — необязательный уровень прозрачности [0..1].
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.bg-gradient-primary {
  @include styles.gradient(styles.getColor('primary'), 0.9);
}

.bg-gradient-reset {
  @include styles.gradient('init');
}
```
