# `outlineColor`

The mixin sets the `outline-color` property using the design system's variables, allowing for dynamic updates and opacity control.

**Parameters:**
- `$color: Color | String` — color value, `'as-color'`, or `'init'`.
- `$opacity: Number | null` [null] — optional opacity level [0..1].
- `$important: Boolean` [false] — if `true`, adds `!important`.

## Example

```scss
@use '@dxtmisha/styles';

.outline-primary {
  @include styles.outlineColor(styles.getColor('primary'), 0.8);
}

.outline-reset {
  @include styles.outlineColor('init');
}
```
