# `fillInit`

Миксин применяет текущий вычисленный системный цвет к свойству SVG `fill`.

**Параметры:**
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.icon-update {
  @include styles.fillInit;
}

.icon-force {
  @include styles.fillInit(true);
}
```
