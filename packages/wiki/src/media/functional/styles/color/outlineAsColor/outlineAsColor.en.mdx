# `outlineAsColor`

The mixin sets the `outline-color` property to match the current system text color (`--sys-color`).

**Parameters:**
- None.

## Example

```scss
@use '@dxtmisha/styles';

.outline-follows-text {
  @include styles.outlineAsColor;
}
```
