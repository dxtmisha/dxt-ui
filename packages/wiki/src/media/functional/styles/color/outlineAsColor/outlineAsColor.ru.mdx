# `outlineAsColor`

Миксин устанавливает свойство `outline-color` в соответствии с текущим системным цветом текста (`--sys-color`).

**Параметры:**
- Нет.

## Пример

```scss
@use '@dxtmisha/styles';

.outline-follows-text {
  @include styles.outlineAsColor;
}
```
