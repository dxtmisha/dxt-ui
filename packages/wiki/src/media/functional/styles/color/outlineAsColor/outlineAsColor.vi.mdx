# `outlineAsColor`

Mixin thiết lập thuộc tính `outline-color` để khớp với màu văn bản hệ thống hiện tại (`--sys-color`).

**Tham số:**
- Không.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.outline-follows-text {
  @include styles.outlineAsColor;
}
```
