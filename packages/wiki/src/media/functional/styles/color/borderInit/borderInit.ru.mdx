# `borderInit`

Миксин применяет текущий вычисленный системный цвет к свойству `border-color`.

**Параметры:**
- `$important: Boolean` [false] — если `true`, добавляет `!important`.

## Пример

```scss
@use '@dxtmisha/styles';

.border-update {
  @include styles.borderInit;
}

.border-force {
  @include styles.borderInit(true);
}
```
