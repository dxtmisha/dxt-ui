# `borderInit`

Mixin áp dụng màu hệ thống hiện tại đã được tính toán cho thuộc tính `border-color`.

**Tham số:**
- `$important: Boolean` [false] — nếu `true`, thêm `!important`.

## Ví dụ

```scss
@use '@dxtmisha/styles';

.border-update {
  @include styles.borderInit;
}

.border-force {
  @include styles.borderInit(true);
}
```
