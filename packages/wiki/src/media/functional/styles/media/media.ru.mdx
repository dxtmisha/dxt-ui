# Адаптивная раскладка (Media Queries)

Библиотека предоставляет функцию и миксины для работы с адаптивными медиазапросами экрана, предпочтениями анимации и контейнерными запросами (Container Queries).

## Функции

- `media($width): Number` — Возвращает значение ширины для указанного имени размера (например, `'sm'` -> `640px`) из дизайн-переменных.

## Миксины

### Медиазапросы экрана (Viewport)

- `mediaMinWidth($width)` — Генерирует медиазапрос `@media screen` для экранов с шириной не меньше указанной.
- `mediaMaxWidth($width)` — Генерирует медиазапрос `@media screen` для экранов с шириной не больше указанной.
- `mediaMinMaxWidth($min, $max)` — Генерирует медиазапрос для экранов с шириной в диапазоне между указанными размерами.
- `mediaPrefersReducedMotion()` — Генерирует медиазапрос для пользователей с предпочтением уменьшения движения (анимаций).

### Контейнерные запросы (Container Queries)

- `containerMinWidth($width)` — Генерирует запрос `@container` для контейнеров с шириной не меньше указанной.
- `containerMaxWidth($width)` — Генерирует запрос `@container` для контейнеров с шириной не больше указанной.
- `containerMinMaxWidth($min, $max)` — Генерирует контейнерный запрос для ширины в диапазоне между указанными размерами.
