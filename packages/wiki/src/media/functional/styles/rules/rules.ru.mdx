# Вспомогательные функции (Rules)

Библиотека `@dxtmisha/styles` предоставляет низкоуровневые вспомогательные функции для проверки типов данных, обхода древовидных структур карт (Sass map), манипуляций со строками и извлечения селекторов.

## Функции

### Свойства CSS и селекторы

- `isCustomVar($value): Boolean` — Проверяет, отформатировано ли значение как CSS Custom Property (например, начинается с `var(` и заканчивается на `)`).
- `getFirstSelector($selector: &): String` — Извлекает имя первого простого селектора из текущего контекста вложенности.

### Работа с картами (Sass Maps)

- `getMapItem($map, $index, $default: null): Any` — Рекурсивно извлекает значение из древовидной структуры карт по списку ключей или по одному ключу. Возвращает `$default`, если значение не найдено.
- `getMapByIndex($map, $index, $default: null): Any` — Напрямую извлекает значение из карты по ключу, возвращая `$default`, если ключ отсутствует.
- `setMapItem($map, $index, $value): Map` — Добавляет или обновляет пару ключ-значение в карте и возвращает обновленный объект.

### Обработка строк

- `strToList($value, $separator: '.'): List` — Разбивает исходную строку на список подстрок с использованием указанного разделителя.
- `strReplace($value, $pattern, $replacement): String` — Рекурсивно заменяет все вхождения подстроки поиска на указанную строку замены.
