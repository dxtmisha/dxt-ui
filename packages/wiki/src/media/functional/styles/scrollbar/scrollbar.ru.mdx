# Полосы прокрутки (Scrollbars)

Библиотека предоставляет миксин для управления внешним видом полос прокрутки элементов.

## Миксины

- `scrollbarHide()` — Визуально скрывает полосу прокрутки браузера, сохраняя при этом возможность прокрутки элемента.
