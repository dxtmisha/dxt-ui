# Селекторы состояний

Библиотека `@dxtmisha/styles` предоставляет вспомогательные миксины для управления интерактивными состояниями элементов (наведение, фокус, нажатие, отключенное состояние, режим чтения) с автоматической фильтрацией активных (enabled) элементов. Это предотвращает срабатывание стилей наведения и активации для отключенных, загружающихся или заблокированных элементов.

## Миксины

### Интерактивные состояния

- `hover($parent: &)` — Применяет стили наведения `:hover` только для активных элементов.
- `active($parent: &)` — Применяет стили нажатия `:active` только для активных элементов.
- `focus($parent: &)` — Применяет стили фокуса `:focus` только для активных элементов.

### Базовые селекторы состояний

- `selector($name, $parent: &)` — Применяет псевдокласс или модификатор состояния (например, для `focus` автоматически задает `:focus`, `:focus-within` и класс `--focus`).
- `enabled($parent: &)` — Ограничивает область действия стилей только активными элементами (исключает `:disabled`, `--disabled`, `--readonly` и `--loading` с помощью `:not`).
- `enabledSelector($name, $parent: &)` — Объединяет проверку активности `enabled` и выбор псевдокласса `selector`.
- `disabled($parent: &)` — Задает стили для заблокированного состояния (целится в `:disabled`, а также BEM-классы `--disabled`).
- `readonly($parent: &)` — Задает стили для режима "только чтение" (целится в BEM-классы `--readonly`).
