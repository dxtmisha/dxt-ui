# Система размеров

Библиотека предоставляет миксины и функции для управления размерами, пропорциями и соотношением сторон с использованием CSS-переменных.

## Функции

- `getWidthByRatio($value: String): String` — Возвращает ширину из строки соотношения сторон (например, '16/9' -> 16).
- `getHeightByRatio($value: String): String` — Возвращает высоту из строки соотношения сторон.

## Миксины

### Базовые размеры

- `width($value: String | Number)` — Устанавливает свойство `width` и CSS-переменную `--sys-width`.
- `height($value: String | Number)` — Устанавливает свойство `height` и CSS-переменную `--sys-height`.
- `widthBasis($value: String | Number)` — Устанавливает `width`, `flex-basis` и ограничивает `max-width` до 100%.
- `heightBasis($value: String | Number)` — Устанавливает `height`, `flex-basis` и ограничивает `max-height` до 100%.

### Адаптивные размеры (Viewport)

- `widthDevice($value: String | Number)` — Устанавливает ширину в динамических единицах области просмотра (`dvw`).
- `heightDevice($value: String | Number)` — Устанавливает высоту в динамических единицах области просмотра (`dvh`).
- `maxWidthDevice($value: String | Number)` — Устанавливает `max-width` в динамических единицах области просмотра (`dvw`).
- `maxHeightDevice($value: String | Number)` — Устанавливает `max-height` в динамических единицах области просмотра (`dvh`).

### Соотношение сторон и формы

- `aspectRatio($value: String | Number)` — Устанавливает свойство `aspect-ratio`.
- `aspectRatioWidth($value: String | Number)` — Псевдоним для миксина `aspectRatio`.
- `aspectRatioHeight($value: String | Number)` — Псевдоним для миксина `aspectRatio`.
- `squared($value: String | Number)` — Применяет `widthBasis` и устанавливает `aspect-ratio: 1/1` для создания квадрата.
- `circle($value: String | Number)` — Применяет миксин `squared` и устанавливает `border-radius: 100%` для создания круга.
