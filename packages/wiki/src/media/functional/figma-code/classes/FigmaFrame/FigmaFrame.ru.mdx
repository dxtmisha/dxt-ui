# Класс `FigmaFrame`

Утилитарный класс для анализа и взаимодействия со структурой страниц или выделенных областей в Figma. Он рекурсивно сканирует дерево узлов, фильтрует элементы по типу и предоставляет вспомогательные методы для извлечения текста и генерации скриншотов.

## Ключевые особенности

- **Рекурсивное сканирование** — Автоматически индексирует все вложенные элементы внутри предоставленного контекста (страница или выделение).
- **Поддержка выделения** — Может быть инициализирован для работы как со всей страницей документа, так и только с текущим выделением пользователя.
- **Агрегация текстов** — Метод `getTexts()` группирует идентичные текстовые строки, возвращая список уникальных значений и связанных с ними ID узлов.
- **Автоматизированный экспорт** — Упрощает процесс создания скриншотов для основных элементов внутри фрейма.

## Инициализация

Создайте новый экземпляр, передав начальный узел (обычно страницу) и указав, следует ли фокусироваться на выделении.

```typescript
import { FigmaFrame } from '@dxtmisha/figma-code'

// Анализ всей текущей страницы
const frameAnalyzer = new FigmaFrame(figma.currentPage)

// Анализ только выделенных объектов
const selectionAnalyzer = new FigmaFrame(figma.currentPage, true)
```

## Методы

### Фильтрация

Методы для получения конкретных типов элементов из фрейма.

- `getItemsFrame()` — Возвращает все вложенные узлы-фреймы.
- `getItemsSection()` — Возвращает все вложенные узлы-секции.
- `getItemsText()` — Возвращает все вложенные текстовые узлы.

### Анализ

Методы для структурного анализа и извлечения данных.

- `getMainFrames()` — Возвращает фреймы или секции верхнего уровня из корня.
- `getItemsInfo(): Promise<UiFigmaFramesList>` — Возвращает подробную информацию (имя, ID, изображение) для всех элементов во фрейме.
- `getMainItemsInfo(): Promise<UiFigmaFramesList>` — Возвращает подробную информацию (имя, ID, изображение) для всех основных элементов.
- `getTexts(): UiFigmaItemText[]` — Возвращает массив уникальных текстовых элементов, найденных внутри фрейма.

### Действия

Методы для выполнения операций над элементами фрейма.

- `screenshot(): Promise<Uint8Array[]>` — Экспортирует все основные элементы в формате JPG.

## Типы данных

#### `UiFigmaNode`
Объединение типов узлов Figma (SceneNode, FrameNode, TextNode и др.), поддерживающих базовые свойства и вложенность.

#### `UiFigmaItemText`
Структура для представления сгруппированных текстовых данных.
- `id: string[]` — Список идентификаторов узлов с одинаковым текстом.
- `text: string` — Содержимое текстового поля.

#### `UiFigmaFramesItem`
Представляет упрощенный элемент фрейма для пользовательского интерфейса.
- `name: string` — Название фрейма.
- `id: string` — Уникальный идентификатор элемента.
- `image: Uint8Array | string` — Изображение фрейма (опционально).

#### `UiFigmaFramesList`
Список объектов типа `UiFigmaFramesItem`.

#### `UiFigmaMessageTexts`
Объект, передаваемый при отправке данных о текстах в UI.
- `frame: FigmaFrame` — Экземпляр анализатора фрейма.
- `texts: UiFigmaItemText[]` — Список найденных текстов.
- `screenshot: Uint8Array[]` — Массив скриншотов в бинарном формате.
