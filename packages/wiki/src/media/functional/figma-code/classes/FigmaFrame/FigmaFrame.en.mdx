# Class `FigmaFrame`

A utility class for analyzing and interacting with the structure of Figma pages or selections. It recursively scans node trees, filters elements by type, and provides helper methods for text extraction and screenshot generation.

## Key Features

- **Recursive Scanning** — Automatically indexes all nested elements within the provided context (page or selection).
- **Selection Support** — Can be initialized to target either the entire document page or only the user's current selection.
- **Text Aggregation** — The `getTexts()` method groups identical text strings together, providing a list of unique values and their associated node IDs.
- **Automated Export** — Simplifies the process of generating screenshots for main elements within the frame.

## Initialization

Create a new instance by providing a starting node (usually a page) and specifying if it should focus on the selection.

```typescript
import { FigmaFrame } from '@dxtmisha/figma-code'

// Analyze the entire current page
const frameAnalyzer = new FigmaFrame(figma.currentPage)

// Analyze only the selected items
const selectionAnalyzer = new FigmaFrame(figma.currentPage, true)
```

## Methods

### Filtering

Methods for retrieving specific types of elements from the frame.

- `getItemsFrame()` — Returns all nested frame nodes.
- `getItemsSection()` — Returns all nested section nodes.
- `getItemsText()` — Returns all nested text nodes.

### Analysis

Methods for structural analysis and data extraction.

- `getMainFrames()` — Returns top-level frames or sections from the root.
- `getTexts(): UiFigmaItemText[]` — Returns an array of unique text items found within the frame.

### Actions

Methods for performing operations on frame elements.

- `screenshot(): Promise<Uint8Array[]>` — Exports all main items as JPG images.

## Data Types

#### `UiFigmaNode`
A union type of Figma node objects (SceneNode, FrameNode, TextNode, etc.) that support basic properties and children.

#### `UiFigmaItemText`
Structure for representing aggregated text data.
- `id: string[]` — List of node identifiers with identical text content.
- `text: string` — The actual text string.

#### `UiFigmaMessageTexts`
The object structure used when sending text tracking data to the UI.
- `frame: FigmaFrame` — The frame analyzer instance.
- `texts: UiFigmaItemText[]` — List of extracted text items.
- `screenshot: Uint8Array[]` — Array of screenshots as byte arrays.
