# FigmaItem Class

A wrapper class for Figma nodes (`UiFigmaNode`) that provides a simplified, typed interface for common operations such as type checking, tree navigation, and asynchronous export.

## Key Features

- **Typed Access** ‚Äî Provides type guard methods for `DOCUMENT`, `FRAME`, `SECTION`, and `TEXT` nodes.
- **Simplified Navigation** ‚Äî Easily access parent or child nodes wrapped in `FigmaItem` instances.
- **Export Hook** ‚Äî High-level methods for exporting nodes to JPG or JSON formats.
- **Content Analysis** ‚Äî Specialized logic for extracting and validating text content.

## Initialization

To initialize the object, call the `FigmaItem(item)` constructor.

### Generic Constraints

The class supports a generic parameter for strict typing:
- `T extends UiFigmaNode` ‚Äî The specific type of the wrapped Figma node.

**Parameters:**
- `item: T` ‚Äî The native Figma node (e.g., `SceneNode`, `FrameNode`, `TextNode`) to be wrapped.

```typescript
import { FigmaItem } from '@dxtmisha/figma-code'

// 1. Wrapping a selected node
const item = new FigmaItem(figma.currentPage.selection[0])

// 2. Using generics for specific node types
const frameItem = new FigmaItem<FrameNode>(someFrameNode)
```

## Identification

- `isDocument(): boolean` ‚Äî Checks if the wrapped node is a `DOCUMENT`. Example: `item.isDocument()`
- `isFrame(): boolean` ‚Äî Checks if the wrapped node is a `FRAME`. Example: `item.isFrame()`
- `isSection(): boolean` ‚Äî Checks if the wrapped node is a `SECTION`. Example: `item.isSection()`
- `isText(): boolean` ‚Äî Checks if the wrapped node is a `TEXT` node. Example: `item.isText()`
- `isTextNoValue(): boolean` ‚Äî Filters for text nodes with meaningful content. Example: `item.isTextNoValue()`
- `getType(): string | undefined` ‚Äî Returns the string type constant of the node. Example: `const type = item.getType()`
- `getId(): string` ‚Äî Returns the unique ID of the node. Example: `const id = item.getId()`
- `getName(): string` ‚Äî Returns the display name of the node. Example: `const name = item.getName()`

## Navigation

- `getParent(): UiFigmaNode | undefined` ‚Äî Returns the raw parent node. Example: `const parent = item.getParent()`
- `getParentItem(): FigmaItem | undefined` ‚Äî Returns the wrapped parent. Example: `const parentItem = item.getParentItem()`
- `getChildren(): UiFigmaNode[]` ‚Äî Returns an array of raw child nodes. Example: `const children = item.getChildren()`
- `getChildrenItems(): FigmaItem[]` ‚Äî Returns an array of wrapped child nodes. Example: `const items = item.getChildrenItems()`

## Data & Export

- `get(): T` ‚Äî Returns the raw underlying Figma node. Example: `const raw = item.get()`
- `exportJpg(): Promise<Uint8Array | string>` ‚Äî Asynchronously exports the node in JPG format. Example: `const bytes = await item.exportJpg()`
- `exportJson(): Promise<Uint8Array | string>` ‚Äî Asynchronously exports the node in JSON format. Example: `const json = await item.exportJson()`
- `getText(): string` ‚Äî Retrieves the trimmed text content from the node. Example: `const content = item.getText()`

## Types

### `UiFigmaNode`

A union type representing any relevant Figma node that can be wrapped by `FigmaItem`.
- Includes: `DocumentNode`, `PageNode`, `FrameNode`, `SectionNode`, `TextNode`, and other `SceneNode` variants.

### `UiFigmaExportFormat`

Supported formats for node export operations.
- Values: `'PNG'`, `'JPG'`, `'SVG'`, `'PDF'`, `'JSON_REST_V1'`.
