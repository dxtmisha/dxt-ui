# Lớp `FigmaItem`

Một lớp bọc đa năng để tương tác với bất kỳ loại node Figma nào. Nó cung cấp các phương thức được định kiểu để kiểm tra kiểu, điều hướng cây node và các hoạt động phổ biến như xuất dữ liệu và quản lý khung nhìn (viewport).

## Các tính năng chính

- **An toàn kiểu dữ liệu** — Sử dụng Type Guards (`isFrame`, `isText`, v.v.) để truy cập an toàn vào các thuộc tính cụ thể của một số loại node.
- **Điều hướng linh hoạt** — Các phương thức thuận tiện để truy xuất các phần tử cha hoặc các node con đã được bọc trong các thực thể `FigmaItem`.
- **Hành động tích hợp** — Hỗ trợ tích hợp để xuất các node sang JPG/JSON và điều khiển khung nhìn Figma (lựa chọn và thu phóng).

## Khởi tạo

Bọc bất kỳ node Figma nào trong một thực thể `FigmaItem` để truy cập các phương thức hỗ trợ của nó.

```typescript
import { FigmaItem } from '@dxtmisha/figma-code'

// Bọc một node (ví dụ: từ vùng lựa chọn)
const item = new FigmaItem(figma.currentPage.selection[0])

if (item.isFrame()) {
  console.log('Node là một frame:', item.getName())
}
```

## Các phương thức

### Kiểm tra kiểu (Type Checks)

Các phương thức để xác định kiểu node và trạng thái của nó.

- `isDocument(): boolean` — Trả về true nếu node là một Tài liệu (Document).
- `isFrame(): boolean` — Trả về true nếu node là một Frame.
- `isSection(): boolean` — Trả về true nếu node là một Section.
- `isText(): boolean` — Trả về true nếu node là một node Văn bản (Text).
- `isTextNoValue(): boolean` — Trả về true nếu node là một node văn bản có nội dung ý nghĩa.
- `inCurrentPage(): boolean` — Kiểm tra xem node có thuộc trang hiện đang hoạt động hay không.

### Truy cập dữ liệu (Accessors)

Các phương thức để truy xuất dữ liệu và các node liên quan.

- `get(): T` — Trả về node Figma gốc.
- `getId(): string` — Trả về định danh duy nhất của node.
- `getName(): string` — Trả về tên của node.
- `getType(): string` — Trả về chuỗi kiểu của node.
- `getText(): string` — Trả về nội dung văn bản (đối với node văn bản).
- `getParentItem(): FigmaItem | undefined` — Trả về node cha được bọc trong `FigmaItem`.
- `getParentPageItem(): FigmaItem | undefined` — Trả về trang cha được bọc trong `FigmaItem`.
- `getChildrenItems(): FigmaItem[]` — Trả về tất cả các node con được bọc trong `FigmaItem`.

### Hành động (Actions)

Các phương thức để thực hiện các hoạt động trên node.

- `exportJpg(): Promise<Uint8Array | string>` — Xuất node dưới dạng hình ảnh JPG.
- `exportJson(): Promise<Uint8Array | string>` — Xuất dữ liệu của node dưới định dạng JSON.
- `toSelection(): void` — Chọn node và thu phóng khung nhìn vào nó.
- `toPageAndSelection(): Promise<void>` — Chuyển sang trang của node và chọn nó.
