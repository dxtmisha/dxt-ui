# `useFigmaStorage`

Composable для управления данными в `PluginData` Figma. Использует паттерн singleton, гарантируя, что несколько вызовов с одинаковым ключом и узлом вернут один и тот же экземпляр `FigmaStorage`.

**Параметры:**
- `name: string` — уникальное имя ключа для элемента хранилища.
- `item?: PluginDataMixin` — (Опционально) узел Figma, в котором хранятся данные. По умолчанию — `figma.root`.
- `cache?: number` — (Опционально) время кэширования в секундах (TTL).

**Возвращает:**
- `FigmaStorage<T>` — экземпляр класса `FigmaStorage` для управления данными.

## Пример использования

```typescript
import { useFigmaStorage } from '@dxtmisha/figma-code'

// 1. Инициализация хранилища для настроек плагина
const settings = useFigmaStorage('settings')

// 2. Получение данных со значением по умолчанию
const theme = settings.get('dark')

// 3. Обновление данных
settings.set('light')
```
