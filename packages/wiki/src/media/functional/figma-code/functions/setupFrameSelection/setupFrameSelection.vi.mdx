# `setupFrameSelectionByMessage`

Một tiện ích để thiết lập trình xử lý tin nhắn cho phép giao diện người dùng (UI) yêu cầu chọn một frame cụ thể trong Figma. Nó đóng vai trò là cầu nối giữa `sendFrameSelection` của UI và `toFrameSelection` của plugin.

**Tham số:**
Không có.

**Trả về:**
`void`

## Ví dụ

<Source
  language='typescript'
  code={`
import { setupFrameSelectionByMessage } from '@dxtmisha/figma-code'

// Khởi tạo trong phần chính của plugin (thường là code.ts)
setupFrameSelectionByMessage()

// Bây giờ UI có thể gửi tin nhắn để thay đổi lựa chọn trong tài liệu.
  `}
/>
