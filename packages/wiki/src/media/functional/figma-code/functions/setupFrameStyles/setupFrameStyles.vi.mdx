# `setupFrameStyles`

Khởi tạo trình xử lý để truy xuất dữ liệu mở rộng về style của frame.

**Trả về:**
`void`

## Ví dụ

```typescript
import { setupFrameStyles } from '@dxtmisha/figma-code'

// Khởi tạo trong mã Backend của plugin
setupFrameStyles()
```
