# `initHeaders`

Khởi tạo các header API tiêu chuẩn và cung cấp chúng cho ứng dụng Vue thông qua cơ chế `provide` với khóa `NITRO_API_HEADERS`. Theo mặc định, nó thiết lập header `Content-Type` thành `text/html;charset=UTF-8`.

**Tham số:**
- `app: App<T>` — Thực thể ứng dụng Vue.

```typescript
import { initHeaders } from '@dxtmisha/nitro-basic'

initHeaders(app)
```
