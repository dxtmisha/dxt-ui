# `initServerStorage`

Khởi tạo `ServerStorage` để sử dụng ngữ cảnh sự kiện H3 (`event.context`). Điều này rất quan trọng để cách ly dữ liệu trong quá trình Server-Side Rendering (SSR) nhằm ngăn chặn rò rỉ dữ liệu giữa các yêu cầu.

**Tham số:**
- `event: H3Event` — Sự kiện H3 hiện tại.

**Trả về:**
`void`

```typescript
import { initCookieStorage } from '@dxtmisha/nitro-basic'
import { defineEventHandler } from 'h3'

export default defineEventHandler((event) => {
  initServerStorage(event)
})
```
