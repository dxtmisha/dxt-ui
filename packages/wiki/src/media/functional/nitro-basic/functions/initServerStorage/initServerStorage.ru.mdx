# `initServerStorage`

Инициализирует `ServerStorage` для использования контекста события H3 (`event.context`). Это критически важно для изоляции данных во время серверного рендеринга (SSR), чтобы предотвратить утечку данных между запросами.

**Параметры:**
- `event: H3Event` — Текущее событие H3.

**Возвращает:**
`void`

```typescript
import { initServerStorage } from '@dxtmisha/nitro-basic'
import { defineEventHandler } from 'h3'

export default defineEventHandler((event) => {
  initServerStorage(event)
})
```
