# `uiCookieStorage`

Cấu hình đối tượng `CookieStorage` toàn cục bằng cách liên kết nó với ngữ cảnh ứng dụng Nitro. Nó thiết lập các listener để lấy cookie từ yêu cầu hiện tại và ghi chúng vào các header phản hồi (`Set-Cookie`) bằng cách sử dụng composable `useHeaders`.

```typescript
import { uiCookieStorage } from '@dxtmisha/nitro-basic'

uiCookieStorage()
```
