# `uiIdServer`

Khởi tạo hàm tạo định danh duy nhất toàn cục cho các thành phần ở phía server. Nó sử dụng các giá trị của ngữ cảnh yêu cầu để tạo các ID duy nhất trong quá trình SSR.

```typescript
import { uiIdServer } from '@dxtmisha/nitro-basic'

uiIdServer()
```
