# `uiIdServer`

Initializes the global unique identifier generation function for elements on the server side. It uses request context values (retrieved via context helper) to generate safe IDs during SSR.

```typescript
import { uiIdServer } from '@dxtmisha/nitro-basic'

uiIdServer()
```
