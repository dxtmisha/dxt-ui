# `uiIdServer`

Инициализирует глобальную функцию генерации уникальных идентификаторов для элементов на стороне сервера. Она использует значения контекста запроса для генерации уникальных идентификаторов во время SSR.

```typescript
import { uiIdServer } from '@dxtmisha/nitro-basic'

uiIdServer()
```
