# `initCookieStorage`

Khởi tạo `CookieStorage` để làm việc với các cookie yêu cầu và phản hồi của Nitro/h3. Điều này đảm bảo rằng các thao tác cookie (lấy/thiết lập) hoạt động chính xác trong quá trình Server-Side Rendering (SSR).

**Tham số:**
- `event: H3Event` — Sự kiện H3 hiện tại (từ ngữ cảnh yêu cầu Nitro).
- `ageDefault: number = 604800` — Thời gian sống mặc định của cookie tính bằng giây (mặc định 7 ngày).
- `sameSiteDefault: 'strict' | 'lax' = 'strict'` — Giá trị thuộc tính SameSite mặc định.

**Trả về:**
`void`

```typescript
import { initCookieStorage } from '@dxtmisha/nitro-basic'
import { defineEventHandler } from 'h3'

export default defineEventHandler((event) => {
  initCookieStorage(event)

  // Bây giờ CookieStorage.get và CookieStorage.set sẽ sử dụng ngữ cảnh của Nitro
})
```
