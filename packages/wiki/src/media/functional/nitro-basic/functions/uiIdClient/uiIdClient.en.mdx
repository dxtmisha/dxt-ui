# `uiIdClient`

Initializes the global unique identifier generation function for elements on the client side. It uses an incremental counter starting from `10000` with the `ui-` prefix.

```typescript
import { uiIdClient } from '@dxtmisha/nitro-basic'

uiIdClient()
```
