# `uiIdClient`

Инициализирует глобальную функцию генерации уникальных идентификаторов для элементов на стороне клиента. Использует инкрементный счетчик, начиная с `10000` с префиксом `ui-`.

```typescript
import { uiIdClient } from '@dxtmisha/nitro-basic'

uiIdClient()
```
