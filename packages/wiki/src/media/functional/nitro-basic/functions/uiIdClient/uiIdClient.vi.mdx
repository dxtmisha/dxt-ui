# `uiIdClient`

Khởi tạo hàm tạo định danh duy nhất toàn cục cho các thành phần ở phía client. Nó sử dụng một bộ đếm tăng dần bắt đầu từ `10000` với tiền tố `ui-`.

```typescript
import { uiIdClient } from '@dxtmisha/nitro-basic'

uiIdClient()
```
