# `uiBootstrapClient`

Initializes the global unique identifier generator for elements on the client side. This function calls `uiId()` to ensure that ID generation is consistent with the Vue `useId` mechanism.

```typescript
import { uiBootstrapClient } from '@dxtmisha/nitro-basic'

// Called during client bootstrap
uiBootstrapClient()
```
