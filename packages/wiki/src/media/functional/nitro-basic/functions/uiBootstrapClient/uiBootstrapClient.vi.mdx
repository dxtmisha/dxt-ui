# `uiBootstrapClient`

Khởi tạo trình tạo định danh duy nhất toàn cục cho các thành phần ở phía client. Hàm này gọi `uiId()` để đảm bảo rằng việc tạo ID sẽ nhất quán với cơ chế `useId` của Vue.

```typescript
import { uiBootstrapClient } from '@dxtmisha/nitro-basic'

// Được gọi trong quá trình khởi động client
uiBootstrapClient()
```
