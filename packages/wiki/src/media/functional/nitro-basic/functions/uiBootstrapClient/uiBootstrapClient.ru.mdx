# `uiBootstrapClient`

Инициализирует глобальный генератор уникальных идентификаторов для элементов на стороне клиента. Эта функция вызывает `uiId()`, чтобы гарантировать, что генерация идентификаторов будет согласована с механизмом Vue `useId`.

```typescript
import { uiBootstrapClient } from '@dxtmisha/nitro-basic'

// Вызывается при запуске клиента
uiBootstrapClient()
```
