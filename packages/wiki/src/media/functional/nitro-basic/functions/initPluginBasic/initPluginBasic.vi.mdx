# `initPluginBasic`

Một hàm khởi tạo cho các khả năng cơ bản trong ứng dụng Nitro. Hiện tại, nó chịu trách nhiệm chính trong việc cấu hình bộ nhớ đệm API bằng cách sử dụng cấu hình được cung cấp.

**Tham số:**
- `_ : NitroApp` — Thực thể ứng dụng Nitro.
- `config: NitroAppBasicConfig` — Đối tượng cấu hình.

## Các kiểu

### NitroAppBasicConfig

- `api?: object` — Cài đặt API.
  - `cacheStorageKey: string` — tiền tố khóa sẽ được sử dụng trong bộ lưu trữ Nitro cho bộ nhớ đệm (mặc định `Ui_ApiCache`).
  - `cacheStepAgeClearOld: number` — ngưỡng tuổi của bộ nhớ đệm mà sau đó các mục cũ sẽ bị xóa.

**Ví dụ sử dụng:**

```typescript
import { initPluginBasic } from '@dxtmisha/nitro-basic'

export default defineNitroPlugin((nitroApp) => {
  initPluginBasic(nitroApp, {
    api: {
      cacheStorageKey: 'my_app_cache',
      cacheStepAgeClearOld: 3600000 // 1 giờ
    }
  })
})
```
