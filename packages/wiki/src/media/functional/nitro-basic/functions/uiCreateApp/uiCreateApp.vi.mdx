# `uiCreateApp`

Điểm nhập chính để tạo ứng dụng Vue SSR với các cấu hình hệ thống tiêu chuẩn. Nó tự động khởi tạo router và plugin chức năng cốt lõi.

**Tham số:**
- `appComponent: any` — Thành phần Vue gốc.
- `options: NitroAppOptions = {}` — Các tùy chọn cấu hình bao gồm API, bản dịch, icon và định tuyến.

**Trả về:**
- `app: App` — Thực thể ứng dụng Vue.
- `router: Router | undefined` — Thực thể router đã tạo hoặc được cung cấp.

```typescript
import { uiCreateApp } from '@dxtmisha/nitro-basic'
import App from './App.vue'

const { app, router } = uiCreateApp(App, {
  appRouter: {
    routes: [
      { path: '/', component: () => import('./Home.vue') }
    ]
  },
  api: { url: 'https://api.example.com' }
})
```
