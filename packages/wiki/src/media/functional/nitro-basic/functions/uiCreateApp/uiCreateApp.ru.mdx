# `uiCreateApp`

Основная точка входа для создания приложения Vue SSR со стандартными системными конфигурациями. Она автоматически инициализирует роутер и основной функциональный плагин.

**Параметры:**
- `appComponent: any` — Корневой компонент Vue.
- `options: NitroAppOptions = {}` — Опции конфигурации, включая API, переводы, иконки и маршрутизацию.

**Возвращает:**
- `app: App` — Экземпляр приложения Vue.
- `router: Router | undefined` — Созданный или предоставленный экземпляр роутера.

```typescript
import { uiCreateApp } from '@dxtmisha/nitro-basic'
import App from './App.vue'

const { app, router } = uiCreateApp(App, {
  appRouter: {
    routes: [
      { path: '/', component: () => import('./Home.vue') }
    ]
  },
  api: { url: 'https://api.example.com' }
})
```
