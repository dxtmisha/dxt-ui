# `uiCreateApp`

The main entry point for creating a Vue SSR application with standard system configurations. It automatically initializes the router and the core functional plugin.

**Parameters:**
- `appComponent: any` — The root Vue component.
- `options: NitroAppOptions = {}` — Configuration options including API, translations, icons, and routing.

**Returns:**
- `app: App` — The Vue application instance.
- `router: Router | undefined` — The created or provided router instance.

```typescript
import { uiCreateApp } from '@dxtmisha/nitro-basic'
import App from './App.vue'

const { app, router } = uiCreateApp(App, {
  appRouter: {
    routes: [
      { path: '/', component: () => import('./Home.vue') }
    ]
  },
  api: { url: 'https://api.example.com' }
})
```
