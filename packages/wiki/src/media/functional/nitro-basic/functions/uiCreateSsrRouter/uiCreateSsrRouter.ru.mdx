# `uiCreateSsrRouter`

Создаёт экземпляр Vue Router с автоматически выбранным режимом истории в зависимости от среды (Web History для браузера, Memory History для SSR).

**Параметры:**
- `routes: RouteRecordRaw[]` — Массив записей маршрутов.
- `options: RouterOptions = {}` — Дополнительные опции конфигурации маршрутизатора.

**Возвращает:**
`Router` — Экземпляр Vue Router.

```typescript
import { uiCreateSsrRouter } from '@dxtmisha/nitro-basic'

const routes = [
  { path: '/', component: () => import('./Home.vue') },
  { path: '/about', component: () => import('./About.vue') }
]

const router = uiCreateSsrRouter(routes, {
  scrollBehavior: () => ({ top: 0 })
})
```
