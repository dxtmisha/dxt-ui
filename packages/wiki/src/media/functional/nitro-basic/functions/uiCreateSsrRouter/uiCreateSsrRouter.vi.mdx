# `uiCreateSsrRouter`

Tạo một thực thể Vue Router với chế độ history được tự động lựa chọn dựa trên môi trường (Web History cho trình duyệt, Memory History cho SSR).

**Tham số:**
- `routes: RouteRecordRaw[]` — Một mảng các bản ghi route.
- `options: RouterOptions = {}` — Các tùy chọn cấu hình router bổ sung.

**Trả về:**
`Router` — Một thực thể Vue Router.

```typescript
import { uiCreateSsrRouter } from '@dxtmisha/nitro-basic'

const routes = [
  { path: '/', component: () => import('./Home.vue') },
  { path: '/about', component: () => import('./About.vue') }
]

const router = uiCreateSsrRouter(routes, {
  scrollBehavior: () => ({ top: 0 })
})
```
