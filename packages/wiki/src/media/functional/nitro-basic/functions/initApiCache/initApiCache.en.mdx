# `initApiCache`

Initializes the `ApiCache` to use Nitro's built-in storage engine (`useStorage`). This allows API response caching to persist using various Nitro storage drivers (memory, redis, fs, etc.).

**Parameters:**
- `storageKey: string = 'Ui_ApiCache'` — The Nitro storage mount point or prefix.
- `cacheStepAgeClearOld?: number` — Optional threshold for automatic cleanup of stale cache entries.

**Returns:**
`void`

```typescript
import { initApiCache } from '@dxtmisha/nitro-basic'

// Initialize in a Nitro plugin or server entry
initApiCache('cache', 3600)
```
