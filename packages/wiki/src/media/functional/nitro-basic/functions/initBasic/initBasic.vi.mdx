# `initBasic`

Tích hợp và khởi tạo các chức năng hệ thống cốt lõi cho ứng dụng dựa trên Nitro. Nó thiết lập bộ lưu trữ máy chủ, lưu trữ cache API và quản lý cookie trong vòng đời của Nitro.

**Tham số:**
- `nitroApp: NitroApp` — Thực thể ứng dụng Nitro.
- `config: NitroAppBasicConfig = {}` — Cấu hình cơ bản cho cache API và cookie.

**Trả về:**
`void`

```typescript
import { defineNitroPlugin } from 'nitropack/runtime'
import { initBasic } from '@dxtmisha/nitro-basic'

export default defineNitroPlugin((nitroApp) => {
  initBasic(nitroApp, {
    api: {
      cacheStorageKey: 'Ui_ApiCache'
    },
    cookie: {
      age: 60 * 60 * 24 * 7, // 7 ngày
      sameSite: 'lax'
    }
  })
})
```
