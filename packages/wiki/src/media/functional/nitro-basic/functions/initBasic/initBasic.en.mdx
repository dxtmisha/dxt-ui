# `initBasic`

Integrates and initializes core system functions for a Nitro-based application. It sets up server storage, API caching, and cookie management within the Nitro lifecycle.

**Parameters:**
- `nitroApp: NitroApp` — The Nitro application instance.
- `config: NitroAppBasicConfig = {}` — Basic configuration for API cache and cookies.

**Returns:**
`void`

```typescript
import { defineNitroPlugin } from 'nitropack/runtime'
import { initBasic } from '@dxtmisha/nitro-basic'

export default defineNitroPlugin((nitroApp) => {
  initBasic(nitroApp, {
    api: {
      cacheStorageKey: 'Ui_ApiCache'
    },
    cookie: {
      age: 60 * 60 * 24 * 7, // 7 days
      sameSite: 'lax'
    }
  })
})
```
