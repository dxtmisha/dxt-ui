# `initClientRouter`

Một hàm bất đối xứng giúp đợi việc khởi tạo và sự sẵn sàng hoàn toàn của Vue router. Điều này đảm bảo rằng tất cả các thành phần bất đối xứng và các hook điều hướng ban đầu được giải quyết trước khi ứng dụng tiếp tục thực thi (ví dụ: trước khi hydrat hóa trên client hoặc render trên server).

**Tham số:**
- `router: Router | undefined` — Thực thể Vue Router.

**Trả về:**
`Promise<void>`

```typescript
import { initClientRouter } from '@dxtmisha/nitro-basic'
import { createRouter } from './router'

const router = createRouter()
await initClientRouter(router)
```
