# `initScriptsJson`

Một hàm để tạo các kịch bản JSON cần thiết cho việc hydrat hóa dữ liệu ở phía client. Nó thu thập trạng thái hiện tại từ `ServerStorage` và dữ liệu hydrat hóa của `Api`, kết hợp chúng thành một chuỗi duy nhất của các script HTML.

**Trả về:**
`string` — Một chuỗi chứa các script JSON (ví dụ: `<script type="application/json">...</script>`).

```typescript
import { initScriptsJson } from '@dxtmisha/nitro-basic'

const hydrationScripts = initScriptsJson()
// Chèn hydrationScripts vào template HTML của bạn trước thẻ đóng </body>
```
