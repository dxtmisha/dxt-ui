# `initSsrApp`

Một hàm bất đối xứng giúp render ứng dụng Vue thành một chuỗi HTML và trích xuất thêm dữ liệu ngữ cảnh (ví dụ: teleports). Đây là bước cốt lõi trong việc tạo phản hồi từ server.

**Tham số:**
- `app: App<T>` — Thực thể ứng dụng Vue.
- `context: SSRContext` (mặc định `{}`) — Ngữ cảnh render server.

**Trả về:**
`Promise<{ appHtml, teleportsHtml, context }>` — Một đối tượng chứa HTML của ứng dụng, HTML của teleports và ngữ cảnh đã được cập nhật.

```typescript
import { initSsrApp } from '@dxtmisha/nitro-basic'

const { appHtml, teleportsHtml } = await initSsrApp(app)
```
