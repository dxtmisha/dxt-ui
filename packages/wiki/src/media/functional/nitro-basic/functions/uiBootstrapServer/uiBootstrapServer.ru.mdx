# `uiBootstrapServer`

Координирует глобальную инициализацию базовых компонентов в серверной среде. Она подготавливает сервер к обработке запросов, настраивая хранилище cookie, серверное хранилище и генератор уникальных идентификаторов.

Эта функция последовательно вызывает:
1. `uiCookieStorage()`
2. `uiServerStorage()`
3. `uiId()`

```typescript
import { uiBootstrapServer } from '@dxtmisha/nitro-basic'

// Вызывается при запуске сервера
uiBootstrapServer()
```
