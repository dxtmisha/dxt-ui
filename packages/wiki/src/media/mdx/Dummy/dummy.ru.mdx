Базовый компонент-заглушка (Dummy) для прототипирования и визуального тестирования интерфейсов. Позволяет быстро создавать блоки определенных размеров и цветов.

Dummy разработан как инструмент для разработчиков, помогающий структурировать интерфейсы до внедрения реальных компонентов.

**Основные возможности:**

- Настройка размеров через свойство size
- Установка цветовых схем через свойство palette
- Использование в качестве заполнителя пространства в макетах

**Типичные сценарии использования:**

- Прототипирование сеток и макетов
- Визуальное тестирование отступов и выравнивания
- Заглушки для еще не реализованных компонентов

```html
<script setup>
</script>

<template>
  <div class="dummy-examples" style="display: flex; gap: 16px; flex-direction: column;">
    <!-- Базовая заглушка -->
    <Dummy />

    <!-- Настройка размера и палитры -->
    <Dummy size="lg" palette="blue" />

    <!-- Маленькая красная заглушка -->
    <Dummy size="sm" palette="red" />
  </div>
</template>
```

> Dummy разработан как универсальный вспомогательный компонент для визуального планирования структуры интерфейса.
