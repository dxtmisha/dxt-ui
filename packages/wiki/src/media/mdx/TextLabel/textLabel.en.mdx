## Description

A simple yet flexible text display and label component with dynamic HTML wrapper support.

TextLabel is optimized for clean DOM output and standard typographic styling. It serves as a unified component for presenting headings, paragraphs, descriptions, or short metadata labels using the design system's cohesive styles.

**Key Features:**

- Dynamic wrapper tag support (e.g., span, div, p, h1-h6) via the `tag` prop
- Standardized text rendering using `LabelInclude` functionality
- Clean, semantic, and lightweight DOM footprints
- Fully integrated with layout design tokens

**Typical Usage Scenarios:**

- Formatting header/title text sections
- Presenting semantic description blocks
- Quick label annotations next to complex UI controls
- Applying unified typographic styles to raw strings
