## Button Component

The Button component is an interactive element for user actions. It supports various visual styles, sizes, and states to create a consistent user interface.

### Key Features

- Multiple visual variants for different contexts
- Responsive sizing system
- Built-in accessibility support
- State management (active, disabled, loading)
- Flexible styling options

### Usage

Buttons are used for:
- Primary actions (form submissions, confirmations)
- Secondary actions (cancellations, alternatives)
- Navigation between sections
- Triggering modal dialogs

### Best Practices

- Use clear and understandable labels
- Ensure adequate touch target size (minimum 44px)
- Provide visual feedback for all states
