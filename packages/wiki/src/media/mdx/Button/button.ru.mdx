## Компонент Button

Компонент Button представляет собой интерактивный элемент для выполнения действий пользователя. Он поддерживает различные визуальные стили, размеры и состояния для создания согласованного пользовательского интерфейса.

### Основные возможности

- Множественные визуальные варианты для разных контекстов
- Адаптивная система размеров
- Встроенная поддержка доступности
- Управление состояниями (активное, отключенное, загрузка)
- Гибкие возможности стилизации

### Использование

Кнопки используются для:
- Основных действий (отправка форм, подтверждения)
- Вторичных действий (отмена, альтернативы)
- Навигации между разделами
- Запуска модальных окон

### Рекомендации

- Используйте четкие и понятные подписи
- Обеспечивайте достаточный размер для касания (минимум 44px)
- Предоставляйте визуальную обратную связь для всех состояний
