## Управление значениями

Свойства `value`, `valueVariant` и `valueVariantHide` предназначены для управления значениями чекбокса при отправке форм и работе с двусторонней привязкой данных.

**Свойства:**

- `value` — текущее состояние чекбокса (true/false или пользовательские значения)
- `valueVariant` — значение, которое передаётся при отмеченном чекбоксе
- `valueVariantHide` — значение, которое передаётся при неотмеченном чекбоксе (по умолчанию: "0")

Свойства работают совместно: `value` определяет текущее состояние чекбокса и работает с `v-model` для двусторонней привязки. `valueVariant` позволяет установить пользовательское значение для отмеченного состояния (например, "yes", "1", "active"), которое будет передано при отправке формы. `valueVariantHide` автоматически создаёт скрытое поле input с указанным значением, которое отправляется, когда чекбокс не отмечен, обеспечивая наличие данных в форме даже при неактивном чекбоксе.

  />
