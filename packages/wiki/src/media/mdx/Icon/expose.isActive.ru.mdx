### `isActive`

Вычисляемое состояние, указывающее активно ли альтернативное представление (когда одновременно установлены `active` и `iconActive`).

**Тип:** `ComputedRef<boolean>`

```html
<script setup>
const componentRef = ref()

watch(() => componentRef.value?.isActive, (isActive) => {
  console.log('Альтернативное состояние активно:', isActive)

  if (isActive) {
    console.log('Отображается альтернативное представление')
  }
})
</script>

<template>
  <Component ref="componentRef" icon="home" iconActive="favorite" :active="true">
    <template #default>
      <p v-if="componentRef?.isActive">Активное состояние</p>
    </template>
  </Component>
</template>
```
