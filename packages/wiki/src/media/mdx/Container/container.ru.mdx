## Описание

Container - это базовый layout-компонент для ограничения ширины контента и управления горизонтальными отступами.
По умолчанию он помогает держать читаемую ширину блока и обеспечивает единый ритм макета между экранами.

Компонент поддерживает выравнивание через `align` (`left`, `center`, `right`) и работает с дизайн-токенами `width` и `marginX`.
Также можно использовать свойство `area` для интеграции с контекстом областей.

**Основные возможности:**

- Ограничение ширины контента через токен `width`
- Управление горизонтальными отступами через `marginX`
- Выравнивание контейнера: `left`, `center`, `right`
- Поддержка слота `default` для произвольного содержимого
- Поддержка свойства `area` для контекстного размещения

**Типичные сценарии использования:**

- Внешняя обертка секций страницы
- Ограничение ширины текста и форм
- Выравнивание контента в сетке и блочной структуре
- Базовый контейнер для повторно используемых layout-блоков
