Универсальный компонент для создания вкладок, элементов меню и навигационных ссылок с поддержкой иконок и бейджей.

TabItem объединяет в себе функциональность кнопки, ссылки и элемента списка. Он поддерживает различные состояния (активное, отключенное), может отображать иконки слева и справа от текста, а также интегрируется с компонентом Badge для отображения уведомлений или счетчиков. Компонент автоматически управляет ARIA-атрибутами для обеспечения доступности.

**Основные возможности:**

- Отображение метки (label) и иконок (icon, iconTrailing)
- Встроенная поддержка бейджа (badge) для счетчиков
- Управление состоянием выбора (selected) и активности
- Поддержка навигации через href (ссылка) или to (router-link)
- Настраиваемый корневой тег (button, a, div, li)
- Эффект ряби (ripple) при клике
- Поддержка скелетона (skeleton) для состояния загрузки

**Типичные сценарии использования:**

- Вкладки в навигационных панелях
- Элементы выпадающих меню
- Списки выбора (select options)
- Боковое меню навигации
- Ссылки с иконками и счетчиками

```html
<script setup>
import { ref } from 'vue'

const activeTab = ref('home')
</script>

<template>
  <TabItem
      label="Главная"
      icon="home"
      :selected="activeTab === 'home'"
      @click="activeTab = 'home'"
    />
</template>
```
