### `default`

Дефолтный слот, содержащий выделяемые дочерние элементы.

**Свойства:**
- `className: string` — Комбинированный CSS-класс для дочернего элемента
- `classSelection: string` — Комбинированный CSS-класс для состояния выделения дочернего элемента
- `classClick: string` — CSS-класс для элемента триггера клика
- `selected: string[]` — Массив текущих выбранных значений
- `onClick: (event: MouseEvent | TouchEvent) => void` — Функция для обработки клика по элементу
