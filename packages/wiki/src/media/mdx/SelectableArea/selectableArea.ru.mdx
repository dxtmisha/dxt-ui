Компонент для выбора дочерних элементов внутри области путем рисования прямоугольника выделения (выделение рамкой) или клика.

SelectableArea управляет выбором элементов по координатам, автоматически отображая визуальный прямоугольник выделения при перетаскивании курсора по контейнеру. Компонент вычисляет границы пересечения для дочерних элементов, соответствующих целевым селекторам, помечает их как выбранные и вызывает соответствующие события изменения состояния. Это полезно для реализации выделений в файловых менеджерах, сетках элементов или на интерактивных холстах.

**Основные возможности:**

- Выделение перетаскиванием мыши (выделение рамкой)
- Поддержка выделения по клику
- Настройка HTML-тега обертки (свойство `tag`)
- Расчет пересечений для вложенных элементов
- Реактивное связывание состояний для свойств `active` и `selected`
- Полная поддержка клавиатурной навигации и событий

**Типичные сценарии использования:**

- Выделение файлов/папок в стиле рабочего стола
- Групповое выделение элементов в таблицах и сетках
- Выделение графических объектов на холстах редактирования
- Интерактивные компоненты дашбордов
