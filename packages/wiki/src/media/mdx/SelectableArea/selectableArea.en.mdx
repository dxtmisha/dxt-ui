Component for selecting child elements within an area by drawing a selection rectangle (drag-to-select) or clicking.

SelectableArea manages items selection by coordinates, automatically drawing a visual selection rectangle as the user drags their cursor across the container. The component calculates collision bounds of child elements matching the target selectors, marks them as selected, and triggers corresponding status events. It is useful for implementing advanced file manager selections, grid item selections, or drawing areas.

**Key Features:**

- Mouse drag selection (box selection / marquee selection)
- Click selection support
- Custom wrapper HTML tag (`tag` property)
- Collision box calculations for nested elements
- Reactive state bindings for `active` and `selected` properties
- Fully accessible keyboard navigation and events support

**Common Use Cases:**

- Desktop-like file/folder selections in directories
- Batch item selection in grid and list views
- Selecting multiple graphic objects in editing canvases
- Interactive dashboard components
