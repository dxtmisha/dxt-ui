## Управление состоянием через v-model

Двусторонняя привязка активного состояния выделения через `v-model:selected`.

**Параметры:**
- `selected: string | string[]` — массив или строка текущих выбранных значений

<Source
    code={`
<script setup>
import { ref } from 'vue'

const selectedItems = ref(['item-1'])
</script>

<template>
  <SelectableArea v-model:selected="selectedItems">
    <template #default="{ className, classSelectionName, classNameClick, onClick }">
      <div
        :class="[className, selectedItems.includes('item-1') && classSelectionName]"
        data-value="item-1"
        @click="onClick"
      >
        Элемент 1
      </div>
    </template>
  </SelectableArea>
</template>
  `}
    language="html"
/>
