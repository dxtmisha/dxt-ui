### `default`

Default slot containing the selectable child elements.

**Properties:**
- `className: string` — Combined CSS class for the child item
- `classSelection: string` — Combined CSS class for the selected state of the child item
- `classClick: string` — CSS class for the click trigger element
- `selected: string[]` — Array of currently selected values
- `onClick: (event: MouseEvent | TouchEvent) => void` — Function to handle item clicks
