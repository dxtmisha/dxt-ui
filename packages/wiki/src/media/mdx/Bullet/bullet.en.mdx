## Description

A bullet list container component designed to present structured, clean lists.

The Bullet component is highly flexible, supporting rendering list items dynamically from a string array, forwarding properties and attributes to child items, or rendering direct HTML content with auto-styling of list items. It integrates seamlessly into standard content layout flows and respects the theme's spacing and list aesthetics.

**Key Features:**

- Automated list rendering from a string array (`list` prop)
- Forwarding custom attributes and settings to child list elements (`itemAttrs` prop)
- Support for direct HTML lists (`html` prop), parsing `<li>` tags and wrapping them with standard design system class names automatically
- Default slot to render custom layout structures or nested bullet list items manually

**Typical Usage Scenarios:**

- Displaying bulleted lists of text features, rules, or key metrics
- Displaying formatted lists inside information blocks, cards, or notifications
- Rendering raw HTML lists from markdown or CMS endpoints while maintaining uniform layout styles
