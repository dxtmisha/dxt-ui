## Описание

Компонент-контейнер для маркированного списка, разработанный для отображения чистых и структурированных списков.

Компонент Bullet обладает высокой гибкостью: он поддерживает динамический рендеринг элементов списка из массива строк, передачу свойств и атрибутов дочерним элементам, а также рендеринг прямого HTML-содержимого с автостилизацией элементов списка. Он органично интегрируется в макет страницы и поддерживает единые отступы и стили.

**Основные возможности:**

- Автоматический рендеринг списка из массива строк (пропс `list`)
- Передача пользовательских атрибутов и настроек дочерним элементам списка (пропс `itemAttrs`)
- Поддержка списков в формате прямого HTML (пропс `html`), с автоматическим парсингом тегов `<li>` и оборачиванием их в стандартные классы дизайн-системы
- Слот по умолчанию для ручного вывода кастомных структур разметки или вложенных элементов списка

**Типичные сценарии использования:**

- Отображение маркированных списков текстовых характеристик, правил или ключевых показателей
- Вывод форматированных списков внутри информационных блоков, карточек или уведомлений
- Рендеринг сырых HTML-списков из Markdown или API CMS с сохранением единых стилей оформления
