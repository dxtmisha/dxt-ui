### Содержимое HTML

Компонент Bullet поддерживает рендеринг прямых HTML-строк с помощью свойства `html`.

Когда предоставляется HTML, компонент сканирует строку на наличие стандартных элементов `<li>` и автоматически заменяет/добавляет к ним имена классов элементов списка системы. Это гарантирует, что любые списки, отображаемые из внешних API или редакторов форматированного текста, наследуют точный внешний вид дизайн-системы.
