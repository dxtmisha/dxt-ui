## Описание

Компонент для отображения заголовков с интегрированной поддержкой иконок и подписей.

Header позволяет гибко управлять HTML-тегом (h1-h6, div), что важно для соблюдения иерархии документа. Компонент поддерживает автоматическое управление областями (area) и предоставляет слот для кастомизации контента в правой части.

**Основные возможности:**

- Отображение заголовка (label), подписи (caption) и иконки
- Настройка HTML-тега (h1, h2, h3, h4, h5, h6, div и др.)
- Автоматическое управление значением области через AreaInclude
- Поддержка слота trailing для дополнительного контента
- Стандартизированная типографика и отступы

**Типичные сценарии использования:**

- Заголовки разделов и страниц
- Группировка контента с поясняющими подписями
- Навигационные заголовки с иконками
- Шапки карточек и блоков
