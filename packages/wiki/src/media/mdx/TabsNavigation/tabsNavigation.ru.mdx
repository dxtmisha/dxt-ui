Компонент для создания горизонтальной навигации с вкладками.

TabsNavigation управляет списком вкладок, их выбором и прокруткой. Поддерживает различные стили отображения, управление с клавиатуры и адаптивность.

**Основные возможности:**

- Горизонтальная прокрутка вкладок
- Управление выбранной вкладкой через `v-model:selected`
- Поддержка различных типов данных (массив объектов или объект)
- Настройка внешнего вида (теги, атрибуты, разделители)
- Поддержка клавиатурной навигации (стрелки, Enter, Пробел)
- Анимация индикатора активной вкладки

**Типичные сценарии использования:**

- Навигация между разделами страницы
- Переключение видов отображения данных
- Фильтрация контента по категориям
- Главное меню приложения (в стиле вкладок)

```html
<script setup>
import { ref } from 'vue'

const selectedTab = ref('home')
const tabs = [
  { label: 'Главная', value: 'home', icon: 'home' },
  { label: 'Профиль', value: 'profile', icon: 'person' },
  { label: 'Настройки', value: 'settings', icon: 'settings' }
]
</script>

<template>
  <TabsNavigation
    v-model:selected="selectedTab"
    :list="tabs"
    horizontal-scroll-align="center"
  />
</template>
```
