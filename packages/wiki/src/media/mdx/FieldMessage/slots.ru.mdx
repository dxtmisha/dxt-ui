### `helper`

Слот для размещения вспомогательного сообщения. Отображается только когда отсутствует сообщение валидации.

**Параметры:**
- `props: FieldMessageSlot` — объект с данными сообщений

### `validation`

Слот для размещения сообщения валидации. Имеет приоритет над вспомогательным сообщением.

**Параметры:**
- `props: FieldMessageSlot` — объект с данными сообщений

**Структура FieldMessageSlot:**

Объект, передаваемый в слоты компонента FieldMessage, содержащий данные для отображения сообщений.

- `message: string` — текущее сообщение (validation или helper) для отображения в активном слоте.
- `helperMessage: string` — текст вспомогательного сообщения, переданный через свойства компонента.
- `validationMessage: string` — текст сообщения валидации, переданный через свойства компонента. При наличии перекрывает `helperMessage`.
