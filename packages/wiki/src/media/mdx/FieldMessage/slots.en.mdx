### `helper`

Slot for placing helper message. Displays only when validation message is absent.

**Parameters:**
- `props: FieldMessageSlot` — object with message data

### `validation`

Slot for placing validation message. Has priority over helper message.

**Parameters:**
- `props: FieldMessageSlot` — object with message data

**FieldMessageSlot Structure:**

Object passed to FieldMessage component slots, containing data for displaying messages.

- `message: string` — current message (validation or helper) for display in the active slot.
- `helperMessage: string` — helper message text passed through component properties.
- `validationMessage: string` — validation message text passed through component properties. When present, it overrides the `helperMessage`.
