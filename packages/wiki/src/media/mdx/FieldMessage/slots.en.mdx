- `helper(props: FieldMessageSlot): any` — Slot for custom helper message content.
- `validation(props: FieldMessageSlot): any` — Slot for custom validation message content.

> **FieldMessageSlot:**
> - `message: string` — current message (helper or validation).
> - `helperMessage: string` — helper message text.
> - `validationMessage: string` — validation message text.
