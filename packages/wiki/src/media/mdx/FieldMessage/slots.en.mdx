### `helper`

Slot for placing helper message. Displays only when validation message is absent.

**Parameters:**
- `props: FieldMessageSlot` — object with message data

```html
<FieldMessage helper-message="Enter your name">
  <template #helper="{ helperMessage }">
    <span class="custom-helper">
      <i class="icon-info"></i> {{ helperMessage }}
    </span>
  </template>
</FieldMessage>
```

### `validation`

Slot for placing validation message. Has priority over helper message.

**Parameters:**
- `props: FieldMessageSlot` — object with message data

```html
<FieldMessage validation-message="This field is required">
  <template #validation="{ validationMessage }">
    <span class="custom-error">
      <i class="icon-warning"></i> {{ validationMessage }}
    </span>
  </template>
</FieldMessage>
```

## FieldMessageSlot

`FieldMessageSlot` — object passed to FieldMessage component slots, containing data for displaying messages.

### Type Structure

```typescript
type FieldMessageSlot = {
  /** Current displayed message */
  message?: string

  /** Helper message text */
  helperMessage?: string

  /** Validation message text */
  validationMessage?: string
}
```

### Properties

- **`message`** — current message to display (validation or helper, depending on active slot)
- **`helperMessage`** — helper message text from component property
- **`validationMessage`** — validation message text from component property

### Display Priority

Component shows only one slot at a time:

1. **Validation** — if `validationMessage` is provided or `validation` slot is defined
2. **Helper** — if validation is absent, but `helperMessage` is provided or `helper` slot is defined

### Usage Example

```html
<script setup>
import { ref } from 'vue'

const inputValue = ref('')
const error = ref('')

const validateField = (value) => {
  if (value.length < 3) {
    error.value = 'Minimum 3 characters'
  } else {
    error.value = ''
  }
}
</script>

<template>
  <input
    v-model="inputValue"
    @input="validateField(inputValue)"
  />

  <FieldMessage
    helper-message="Enter name (minimum 3 characters)"
    :validation-message="error"
  >
    <template #helper="{ helperMessage }">
      <div class="hint">
        💡 {{ helperMessage }}
      </div>
    </template>

    <template #validation="{ validationMessage }">
      <div class="error">
        ⚠️ {{ validationMessage }}
      </div>
    </template>
  </FieldMessage>
</template>
```

In this example:
- With empty field, `helper` slot is displayed
- With input less than 3 characters, `validation` slot is displayed
- With input 3+ characters, `helper` slot is displayed again
