## Работа с валютой

Свойства `type`, `currency`, `currencyHide`, `language` и `fraction` предназначены для управления форматированием денежных значений и отображением символа валюты.

**Свойства:**

- `type` — тип инпута, для валюты устанавливается значение `currency`
- `currency` — код валюты (например, `USD`, `EUR`, `RUB`) для отображения символа
- `currencyHide` — скрывает отображение символа валюты в поле ввода
- `language` — код локали (например, `ru-RU`, `en-US`) для форматирования по региональным стандартам
- `fraction` — количество знаков после запятой (по умолчанию `2` для валюты)

Свойства работают совместно: при установке `type="currency"` автоматически активируется режим валютного форматирования с двумя знаками после запятой.

Свойство `currency` определяет символ валюты, который добавляется к отформатированному числу (например, `$`, `€`, `₽`).

Если установлен флаг `currencyHide = true`, символ валюты не отображается в поле, но форматирование числа сохраняется.

Свойство `language` влияет на региональные особенности форматирования: разделители тысяч, десятичный разделитель и позицию символа валюты.

Количество десятичных знаков можно изменить через `fraction`, перезаписав значение по умолчанию.

```html
<script setup>
import { ref } from 'vue'

const price = ref('1234.56')
const salary = ref('50000')
const amount = ref('999.999')
</script>

<template>
  <!-- Стандартный валютный инпут с долларом -->
  <Input
    v-model="price"
    label="Цена"
    type="currency"
    currency="USD"
    language="en-US"
  />

  <!-- Валюта с рублём и русской локалью -->
  <Input
    v-model="salary"
    label="Зарплата"
    type="currency"
    currency="RUB"
    language="ru-RU"
  />

  <!-- Валюта без отображения символа -->
  <Input
    v-model="price"
    label="Сумма"
    type="currency"
    currency="EUR"
    currency-hide
    language="de-DE"
  />

  <!-- Валюта с тремя знаками после запятой -->
  <Input
    v-model="amount"
    label="Точная сумма"
    type="currency"
    currency="USD"
    language="en-US"
    :fraction="3"
  />

  <!-- Только числовое форматирование без валюты -->
  <Input
    v-model="price"
    label="Число"
    type="number-format"
    language="ru-RU"
  />
</template>
```
