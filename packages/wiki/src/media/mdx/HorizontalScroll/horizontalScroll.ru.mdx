Компонент для создания горизонтально прокручиваемого контейнера с поддержкой плавного скролла и автоматического позиционирования.

HorizontalScroll предоставляет удобный способ отображения контента, который не помещается по ширине экрана, например, галерей изображений, списков карточек или навигационных меню. Компонент поддерживает автоматическую прокрутку к выбранному элементу при инициализации и предоставляет методы для программного управления скроллом.

**Основные возможности:**

- Горизонтальная прокрутка контента
- Плавный скролл (smooth scrolling)
- Автоматическая прокрутка к активному элементу при загрузке
- Поддержка пользовательских тегов для контейнера
- Слот с передачей привязок (binds) для дочерних элементов
- Управление видимостью скроллбара (flush режим)

**Типичные сценарии использования:**

- Галереи изображений и медиа-контента
- Горизонтальные списки карточек товаров
- Навигационные меню с прокруткой (табы)
- Списки тегов или категорий
- Таймлайны и последовательности шагов

```html
<script setup>
import { ref } from 'vue'

const items = ref([
  { id: 1, title: 'Элемент 1' },
  { id: 2, title: 'Элемент 2' },
  { id: 3, title: 'Элемент 3' },
  { id: 4, title: 'Элемент 4' },
  { id: 5, title: 'Элемент 5' }
])
</script>

<template>
  <HorizontalScroll>
    <template #default="{ binds }">
      <div
        v-for="item in items"
        :key="item.id"
        v-bind="binds"
        class="scroll-item"
      >
        {{ item.title }}
      </div>
    </template>
  </HorizontalScroll>
</template>

<style scoped>
.scroll-item {
  min-width: 150px;
  height: 100px;
  margin-right: 16px;
  background: #f5f5f5;
  border-radius: 8px;
  display: flex;
  align-items: center;
  justify-content: center;
}
</style>
```
