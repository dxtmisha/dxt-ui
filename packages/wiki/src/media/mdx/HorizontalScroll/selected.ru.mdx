## Выбранный элемент

Свойство `selected` позволяет управлять визуальным состоянием компонента, указывая, что он находится в выбранном или активном состоянии.

**Свойства:**

- `selected` — булево значение, определяющее, выбран ли элемент

Когда свойство `selected` установлено в `true`, к компоненту добавляется специальный класс (обычно `--selected`), который изменяет его внешний вид (например, цвет фона, границы или текста), чтобы выделить его среди других элементов. Это полезно для реализации списков выбора, активных полей ввода или других интерактивных элементов, где необходимо визуально обозначить текущий выбор пользователя.

```html
<script setup>
import { ref } from 'vue'

const isSelected = ref(true)
</script>

<template>
  <!-- Выбранное состояние -->
  <Textarea
    v-model="value"
    :selected="isSelected"
    label="Активное поле"
  />

  <!-- Обычное состояние -->
  <Textarea
    v-model="value2"
    :selected="false"
    label="Обычное поле"
  />
</template>
```
