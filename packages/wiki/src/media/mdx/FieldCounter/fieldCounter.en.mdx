FieldCounter is a component that displays character count for form fields. Shows current character count and input limits.

### Purpose

- **Character counting** — Display current input character count
- **Input limits** — Show character restrictions
- **Custom templates** — Flexible display formats
- **Auto visibility** — Smart display management

### Core capabilities

- **Real-time counting** — Instant counter updates
- **Limit management** — Maximum character limit display
- **Template rendering** — Customizable formats with [c] and [m] placeholders
- **Flexible values** — Support for strings and numbers

### Integration

Use FieldCounter with text inputs and textareas to display character count and limit information.

> This component focuses on counter display and should be used alongside input fields.
