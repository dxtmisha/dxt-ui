## Пользовательские шаблоны

Используйте простые строковые шаблоны с плейсхолдерами для форматирования счетчика.

### Плейсхолдеры

- `[c]` — текущее число символов
- `[m]` — максимальный лимит символов

### Примеры

```html
<!-- Описательный формат -->
<FieldCounter
  :counter="75"
  :maxlength="150"
  template="Символов: [c] из [m]"
/>
```
