- `[area]: any` — Динамический слот для рендеринга контента. Текущее значение области (area) используется как имя слота для выбора соответствующего контента.
- `default: any` — Слот по умолчанию, который отображается, если для текущего значения области не предоставлен специфический слот или если значение области не определено.
