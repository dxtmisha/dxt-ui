The baseline table cell component representing a `<td>` or other custom tags. Used inside rows to display text, labels, and descriptions, and supports structural layout attributes like colspan and rowspan.

**Key Features:**

- Standard and custom HTML tag support (`tag` prop, defaults to `td`)
- Cell alignment options (`align` and `alignVertical`)
- Text clamping for cells with long labels (`labelClamp`)
- Dynamic skeleton states (`isSkeleton`) for loading phases
- Sticky columns and headers support (`stickyLeft`, `stickyTop`)
- Selectable and interactive states (`selected`, `disabled`)
