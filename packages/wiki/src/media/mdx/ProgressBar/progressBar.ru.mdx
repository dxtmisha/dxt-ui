## ProgressBar и связанные параметры

Компонент `ProgressBar` отображает линейную шкалу прогресса с одним или несколькими сегментами. Он поддерживает расчет процентов, пользовательские цвета, состояния фокусировки и настройки разметки.

**Свойства:**

- `value` — текущее значение прогресса (число) или список сегментов шкалы прогресса (массив элементов с `value`, `label`, `color` и др.)
- `max` — максимальное значение прогресса (по умолчанию `100` или сумма значений сегментов)
- `showLabel` — включает отображение меток с деталями внутри отдельных сегментов

<Source
    code={`
<!-- Прогресс с одиночным значением -->
<ProgressBar :value="45" />

<!-- Многосегментный прогресс с пользовательскими метками -->
<ProgressBar :value="[
  { value: 20, label: 'Часть А', color: '#ff0000' },
  { value: 50, label: 'Часть Б', palette: 'primary' }
]" showLabel />
  `}
