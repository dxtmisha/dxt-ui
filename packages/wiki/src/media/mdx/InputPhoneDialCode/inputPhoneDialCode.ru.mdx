Кнопка-триггер для выбора кода страны телефонного номера через всплывающее меню стран.

InputPhoneDialCode — это компактная кнопка-триггер, которая открывает всплывающее меню **MenuCountry**, позволяя пользователю выбрать страну и соответствующий телефонный код. Компонент автоматически определяет текущую страну пользователя через `Geo.getCountry()` в качестве значения по умолчанию и реактивно привязывает выбор через `v-model:selected`.

**Основные возможности:**

- Автоматическое определение страны по умолчанию через `Geo.getCountry()`
- Выбор страны через всплывающее меню `MenuCountry` с флагом и кодом телефона
- Настраиваемая метка: телефонный код (например `+7`), название страны или без метки
- Иконка-стрелка отражает состояние открытия/закрытия меню
- Наследует все возможности компонентов Button и MenuCountry

**Типичные сценарии использования:**

- Префикс поля ввода телефонного номера для выбора кода страны
- Выбор страны/региона в формах регистрации или оформления заказа
