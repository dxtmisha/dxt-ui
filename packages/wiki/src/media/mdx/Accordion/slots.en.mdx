- `default(props: MotionTransformControlItem): any` — Slot for placing the main accordion content that appears when the section is expanded.

> **MotionTransformControlItem:**
> - `isOpen: ComputedRef<boolean>` — reactive open state.
> - `isShow: ComputedRef<boolean>` — visible or preparing to show.
> - `classes: MotionTransformClassList` — helper CSS classes.
> - `idControl: string` — identifier for the control element.
> - `idBody: string` — identifier for the body element.
> - `binds: AriaRoleControlPropsInclude` — binding attributes for accessibility.
