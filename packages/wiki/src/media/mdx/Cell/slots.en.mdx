- `trailing(props: CellClassesSub): any` — Slot for placing control elements or indicators at the end of the cell (icons, switches, buttons).
- `body(props: CellClassesSub): any` — Slot for placing additional content at the bottom of the cell (metadata, secondary actions, details).

> **CellClassesSub:**
> - `label: string` — CSS class for the label element (`{className}__label`).
> - `description: string` — CSS class for the description element (`{className}__description`).
> - `caption: string` — CSS class for the caption element (`{className}__caption`).
> - `trailing: string` — CSS class for the trailing area element (`{className}__trailing`).
