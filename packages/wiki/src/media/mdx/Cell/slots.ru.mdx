- `trailing(props: CellClassesSub): any` — Слот для размещения элементов управления или индикаторов в конце ячейки (иконки, переключатели, кнопки).
- `body(props: CellClassesSub): any` — Слот для размещения дополнительного контента в нижней части ячейки (метаданные, вторичные действия, детали).

> **CellClassesSub:**
> - `label: string` — CSS-класс для элемента метки (`{className}__label`).
> - `description: string` — CSS-класс для элемента описания (`{className}__description`).
> - `caption: string` — CSS-класс для элемента подписи (`{className}__caption`).
> - `trailing: string` — CSS-класс для элемента области trailing (`{className}__trailing`).
