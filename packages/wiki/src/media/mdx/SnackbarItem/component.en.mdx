## Custom Component

The `component` and `componentProps` properties allow you to insert a custom Vue component inside the SnackbarItem. This is useful when the standard fields (label, description) are not enough to display complex content.

**Features:**
- `component` — accepts a component object or its name.
- `componentProps` — an object with props that will be passed to the component.

The component is rendered inside a special container with the main content area class, ensuring proper padding and styling.
