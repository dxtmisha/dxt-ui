### `close`

The event is triggered when the element is closed (clicking the close button or programmatic closing).

**Parameters:**
- `value: string | undefined` — element identification (the value of the `value` property)

```html
<script setup>
const onClose = (value) => {
  console.log('Element closed:', value)
}
</script>

<template>
  <SnackbarItem value="item-1" @close="onClose" />
</template>
```
