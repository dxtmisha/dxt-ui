## Описание

Компонент для управления рендерингом контента исключительно на стороне клиента.

ClientOnly является важным инструментом для обеспечения совместимости с Server-Side Rendering (SSR) и Static Site Generation (SSG). Он предотвращает попытки выполнения кода, зависящего от браузерных API (например, window или document), на стороне сервера, тем самым избегая ошибок гидратации.

**Основные возможности:**

- Отложенный рендеринг до момента монтирования в браузере
- Предотвращение ошибок рассогласования гидратации (hydration mismatch)
- Возможность принудительного включения/выключения через свойство clientOnly
- Легкая интеграция любых сторонних библиотек, не поддерживающих SSR

**Типичные сценарии использования:**

- Интерактивные карты и графики
- Компоненты, использующие localStorage или другие Web API
- Сторонние виджеты, требующие наличия объекта window
- Оптимизация производительности в средах SSR
