# Adaptive

Свойство `adaptive` включает режим выравнивания масштаба изображений.
Для корректной работы нужно задать физический размер объекта на
изображении при помощи `object-width` или `object-height`. Элементы
с одинаковым значением `adaptive-group` будут вычислять общий масштаб
и выглядеть пропорционально.

```html
<D1Image adaptive object-width="63.5"/>
```

При необходимости постоянных вычислений, даже когда элемент скрыт за
пределами экрана, используйте `adaptive-always`.
