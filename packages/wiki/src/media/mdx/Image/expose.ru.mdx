### `type`

Тип загруженного изображения.

**Тип:** `ComputedRef<ImageTypeItem>`

**Возможные значения:**

```ts
enum ImageTypeValue {
  pdf = 'pdf',
  file = 'file',
  image = 'image',
  flag = 'flag',
  color = 'color',
  public = 'public',
  filled = 'filled',
  outlined = 'outlined',
  round = 'round',
  sharp = 'sharp',
  twoTone = 'two-tone',
  material = 'material',
  icon = 'icon'
}
```

### `data`

Информация о загруженном изображении, включая его размеры и источник.

**Тип:** `Ref<ImageEventItem>`
