## Description

Компонент навигации, показывающий иерархическую цепочку ссылок.

Компонент Breadcrumbs отображает цепочку элементов BreadcrumbItem, представляющих собой путь к текущей странице. Он поддерживает автоматический рендеринг списка из структурированного массива свойств и автоматически настраивает последний элемент как статический (только для чтения), чтобы обозначить текущую активную страницу.

**Ключевые особенности:**

- Автоматический рендеринг списка на основе массива свойств
- Автоматическое конфигурирование последнего элемента как статического/только для чтения
- Распространение событий клика с полной информацией
- Пользовательские стили обертки и flex-выравнивание
