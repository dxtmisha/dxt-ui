## Description

Navigation helper component showing a hierarchical trail of links.

Breadcrumbs displays a trail of BreadcrumbItem elements representing the path to the current page. It supports automatic list rendering from a structured properties array and automatically configures the final item as static (readonly) to denote the current active page.

**Key features:**

- Automatic list rendering based on properties array
- Last item automatically configured as static/readonly
- Click event propagation with full details
- Custom wrapper styles and flex alignment
