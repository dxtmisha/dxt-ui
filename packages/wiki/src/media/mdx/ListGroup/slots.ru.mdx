### `head`

Слот для размещения содержимого заголовка группы с функциональностью переключения.

**Параметры:**
- `open: boolean` — текущее состояние открытия/закрытия группы

```html
<ListGroup>
  <template #head="{ open }">
    <div class="group-header">
      <h3>{{ open ? '▼' : '▶' }} Название группы</h3>
    </div>
  </template>
</ListGroup>
```

### `list`

Слот для размещения содержимого сгруппированных элементов списка.

```html
<ListGroup>
  <template #list>
    <ListItem text="Элемент 1" />
    <ListItem text="Элемент 2" />
    <ListItem text="Элемент 3" />
  </template>
</ListGroup>
```
