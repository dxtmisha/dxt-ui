Компонент для группировки и организации элементов списка со сворачиваемой структурой.

ListGroup является вспомогательным компонентом для List, управляющим иерархической организацией контента с возможностью разворачивания/сворачивания групп. Компонент интегрируется с MotionTransform для плавных анимаций, поддерживает неограниченную вложенность групп и обеспечивает удобную навигацию по структурированным данным.

**Основные возможности:**

- Иерархическая организация контента с неограниченной вложенностью
- Анимации разворачивания/сворачивания групп через MotionTransform
- Управление состоянием открытия через props или внутренний state
- Автоматическое управление видимостью содержимого группы
- Интеграция с экосистемой компонентов List
- Поддержка кастомизации через слоты head и list
- События изменения состояния группы

**Типичные сценарии использования:**

- Группировка элементов списка по категориям
- Создание древовидных структур навигации
- Организация настроек и опций по разделам
- Многоуровневые меню и каталоги
- FAQ секции с раскрывающимися ответами

```html
<script setup>
import { ref } from 'vue'

const items = [
  { text: 'Элемент 1', value: '1' },
  { text: 'Элемент 2', value: '2' },
  { text: 'Элемент 3', value: '3' }
]

const isOpen = ref(true)
</script>

<template>
  <List>
    <ListGroup v-model:open="isOpen">
      <template #head="{ open }">
        <div class="group-header">
          <span>{{ open ? '▼' : '▶' }}</span>
          <span>Группа элементов</span>
        </div>
      </template>

      <template #list>
        <ListItem
          v-for="item in items"
          :key="item.value"
          :text="item.text"
        />
      </template>
    </ListGroup>
  </List>
</template>
```
