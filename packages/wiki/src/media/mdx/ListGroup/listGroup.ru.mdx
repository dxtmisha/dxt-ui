Компонент ListGroup представляет собой вспомогательный компонент для компонента List, предназначенный для группировки и организации элементов списка со сворачиваемой структурой. Он обеспечивает иерархическую организацию контента с возможностью сворачивания/разворачивания групп, поддерживает вложенные группы и интегрируется с экосистемой компонентов List.
