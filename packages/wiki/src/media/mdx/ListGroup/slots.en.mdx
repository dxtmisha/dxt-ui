### `head`

Slot for placing the group header content with toggle functionality.

**Parameters:**
- `open: boolean` — current open/close state of the group

```html
<ListGroup>
  <template #head="{ open }">
    <div class="group-header">
      <h3>{{ open ? '▼' : '▶' }} Group Name</h3>
    </div>
  </template>
</ListGroup>
```

### `list`

Slot for placing the grouped list items content.

```html
<ListGroup>
  <template #list>
    <ListItem text="Item 1" />
    <ListItem text="Item 2" />
    <ListItem text="Item 3" />
  </template>
</ListGroup>
```
