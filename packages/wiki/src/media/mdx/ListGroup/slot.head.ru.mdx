### `head`

Слот для размещения содержимого заголовка группы с функциональностью переключения.

- Содержит кликабельный элемент заголовка, который управляет разворачиванием/сворачиванием группы
- Получает параметр `open`, указывающий текущее состояние группы
- Обычно содержит название группы, иконку и элементы управления переключением
- Управляет визуальным состоянием заголовка группы
