### `list`

Слот для размещения содержимого сгруппированных элементов списка.

- Содержит все элементы списка, принадлежащие данной группе
- Содержимое показывается/скрывается в зависимости от открытого/закрытого состояния группы
- Обычно содержит компоненты ListItem или вложенные компоненты ListGroup
- Поддерживает иерархическую организацию контента с неограниченными уровнями вложенности
