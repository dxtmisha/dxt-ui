## Описание

Компонент для отображения чисел, валют и единиц измерения с форматированием, автоматически применяющий стандарты локализации.

FormattedNumber предоставляет простой и чистый интерфейс для вывода числовых данных без лишней верстки. Он поддерживает кастомную настройку локали, форматирование валюты по стандартным кодам, форматирование единиц измерения, а также возможность выборочного отключения форматирования.

**Ключевые возможности:**

- Стандартное локализованное форматирование чисел (десятичные знаки, разделители).
- Форматирование валют с поддержкой стандартных кодов (например, USD, EUR, RUB).
- Поддержка форматирования единиц измерения (например, килограмм, литр, процент).
- Обход форматирования: при `formatting` равном `false` выводит исходное значение как строку.
- Переопределение локали через свойство `language`.
- Легковесный строчный рендеринг (рендерится в виде тега `span`).

**Типовые сценарии использования:**

- Отображение цен и финансовых значений в списках товаров.
- Отображение числовых значений с единицами измерения (вес, длина, проценты).
- Отображение локализованной числовой статистики на дашбордах и в отчетах.
- Отображение сырых чисел, где стандартное форматирование нежелательно.

<Source

<template>
  <div class="formatted-number-examples">
    <!-- Базовый локализованный вывод числа -->
    <FormattedNumber :value="1234567.89"/>

    <!-- Вывод валюты -->
    <FormattedNumber :value="99.9" currency="USD"/>

    <!-- Вывод единиц измерения -->
    <FormattedNumber :value="1500" unit="kilogram"/>

    <!-- Без форматирования -->
    <FormattedNumber :value="4500" :formatting="false"/>

    <!-- Переопределенная локаль -->
    <FormattedNumber :value="1234.56" language="ru-RU"/>
  </div>
</template>
  `}
    language="html"
/>

> FormattedNumber разработан как легкий, строчный служебный компонент для аккуратного представления локализованных чисел, валют и единиц измерения в любой части интерфейса.
