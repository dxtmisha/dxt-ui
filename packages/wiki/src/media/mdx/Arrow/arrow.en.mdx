## Description

Component for displaying an arrow or directional indicator.

The Arrow component is used to indicate direction, navigation, or to point to specific elements in the interface. It is often used in tooltips, popovers, and dropdown menus.

**Key Features:**

- Directional indication
- Customizable styling
- Integration with other components

```html
<script setup>
import { ref } from 'vue'
</script>

<template>
  <Arrow />
</template>
```
