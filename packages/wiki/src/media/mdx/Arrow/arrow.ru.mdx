## Описание

Компонент для отображения стрелки или индикатора направления.

Компонент Arrow используется для указания направления, навигации или для указания на конкретные элементы интерфейса. Часто применяется во всплывающих подсказках, поповерах и выпадающих меню.

**Основные возможности:**

- Указание направления
- Настраиваемая стилизация
- Интеграция с другими компонентами

```html
<script setup>
import { ref } from 'vue'
</script>

<template>
  <Arrow />
</template>
```
