## Description

Individual navigation link element in a breadcrumbs trail.

BreadcrumbItem represents a single step in a navigation trail. It supports standard text labels, optional prepended or appended icons, active state handling (disabled), static text mode (readonly), back navigation mode, and customizable HTML tags (such as anchor or span) which integrate cleanly with standard router links or native browser routes.

**Key features:**

- Text labels and icon customization
- Disabled (non-interactive) and Readonly (static) states
- Back navigation mode support
- Flexible tag customization
- Integration with standard RouterLink and native links
