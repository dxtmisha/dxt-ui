## Специальные символы (special)

Свойство `special` определяет специальные символы для групп маски и их расширенные настройки.

### Назначение

Специальные символы в маске обозначают места, где пользователь может вводить данные. Свойство `special` позволяет настроить поведение этих групп ввода с дополнительными возможностями.

### Типы значений

```ts
type MaskSpecialProp = string | string[] | MaskSpecialList

// Расширенная конфигурация группы
interface MaskSpecialItem {
  defaultValue?: string           // Значение по умолчанию
  rubber?: boolean               // Резиновая группа (переменная длина)
  transitionChar?: string | string[]  // Символы перехода
  minLength?: number             // Минимальная длина
  maxLength?: number             // Максимальная длина
  match?: RegExp | string        // Паттерн проверки
  pattern?: FieldPatternItemOrFunction  // Паттерн валидации
  view?: string                  // Отображаемый символ
}
```

### Способы использования

**1. Простая строка (один специальный символ)**
```javascript
special: '*'           // Символ * обозначает места ввода
// Маска: "***-***" → группы: *, *, *, *, *, *
```

**2. Массив символов (несколько типов групп)**
```javascript
special: ['*', '#', 'A']
// * - любые символы, # - цифры, A - буквы
// Маска: "A**-###" → группы разных типов
```

**3. Объект с расширенными настройками**
```javascript
special: {
  '*': {
    pattern: '\\d',        // Только цифры
    view: '_',             // Показывать _ как placeholder
    rubber: false          // Фиксированная длина
  },
  '#': {
    pattern: '[A-Z]',      // Только заглавные буквы
    defaultValue: 'X',     // Значение по умолчанию
    rubber: true,          // Переменная длина
    minLength: 1,
    maxLength: 5
  }
}
```

### Резиновые группы (rubber)

Резиновые группы позволяют вводить переменное количество символов:

```javascript
special: {
  '*': {
    rubber: true,
    minLength: 2,
    maxLength: 8,
    pattern: '\\d'
  }
}
// Маска: "***-***" может принять от 2 до 8 цифр в каждой группе
```

### Значения по умолчанию

```javascript
special: {
  'Y': { defaultValue: '2024' },    // Год по умолчанию
  'M': { defaultValue: '01' },      // Месяц по умолчанию
  'D': { defaultValue: '01' }       // День по умолчанию
}
// Маска: "DD/MM/YYYY" → автозаполнение: "01/01/2024"
```

### Символы перехода (transitionChar)

Определяют символы, которые автоматически переводят курсор к следующей группе:

```javascript
special: {
  '*': {
    transitionChar: [' ', '-'],     // Пробел или дефис переводят дальше
    pattern: '\\d'
  }
}
// При вводе "123 " курсор автоматически перейдёт к следующей группе
```

### Отображение (view)

Настройка placeholder-символов для незаполненных позиций:

```javascript
special: {
  '*': { view: '_' },       // Показывать подчёркивания
  '#': { view: '0' }        // Показывать нули
}
// Маска: "***-###" отображается как "___-000"
```

### Интеграция с типами маски

Для встроенных типов (`currency`, `number`, дата/время) система автоматически генерирует соответствующие специальные символы:

- **currency/number**: специальные символы для разрядов и дробной части
- **date/time**: символы для дня, месяца, года, часов, минут, секунд
- **text**: использует настройки из свойства `special`

### Взаимодействие с другими свойствами

- `pattern` — дополняет специальные символы групповыми паттернами
- `check` — применяется после заполнения всех специальных групп
- `mask` — использует специальные символы для определения мест ввода

Специальные символы являются основой системы масок, обеспечивая гибкое определение групп ввода с индивидуальными правилами валидации и поведения.
