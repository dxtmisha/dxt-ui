## Date and time types

Date types in Mask provide automatic formatting and validation for date and time input with localization support.

All date types automatically create masks according to regional settings, ignoring the `mask` property. The mask is determined by locale via `language` (DD.MM.YYYY for ru). Validation checks for correct numbers, days in month, and value ranges.

### Types

- `date` — full date in locale format (MM/DD/YYYY for US)
- `datetime` — date and time (HH:MM) separated by space
- `full` — full date and time with seconds (HH:MM:SS)
- `time` — time in HH:MM:SS format
- `hour-minute` — hours and minutes (HH:MM)
- `hour`, `minute`, `second` — separate time components
- `year`, `month`, `day` — separate date components
- `year-month`, `day-month` — date component combinations

### Localization

Formats adapt via the `language` property:
- `ru`, `de` — DD.MM.YYYY (31.12.2024)
- `en-US` — MM/DD/YYYY (12/31/2024)

### Examples

```html
<!-- US date format -->
<Mask type="date" language="en-US" />

<!-- Full date and time -->
<Mask type="datetime" language="en-US" />

<!-- Russian date format -->
<Mask type="date" language="ru" />

<!-- Time only -->
<Mask type="time" />
```
