- `getValue(): string` — Возвращает итоговое отформатированное значение маски.
- `getValueBasic(): string` — Возвращает базовое значение маски без форматирования (только введённые символы).
- `setValue(value: string): boolean` — Устанавливает новое значение и переинициализирует маску. Возвращает `true`, если значение было изменено.
- `clear(): boolean` — Очищает текущее значение.
