- `value: Ref<string>` — Итоговое отформатированное значение маски.
- `valueBasic: Ref<string>` — Базовое значение маски без форматирования (только введённые символы).
- `set(value: string): boolean` — Устанавливает новое значение и переинициализирует маску. Возвращает `true`, если значение было изменено.
