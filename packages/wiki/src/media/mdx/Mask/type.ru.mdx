## Типы масок

Свойство `type` определяет режим работы маски и автоматически генерирует соответствующую структуру для различных типов данных.

### Категории типов

#### Текстовый тип
- **text** — Базовая текстовая маска (по умолчанию)

Используется для создания произвольных масок с помощью свойства `mask`. Все символы обрабатываются как есть, без дополнительного форматирования.

#### Числовые типы
- **number** — Простая числовая маска
- **number-format** — Числа с разделителями разрядов
- **currency** — Валютное форматирование

Автоматически создают резиновые маски с валидацией, локализацией и специальным форматированием.

#### Типы дат и времени
- **date** — Полная дата
- **datetime** — Дата и время
- **time** — Время
- **year**, **month**, **day** — Компоненты даты
- **hour-minute** — Компоненты времени

Генерируют маски согласно локальным форматам с валидацией корректности дат.

### Автоматическая генерация

При использовании специализированных типов свойство `mask` игнорируется — маска создаётся автоматически:

```html
<!-- Текстовая маска — требует явного указания mask -->
<Mask :mask="'+7 *** *** ** **'" type="text" />

<!-- Специализированные типы — mask генерируется автоматически -->
<Mask type="date" language="ru" />
<Mask type="currency" currency="RUB" />
<Mask type="number-format" :fraction="2" />
```

### Выбор типа

- **text** — для телефонов, кодов, произвольных форматов
- **Числовые типы** — для количества, сумм, процентов
- **Типы дат** — для временных меток, периодов, расписаний
