## Типы масок

Свойство `type` определяет режим работы маски и автоматически генерирует соответствующую структуру для различных типов данных.

**Возможные значения:**

- `'text'` — базовая текстовая маска (по умолчанию)
- `'number'`, `'number-format'`, `'currency'` — числовые типы с валидацией и локализацией
- `'date'`, `'datetime'`, `'time'`, `'year'`, `'month'`, `'day'`, `'hour-minute'` — типы дат и времени

При использовании `text` требуется явное указание структуры через свойство `mask`. Специализированные числовые типы автоматически создают резиновые маски с разделителями разрядов, валидацией и форматированием согласно региональным настройкам (`language`). Типы дат генерируют маски по локальным форматам с проверкой корректности введённых значений. При активации специализированного типа свойство `mask` игнорируется, а структура маски формируется автоматически на основе `language`, `currency` и `fraction`.

```html
<!-- Текстовая маска -->
<Mask mask="+7 (***) ***-**-**" type="text" />

<!-- Валютная маска -->
<Mask type="currency" currency="RUB" language="ru" />

<!-- Маска даты -->
<Mask type="date" language="ru" />
```
