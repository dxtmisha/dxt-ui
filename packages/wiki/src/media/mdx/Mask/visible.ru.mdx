## Управление видимостью маски

Свойства `visible` и `visiblePartly` предназначены для управления отображением символов-заполнителей маски и видимостью компонента в целом.

**Свойства:**

- `visible` — управляет полной видимостью компонента маски
- `visiblePartly` — управляет частичной видимостью символов-заполнителей

Свойства работают совместно: при `visible: false` компонент маски полностью скрыт, включая все символы-заполнители. При активации `visiblePartly: true` отображаются только заполненные позиции маски, скрывая пустые символы-заполнители. Если `visible` установлен в `false`, то свойство `visiblePartly` игнорируется. Изменение видимости не влияет на логику валидации и обработку данных, позволяя создавать адаптивные интерфейсы с условным или сокращённым отображением маски.

```html
<script setup>
import { ref } from 'vue'

const showMask = ref(true)
const compactMode = ref(false)
</script>

<template>
  <!-- Полное скрытие маски -->
  <Mask
    mask="+7 (***) ***-**-**"
    :visible="false"
  />

  <!-- Показ только заполненных позиций -->
  <Mask
    mask="+7 (***) ***-**-**"
    :visiblePartly="true"
  />

  <!-- Условное отображение с частичной видимостью -->
  <Mask
    mask="+7 (***) ***-**-**"
    :visible="showMask"
    :visiblePartly="compactMode"
  />
</template>
```
