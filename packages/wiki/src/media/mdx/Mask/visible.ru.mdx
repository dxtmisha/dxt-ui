## Управление видимостью маски

Свойства `visible` и `visiblePartly` управляют отображением символов-заполнителей маски и видимостью компонента в целом.

### Свойства

- **visible** — Основное свойство для управления полной видимостью компонента маски
- **visiblePartly** — Управляет частичной видимостью символов-заполнителей маски

### Описание работы

**visible** контролирует полную видимость компонента маски. При значении `false` скрывает весь компонент маски включая все символы-заполнители.

**visiblePartly** управляет частичной видимостью символов-заполнителей. При значении `true` показывает только заполненные позиции, скрывая пустые символы-заполнители.

### Примеры использования

```html
// Полное скрытие маски
<Mask
  :mask="'+1 (***) ***-****'"
  :visible="false"
/>

// Показ только заполненных позиций
<Mask
  :mask="'+1 (***) ***-****'"
  :visiblePartly="true"
/>

// Условное отображение с частичной видимостью
<Mask
  :mask="'+1 (***) ***-****'"
  :visible="showMask"
  :visiblePartly="compactMode"
/>
```

### Особенности поведения

- При **visible: false** компонент маски полностью скрыт
- При **visiblePartly: true** отображаются только заполненные позиции маски
- Если **visible: false**, то свойство **visiblePartly** игнорируется
- Изменение видимости не влияет на логику валидации и обработку данных

### Применение

Используйте эти свойства для создания адаптивных интерфейсов, где маска может показываться условно или в сокращённом виде в зависимости от контекста использования.
