## Numeric mask types

Numeric types in Mask provide specialized formatting and validation for working with numbers, currencies, and financial data.

Numeric types automatically create rubber masks with variable length, ignoring the `mask` property. Negative values and `min`/`max` constraints are supported. Thousand separators and decimal places (via the `fraction` parameter) adapt based on the selected locale.

### Types

- `number` — simple numeric mask without formatting
- `number-format` — numbers with automatic thousand separators
- `currency` — currency with symbol (₽, $, €) and formatting

### Property fraction

Defines the number of decimal places:
- `number` — fixed number of digits (e.g., `2`)
- `true` — auto-detection from input
- `false`, `0` — integers only

### Localization

Formats adapt via the `language` property:
- `ru` — `1 234,56` (space and comma)
- `en-US` — `1,234.56` (comma and dot)
- `de` — `1.234,56` (dot and comma)

### Examples

```html
<!-- Formatted number -->
<Mask type="number-format" language="en-US" :fraction="2" />

<!-- Currency -->
<Mask type="currency" currency="USD" language="en-US" />

<!-- Currency without symbol -->
<Mask type="currency" currency="USD" :currencyHide="true" />
```
