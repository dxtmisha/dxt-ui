## Mask types

The `type` property defines the mask operation mode and automatically generates the appropriate structure for different data types.

**Possible values:**

- `'text'` — basic text mask (default)
- `'number'`, `'number-format'`, `'currency'` — numeric types with validation and localization
- `'date'`, `'datetime'`, `'time'`, `'year'`, `'month'`, `'day'`, `'hour-minute'` — date and time types

When using `text`, explicit structure specification via the `mask` property is required. Specialized numeric types automatically create rubber masks with thousand separators, validation, and formatting according to regional settings (`language`). Date types generate masks according to local formats with validation of entered values correctness. When a specialized type is activated, the `mask` property is ignored, and the mask structure is formed automatically based on `language`, `currency`, and `fraction`.

```html
<!-- Text mask -->
<Mask mask="+1 (***) ***-****" type="text" />

<!-- Currency mask -->
<Mask type="currency" currency="USD" language="en-US" />

<!-- Date mask -->
<Mask type="date" language="en-US" />
```
