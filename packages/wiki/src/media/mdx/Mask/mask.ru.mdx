# Маска ввода

Свойство `mask` определяет шаблон для форматирования пользовательского ввода в реальном времени. Позволяет создавать структурированные поля для телефонов, дат, карт и других форматированных данных.

### Назначение

- **Структурированный ввод** — Направляет пользователя при вводе данных в нужном формате
- **Автоматическое форматирование** — Добавляет разделители и символы во время набора
- **Предотвращение ошибок** — Ограничивает ввод только допустимыми символами
- **Улучшение UX** — Делает процесс ввода более понятным и предсказуемым

### Основные возможности

- **Гибкие шаблоны** — Поддержка одиночных и множественных масок
- **Специальные символы** — Настраиваемые позиции для ввода разных типов данных
- **Автоматический выбор** — При множественных масках выбирается наиболее подходящая
- **Интеграция с валидацией** — Работает совместно с системой проверки данных

### Применение

Используйте маски для:
- Номеров телефонов (`+7 *** *** ** **`)
- Банковских карт (`**** **** **** ****`)
- Дат (`**.**.****`)
- Почтовых индексов (`******`)
- Любых структурированных данных

### Синтаксис

```javascript
// Одиночная маска
mask: "+7 *** *** ** **"

// Множественные маски
mask: ["+7 *** *** ** **", "8 *** *** ** **"]
```

**Специальные символы** (по умолчанию `*`) обозначают позиции для ввода, **литеральные символы** вставляются автоматически.

### Рекомендации

- Используйте понятные разделители для улучшения читаемости
- При поддержке нескольких форматов применяйте массив масок
- Настраивайте специальные символы через свойство `special`
- Комбинируйте с валидацией для полного контроля ввода
