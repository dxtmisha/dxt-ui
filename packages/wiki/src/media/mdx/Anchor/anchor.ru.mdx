Компонент для создания гиперссылок или якорных точек, обеспечивающий навигацию внутри страницы или переход на внешние ресурсы.

Anchor позволяет создавать интерактивные ссылки с поддержкой иконок, текста и различных состояний. Компонент может использоваться как классическая ссылка, кнопка для копирования текста в буфер обмена или как якорная точка для скроллинга к определенному разделу страницы. Поддерживает интеграцию с Tooltip для отображения подсказок.

**Основные возможности:**

- Создание внешних и внутренних ссылок
- Режим якорной точки (без href)
- Копирование содержимого в буфер обмена
- Плавная прокрутка к целевому элементу
- Поддержка иконок и текста
- Интеграция с компонентом Tooltip

**Типичные сценарии использования:**

- Навигация по разделам страницы
- Ссылки на внешние ресурсы
- Кнопки "Копировать ссылку" или "Копировать текст"
- Якоря для заголовков в документации

```html
<script setup>
import { Anchor } from 'dxt-ui'
</script>

<template>
  <div class="flex flex-col gap-4">
    <Anchor href="https://example.com" text="Внешняя ссылка" />
    <Anchor name="section-1" text="Якорь раздела" />
  </div>
</template>
```
