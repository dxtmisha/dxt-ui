## Component Integration

The `DraggableWrapper` is built upon a *renderless* architecture pattern. It encapsulates complex drag-and-drop and sorting mechanics (coordinate calculations, movement calculations, displacement styles) while completely delegating the visual rendering to your template via default slot properties.

To ensure the sorting engine operates correctly, you must bind the provided properties to your target DOM nodes.

### Element Configuration

The elements inside the list or drop zones must implement the following bindings:

- `className` — the core identifier. Bound to the `class` attribute. Required for connecting the draggable elements and drop zones to the wrapper logic and applying shift styles.
- `classClick` — the interactive zone identifier (drag handle). Bound to the `class` attribute. Indicates the specific area of the element (e.g., handle icon) that initiates the drag action. If the entire item is draggable, it must be bound to the item itself.
- `classPosition` — the sorting position identifier. Bound to the `class` attribute. Used by the component to calculate the insertion points of elements. Required for all sortable elements in the list.
- `classDrop` — the drop zone identifier. Bound to the `class` attribute. Indicates additional drop targets (such as a trash bin or external panel) that accept elements.
- `data-value` — the element's unique key (string). Passed as a standard HTML attribute. Required for identifying elements in `position` and `drop` events.

### State and Visual Management

To achieve smooth drag-and-drop animation and sorting performance, `DraggableWrapper` **autonomously** manipulates the DOM tree by directly applying transform styles and shifting elements. This design ensures instant feedback during rapid drag movements.

    code={`
<DraggableWrapper
  @position="onPositionChange"
  @drop="onItemDropped"
>
  <template #default="{ className, classClick, classPosition, classDrop }">
    <div class="list">
      <div :class="['item', className, classPosition]" data-value="item-1">
        <span :class="classClick">☰</span>
        Item 1
      </div>
    </div>
  </template>
</DraggableWrapper>
  `}
    language="html"
/>
