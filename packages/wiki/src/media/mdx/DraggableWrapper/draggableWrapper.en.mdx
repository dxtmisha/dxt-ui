Component-wrapper for creating drag-and-drop sorting lists with support for single and multiple selection, customized styles, delay, and smooth animations.

DraggableWrapper handles the drag session, captures mouse/touch movements, coordinates positions of elements, manages the placeholder spacer element, and fires events on drag completion.

**Key features:**

- Single and multiselect drag and drop sorting
- Custom styles for the placeholder square element
- Delay before drag starts to prevent accidental drags
- Smooth CSS transition animations during element returns
- Auto-scroll and layout shift indicators

**Common use cases:**

- Reordering lists of items (e.g. tasks, files, layers)
- Drag-and-drop between columns or list groups
- Sorting dashboard cards or tabs
