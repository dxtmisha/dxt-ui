## Описание

Компонент управления навигацией, предназначенный для разделения больших наборов данных на страницы, управления лимитом строк на странице и инкрементальной загрузки контента.

Pagination предоставляет расчет количества страниц, быстрый переход к крайним страницам (первая/последняя), настраиваемый выбор лимита строк и загрузчики данных («Показать еще»).

**Основные возможности:**
- Автоматический расчет количества страниц на основе общего числа элементов (count) и строк на странице (rows)
- Кнопки навигации «Вперед», «Назад», «Первая» и «Последняя»
- Разделители в виде многоточия между средним диапазоном и крайними страницами
- Выпадающий список для настройки количества строк на странице (v-model:rows)
- Синхронизация состояния выбора страницы (v-model:value)
- Поддержка кнопки подгрузки дополнительных элементов («Показать еще»)
- Локализованная строка информации о диапазоне (например, «1-10 из 100»)

**Типичные сценарии использования:**
- Разделение списков элементов на страницы
- Таблицы и гриды данных, требующие серверного или клиентского контроля лимитов строк
- Динамические каталоги с возможностью подгрузки данных
- Кастомная навигация по каталогам товаров

    code={`
<script setup>
import { ref } from 'vue'

const page = ref(1)
const rows = ref(10)
</script>

<template>
  <!-- Базовая пагинация -->
  <Pagination :count="100" :rows="10" />

  <!-- Двусторонняя привязка данных -->
  <Pagination
    v-model:value="page"
    v-model:rows="rows"
    :count="100"
    :menu-rows="[5, 10, 20, 50]"
    show-info
  />
</template>
  `}

> Pagination координирует реактивные диапазоны страниц, выпадающие списки количества строк и макеты кнопок для структурированного представления данных.
