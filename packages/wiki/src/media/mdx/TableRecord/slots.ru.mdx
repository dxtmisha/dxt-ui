- `[column: string](props: TableRecordItemSlotProps): any` — Динамический слот для рендеринга пользовательской ячейки для указанного индекса колонки.

> **TableRecordItemSlotProps:**
> - `item: any` — Объект данных строки.
> - `value: any` — Текущее значение ячейки.
> - `classes: { label: string, description: string }` — Вспомогательные имена классов элементов ячейки.
> - `key: string` — Уникальный ключ рендеринга.
> - `index: string` — Индекс колонки.
> - `slotsName: string` — Имя слота.
