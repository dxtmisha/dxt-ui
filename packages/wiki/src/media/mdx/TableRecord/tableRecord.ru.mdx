Компонент строки таблицы, представляющий тег `<tr>`. Он координирует рендеринг набора ячеек столбцов (`TableHeaderItem`, если `isHeader` равен true, или `TableItem` в противном случае) на основе записи данных (свойство `item`) и списка индексов столбцов (свойство `columns`).

**Ключевые особенности:**

- Стандартный контейнер строки таблицы (тег `tr`)
- Динамическая генерация ячеек на основе ключей `columns` и значений `item`
- Автоматический выбор между ячейками заголовка (`TableHeaderItem`) и ячейками данных (`TableItem`) с помощью флага `isHeader`
- Состояния выбора и отключения, применяемые ко всей строке (`selected`, `disabled`)
- Динамические пользовательские слоты, названные в честь индексов столбцов, для кастомного рендеринга содержимого конкретных ячеек

<Source
    code={`
<script setup>
</script>

<template>
  <table>
    <!-- Строка заголовка -->
    <TableRecord
      :isHeader="true"
      :columns="['name', 'role']"
      :item="{ name: 'Имя', role: 'Роль' }"
    />

    <!-- Строка данных -->
    <TableRecord
      :columns="['name', 'role']"
      :item="{ name: 'Алиса', role: 'Разработчик' }"
    />
  </table>
</template>
  `}
    language="html"
/>

> TableRecord упрощает создание таблиц путем автоматизации рендеринга ячеек на основе структур данных «ключ-значение» с полной поддержкой переопределения через пользовательские слоты для каждой колонки.
