- `[column: string](props: TableRecordItemSlotProps): any` — Dynamic slot for rendering a custom cell for the specified column index.

> **TableRecordItemSlotProps:**
> - `item: any` — Row data object.
> - `value: any` — Current cell value.
> - `classes: { label: string, description: string }` — Helper class names of cell elements.
> - `key: string` — Unique rendering key.
> - `index: string` — Column index.
> - `slotsName: string` — Slot name.
