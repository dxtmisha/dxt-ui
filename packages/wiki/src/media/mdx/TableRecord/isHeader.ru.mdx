## Заголовок таблицы

Свойство `isHeader` переключает строку таблицы в режим отображения шапки (заголовка):

- Вместо стандартных ячеек данных (`TableItem` / `<td>`) рендерятся ячейки заголовков (`TableHeaderItem` / `<th>`).
- Применяются соответствующие стили шапки таблицы (шрифт, выравнивание, фон).
- Автоматически активируется поддержка всплывающих подсказок (`tooltip`) при их наличии.

<Source
    code={`
<TableRecord
  :isHeader="true"
  :columns="['name', 'role']"
  :item="{ name: 'Имя', role: 'Роль' }"
/>
  `}
    language="html"
/>
