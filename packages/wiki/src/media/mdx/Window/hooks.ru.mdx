## Хуки жизненного цикла

### `preparation`

Вызывается перед началом процесса открытия/закрытия. Используется для подготовки элементов и данных.

**Тип:** `() => void | Promise<void>`

### `beforeOpening`

Вызывается перед открытием окна. Возврат `false` отменяет открытие.

**Тип:** `() => boolean | Promise<boolean>`

### `beforeClosing`

Вызывается перед закрытием окна. Возврат `false` отменяет закрытие.

**Тип:** `() => boolean | Promise<boolean>`

### `opening`

Вызывается после завершения открытия окна (после анимации).

**Тип:** `() => boolean | Promise<boolean>`

### `closing`

Вызывается после завершения закрытия окна (после анимации).

**Тип:** `() => boolean | Promise<boolean>`
