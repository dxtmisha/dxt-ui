## Lifecycle Hooks

### `preparation`

Called before the opening/closing process begins. Used for preparing elements and data.

**Type:** `() => void | Promise<void>`

```html
<script setup>
const handlePreparation = () => {
  console.log('Preparing...')
}
</script>

<template>
  <Window :preparation="handlePreparation" />
</template>
```

### `beforeOpening`

Called before opening the window. Returning `false` cancels opening.

**Type:** `() => boolean | Promise<boolean>`

```html
<script setup>
const handleBeforeOpening = async () => {
  const canOpen = await checkPermissions()
  return canOpen // false will cancel opening
}
</script>

<template>
  <Window :before-opening="handleBeforeOpening" />
</template>
```

### `beforeClosing`

Called before closing the window. Returning `false` cancels closing.

**Type:** `() => boolean | Promise<boolean>`

```html
<script setup>
const handleBeforeClosing = async () => {
  const canClose = await confirmClose()
  return canClose // false will cancel closing
}
</script>

<template>
  <Window :before-closing="handleBeforeClosing" />
</template>
```

### `opening`

Called after the window finishes opening (after animation).

**Type:** `() => boolean | Promise<boolean>`

```html
<script setup>
const handleOpening = () => {
  console.log('Window opened')
}
</script>

<template>
  <Window :opening="handleOpening" />
</template>
```

### `closing`

Called after the window finishes closing (after animation).

**Type:** `() => boolean | Promise<boolean>`

```html
<script setup>
const handleClosing = () => {
  console.log('Window closed')
}
</script>

<template>
  <Window :closing="handleClosing" />
</template>
```
