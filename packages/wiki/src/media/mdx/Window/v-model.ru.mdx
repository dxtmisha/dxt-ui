## Двусторонняя привязка (v-model)

Компонент Window поддерживает двустороннюю привязку состояния открытия через директиву `v-model`.

### Свойства и события

- `open: boolean` — состояние видимости окна
- `update:open(boolean)` — событие обновления при изменении состояния

```html
<Window v-model:open="isOpen"/>
```

### Заметки

- Можно программно управлять окном через expose‑методы (`open`/`toggle`/`setOpen`) и соответствующие события
- Если нужно статически смонтированное окно (игнорирующее v-model), см. раздел `staticMode`
