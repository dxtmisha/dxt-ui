## Направление позиционирования

Свойство `axis` управляет основной направляющей для размещения окна относительно элемента‑якоря (`overElement`) или области вызова.
Значение по умолчанию: `y`.

> Работает только в режиме меню (`adaptive="menu"`).

### Значения

- `x` — Горизонтально: окно располагается слева или справа от якоря/курсора. Отступ `indent` применяется по горизонтали.
- `y` — Вертикально: окно располагается сверху или снизу от якоря/курсора. Отступ `indent` применяется по вертикали.
- `on` — Поверх элемента: окно перекрывает выбранный элемент/область и ориентируется по его центру. При необходимости прокрутка содержимого подстраивается, чтобы выбранный элемент оставался видимым.

### Поведение

- Автовыбор стороны (слева/справа для `x`, сверху/снизу для `y`) в зависимости от доступного пространства. При нехватке места окно центрируется в доступной области.
- По перпендикулярному направлению окно вписывается во вьюпорт и не выходит за его пределы.
- Режим контекстного меню (`contextmenu = true`) использует координаты курсора вместо прямоугольника якоря.
- Учитываются ограничения вьюпорта и внутренние отступы контейнера.

### Рекомендации

- `x` — боковые меню, панели рядом с элементом, тултипы по бокам.
- `y` — выпадающие списки, меню и подсказки сверху/снизу.
- `on` — поповеры, перекрывающие и фокусирующие конкретный элемент.
