- `getId(): string` — Возвращает уникальный идентификатор окна.
- `getOpen(): boolean` — Возвращает реактивное состояние видимости окна.
- `getControl(): WindowControlItem` — Возвращает данные элемента управления для управления окном.
    
> **WindowControlItem:**
> - `open: Ref<boolean>` — реактивное состояние открытия.
> - `class: string` — CSS-класс окна.
> - `onclick: WindowEventClick` — обработчик события клика.
> - `onKeydown: WindowEventClick` — обработчик события нажатия клавиши.
> - `oncontextmenu: WindowEventClick` — обработчик события контекстного меню.
> - `classesWindow: WindowClassesList` — список CSS-классов для стилизации окна.
> - `binds: Record<string, any>` — свойства привязки для элемента управления.

- `setOpen(open: boolean): Promise<void>` — Устанавливает состояние открытия окна.
- `toOpen(): Promise<void>` — Переход в состояние открытия окна.
- `toClose(): Promise<void>` — Переход в состояние закрытия окна.
- `toggle(): Promise<void>` — Переключает видимость окна (открывает, если закрыто, и закрывает, если открыто).
