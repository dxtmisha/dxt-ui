- `control(props: WindowControlItem): any` — Slot for placing window control elements (close buttons, minimize, etc.).
- `title(props: WindowControlItem): any` — Slot for placing window title in the header.
- `footer(props: WindowControlItem): any` — Slot for placing content at the bottom of the window (action buttons, info, etc.).
- `default(props: WindowControlItem): any` — Main slot for placing the primary window content.

> **WindowControlItem:**
> - `open: Ref<boolean>` — reactive open state.
> - `class: string` — window CSS class.
> - `onclick: WindowEventClick` — click event handler.
> - `onKeydown: WindowEventClick` — keydown event handler.
> - `oncontextmenu: WindowEventClick` — context menu event handler.
> - `classesWindow: WindowClassesList` — list of CSS classes for window styling.
> - `binds: Record<string, any>` — binding properties for the control element.

> **WindowClassesList:**
> - `block: string` — main block class.
> - `blockChildren: string` — block children class.
> - `blockOther: string` — block other windows class.
> - `close: string` — close button class.
> - `static: string` — static positioning class.
> - `control: string` — control element class.
> - `controlId: string` — control ID class.
> - `controlStatic: string` — static control class.
> - `controlOpenOnly: string` — open-only control class.
> - `controlActive: string` — active control class.
> - `persistent: string` — persistent window class.

> **WindowEventClick:**
> - `(event: MouseEvent & TouchEvent | KeyboardEvent) => Promise<void>` — window event click handler.
