## Description

A clean and semantic component for displaying text descriptions with custom dynamic tag wrapping.

TextDescription is built for modular and lightweight layout structures, offering a standardized approach to presenting paragraph text, sub-labels, or detailed information block captions.

**Key Features:**

- Dynamic wrapper tag support (e.g., span, div, p, etc.) via the `tag` prop
- Standardized render logic powered by `DescriptionInclude`
- Highly optimized, clutter-free HTML output
- Cohesive integration with design-system typography tokens

**Typical Usage Scenarios:**

- Displaying auxiliary description strings in block layouts
- Presenting captions and footnotes under list items
- Rendering paragraph text sections
- Form field helper and informative instructions
