## Описание

Вспомогательный компонент для расширения контента за пределы горизонтальных отступов родительского контейнера путем применения отрицательных внешних отступов.

Bleed предоставляет элегантное решение для создания элементов «на вылет» (например, изображений или баннеров во всю ширину) внутри ограниченных по ширине контейнеров. Компонент автоматически вычисляет необходимые отрицательные отступы на основе переданного значения, позволяя контенту визуально прорывать границы родительского padding без необходимости изменения структуры макета.

**Основные возможности:**

- Автоматическое применение отрицательных горизонтальных отступов
- Поддержка настройки любого HTML-тега
- Сохранение вертикального потока документа
- Интеграция с системой токенов дизайна
- Адаптивность через динамическое управление отступами

**Типичные сценарии использования:**

- Полноэкранные изображения в статьях
- Баннеры и breakout-секции в текстовых блоках
- Горизонтальные разделители во всю ширину
- Интеграция сторонних виджетов, требующих большего пространства
- Декоративные элементы, выходящие за сетку

> Bleed упрощает управление сложной версткой, позволяя контенту динамически расширяться за границы родительского контейнера, сохраняя при этом целостность макета и семантическую структуру.
