- `head(props: MotionTransformControlItem): any` — Slot for the header/controls.
- `body(props: MotionTransformControlItem): any` — Slot for the body/content.

> **MotionTransformControlItem:**
> - `isOpen: boolean` — Reactive open state.
> - `isShow: boolean` — Visible or preparing to show.
> - `classes: MotionTransformClassList` — Helper CSS classes.
> - `idControl: string` — Helper style variable/ID for control.
> - `idBody: string` — Helper style variable/ID for body.
> - `binds: AriaRoleControlPropsInclude` — Binding attributes for the element.
