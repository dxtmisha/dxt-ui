- `control(props: MotionTransformControlItem): any` — Слот для размещения триггерного элемента, управляющего анимацией.
- `default: any` — Слот для размещения контента, который будет анимирован.

> **MotionTransformControlItem:**
> - `onclick: (event) => void` — обработчик открытия/закрытия трансформируемого элемента.
