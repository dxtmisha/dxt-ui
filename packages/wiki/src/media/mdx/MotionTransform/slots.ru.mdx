- `head(props: MotionTransformControlItem): any` — Слот шапки для элементов управления.
- `body(props: MotionTransformControlItem): any` — Слот тела для содержимого.

> **MotionTransformControlItem:**
> - `isOpen: boolean` — Реактивное состояние открытия.
> - `isShow: boolean` — Видим либо готовится к показу.
> - `classes: MotionTransformClassList` — Вспомогательные CSS‑классы.
> - `idControl: string` — Вспомогательная стилевая переменная / ID управления.
> - `idBody: string` — Вспомогательная стилевая переменная / ID тела.
> - `binds: AriaRoleControlPropsInclude` — Атрибуты привязки для элемента.
