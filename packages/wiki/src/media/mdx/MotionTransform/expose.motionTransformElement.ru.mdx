- `motionTransformElement: Ref<ConstrBind<MotionTransformExpose> | undefined>` — Реактивная ссылка на expose API вложенного компонента MotionTransform, предоставляющая программный доступ к методам управления.

> **Методы через motionTransformElement.value:**
> - `open: Ref<boolean>` — реактивное состояние открытия
> - `isShow: ComputedRef<boolean>` — вычисляемое состояние видимости
> - `setOpen(open: boolean): void` — установить состояние открытия
> - `toOpen(): void` — открыть компонент
> - `toClose(): void` — закрыть компонент
> - `toggle(): void` — переключить состояние
