- `getMotionTransformElement(): ConstrBind<MotionTransformExpose> | undefined` — Возвращает expose API вложенного компонента MotionTransform, предоставляющая программный доступ к методам управления.

> **Методы через getMotionTransformElement():**
> - `getOpen(): boolean` — возвращает состояние открытия
> - `isShow(): boolean` — возвращает состояние видимости
> - `setOpen(open: boolean): void` — установить состояние открытия
> - `toOpen(): void` — открыть компонент
> - `toClose(): void` — закрыть компонент
> - `toggle(): void` — переключить состояние
