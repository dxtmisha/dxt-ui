### `head`

Слот для рендеринга области заголовка, которая служит триггером для открытия/закрытия компонента.

Этот слот позволяет:
- Размещать интерактивные элементы управления, такие как кнопки, заголовки или иконки
- Получать данные состояния компонента через scoped slot props
- Управлять визуальным представлением области триггера
- Получать доступ к состояниям isOpen, isShow и CSS‑классам для стилизации

Область head обычно обрабатывает события клика для переключения состояния видимости компонента.
