Компонент для создания выпадающих списков выбора на основе чипов с дополнительным поиском в заголовке и кастомными стилями триггера.

MenuChip — это компактный выпадающий селектор, который заменяет стандартную рамку текстового поля элемента select на лаконичный чип-триггер. Он сочетает архитектуры Chip и выпадающего Menu/List для отображения выбранных значений с гибко настраиваемыми разделителями и правилами скрытия текста, а также содержит встроенные скрытые поля ввода input для нативной интеграции с формами.

**Основные возможности:**

- Чип-триггер вместо стандартной рамки текстового поля ввода
- Отображение выбранных значений с настраиваемыми разделителями
- Поддержка скрытия значения при наличии иконки (hideValueIcon)
- Поддержка скрытия метки при выборе значения (hideLabelValue)
- Выпадающее меню опций с поиском и фильтрацией в заголовке (showSearch)
- Поддержка одиночного или множественного выбора значений
- Скрытое поле ввода input для стандартной передачи данных формы
- Адаптивный дизайн для мобильных устройств

**Типичные сценарии использования:**

- Удобные компактные фильтры категорий в горизонтально скроллируемых списках
- Селекторы в тулбарах (например, селекторы статусов, фильтры тегов, размер шрифта, выбор языка)
- Компактные выпадающие списки, где стандартная граница поля или кнопки нежелательна
- Мобильные селекторы с запуском действий
