- `index: number | string` — Index of the input cell.
- `getValue(): string` — Returns the current input cell value.
- `set(value: string): void` — Programmatically sets the input cell value.
- `setTabindex(value: number | string | undefined): void` — Programmatically updates the tabindex of the cell.
- `reset(): void` — Resets the input cell value to empty.
- `focusInput(): void` — Sets keyboard focus onto the cell's input element.
