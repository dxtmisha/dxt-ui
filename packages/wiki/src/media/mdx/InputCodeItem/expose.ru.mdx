- `index: number | string` — Индекс ячейки ввода.
- `getValue(): string` — Возвращает текущее значение ячейки ввода.
- `set(value: string): void` — Программно устанавливает значение ячейки ввода.
- `setTabindex(value: number | string | undefined): void` — Программно изменяет значение tabindex для ячейки.
- `reset(): void` — Сбрасывает текущее значение ячейки ввода до пустого состояния.
- `focusInput(): void` — Устанавливает фокус ввода на элемент input внутри ячейки.
