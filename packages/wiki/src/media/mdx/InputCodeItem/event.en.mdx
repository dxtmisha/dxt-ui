### `focus`

Emitted when the input cell receives focus.

**Parameters:**
- `index: number | string` — the index of the focused cell

### `input`

Emitted when a character is entered or modified in the cell.

**Parameters:**
- `index: number | string` — the index of the cell
- `value: string` — the newly entered character

### `paste`

Emitted when a paste event occurs on the input cell.

**Parameters:**
- `index: number | string` — the index of the cell
- `value: string` — the pasted text value

### `backspace`

Emitted when the backspace key is pressed in the cell.

**Parameters:**
- `index: number | string` — the index of the cell
