Component for creating interactive toggle switches for quickly changing binary settings or on/off states.

Switch serves as a visual alternative to a checkbox, mimicking a physical toggle switch. It supports smooth transition animations, focus management, built-in validation, form integration, and loading state indicators. The component is ideal for instant settings application without requiring a form submission.

**Key features:**

- Quick binary toggling states (on / off)
- Built-in validation with custom error messages
- Support for labels and detailed descriptions
- Customizable and smoothly animated track and slider elements
- Form integration via name and value attributes
- Disabled, loading, and skeleton states
- Ripple effect on interaction
- Adaptive layout and styling

**Common use cases:**

- Toggling options in a settings panel
- Activating application features (e.g., dark mode, push notifications)
- Quick filters in catalogs
- Agreement confirmation (alternative to checkbox)
