Компонент для создания интерактивных переключателей (тоглов) для быстрого изменения бинарных настроек или состояний «включено»/«выключено».

Switch представляет собой визуальную альтернативу чекбоксу, имитирующую физический тумблер. Он поддерживает плавные анимации переходов, управление фокусом, встроенную валидацию, интеграцию с формами и индикацию состояний загрузки. Компонент идеально подходит для мгновенного применения настроек без необходимости отправки формы.

**Основные возможности:**

- Быстрое переключение бинарных состояний (вкл / выкл)
- Встроенная валидация с пользовательскими сообщениями об ошибках
- Поддержка меток (label) и подробного описания (description)
- Настраиваемые и плавно анимированные элементы трека и ползунка
- Интеграция с формами через name и value
- Состояния disabled, loading и skeleton
- Эффект ripple при взаимодействии
- Адаптивная раскладка и стилизация

**Типичные сценарии использования:**

- Включение и выключение опций в панели настроек
- Активация функций приложения (например, темная тема, push-уведомления)
- Быстрые фильтры в каталогах
- Подтверждение согласия (альтернатива чекбоксу)
