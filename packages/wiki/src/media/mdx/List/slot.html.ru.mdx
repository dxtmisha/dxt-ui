### `slot: html`

Компонент List поддерживает именованные слоты для элементов с `type: 'html'`. Имя слота должно совпадать со значением поля `value` у элемента.

Правила использования

- Элемент списка должен иметь `type: 'html'` и строковое поле `value`, которое используется как ключ слота
- Если слот с таким именем существует, его содержимое заменяет рендеринг элемента
- Если слота нет, компонент рендерит HTML из поля `label` через `innerHTML`

Пример

```html
<List
  :list="[
    { type: 'html', value: 'promoBlock', label: '<strong>Запасной HTML</strong>' },
    { label: 'Обычный элемент', value: 'item1' }
  ]"
>
  <template #promoBlock>
    <div class="custom-promo">Пользовательский контент слота</div>
  </template>
</List>
```

Заметки

- Слот не получает пропсы; используйте статический контент или состояние приложения
- Предпочитайте слоты для Vue-контента; используйте `label` с HTML только для простого и доверенного разметки
