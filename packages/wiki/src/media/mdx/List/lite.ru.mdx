## Облегчённый режим списка

Свойства `lite` и `liteThreshold` предназначены для управления облегчённым (упрощённым) отображением списка.

### Свойства

- **lite** — Включает облегчённый режим отображения
- **liteThreshold** — Порог количества элементов, при превышении которого активируется упрощённый режим (когда `lite` включён)

### Взаимосвязь свойств

Свойства работают совместно и определяют, когда применять упрощённое отображение:

- Если `lite = false` (или не задан), список всегда рендерится в полном режиме, а `liteThreshold` игнорируется
- Если `lite = true`, компонент сравнивает общее число элементов с `liteThreshold`
  - При `count <= liteThreshold` используется полный режим (упрощение не требуется)
  - При `count > liteThreshold` включается облегчённый режим для снижения визуальной сложности и ускорения рендера

Такой подход предотвращает излишнюю нагрузку при больших объёмах данных и сохраняет привычное полное отображение для коротких списков.
