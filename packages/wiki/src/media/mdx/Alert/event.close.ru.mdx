### `close`

Событие срабатывает при нажатии на кнопку закрытия для скрытия баннера оповещения.

<Source
    code={`
<script setup>
const handleClose = () => {
  console.log('Оповещение скрыто')
}
</script>

<template>
  <Alert
    label="Alert Title"
    close-button
    @close="handleClose"
  />
</template>
  `}
    language="html"
/>
