## Description

An alert component designed to show contextual feedback messages, banners, or notifications.

The Alert component is highly configurable, supporting a leading icon, a text label, a description, action buttons, custom lists of action buttons, and inline text links (AlertLink). It can be made dismissible by displaying a close button, which triggers a smooth exit transition.

**Key Features:**

- Structured content layout (leading icon, context area with title and description, trailing action/close button)
- Custom action buttons (actions list) and inline links (alertLink)
- Dismissible state with animation end event triggers
- Flexible layout alignment with vertical centering options

**Typical Usage Scenarios:**

- Inline status banners (e.g., success, error, warning, informational notices)
- Page-level notifications or promotional alerts
- Inline call-to-action cards with quick actions
