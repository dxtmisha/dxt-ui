## Описание

Компонент макета для создания сеточных структур с настраиваемыми колонками.

**Основные возможности:**

- Гибкая сеточная система (по умолчанию 12 колонок)
- Настройка отступов между элементами
- Создание адаптивных интерфейсов

> Grid является основой для создания упорядоченных и структурированных макетов для различных размеров экранов.
