## Description

A specialized link component designed specifically to be rendered within status alerts, messages, and banners.

AlertLink integrates seamlessly with the typography and color styling of notification boxes, providing an optimized visual hierarchy. It includes optional support for visual separators (dividers) when rendering multiple links sequentially.

**Key Features:**

- Visual styles optimized for alert contexts
- Configurable link separators (dividers) to separate sibling links
- Fully customizable color-palette options
- Seamless integration with click handler events and routing

**Typical Usage Scenarios:**

- Inline links inside warning, error, info, or success banners
- Help or actions links placed at the bottom of notification messages
- Grouped navigation options inside contextual alert containers

> AlertLink simplifies link placement in alerts, ensuring proper typography contrast and layout consistency without extra styling overhead.
