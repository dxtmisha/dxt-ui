## Positioning

The `vertical`, `horizontal` and `origin` property group allows for determining the exact position of notifications on the screen and their entrance direction.

**Features:**
- `vertical` — defines the position along the vertical axis (`top`, `bottom`).
- `horizontal` — manages the horizontal position of notifications and their width (`right`, `left`, `block`).
- `origin` — sets the direction from which the notification will slide in (`topToBottom`, `bottomToTop`, `rightToLeft`, `leftToRight`).

> These properties are intended solely for configuring behavior and adaptation for different screen sizes. It is recommended to define these values in the `properties.json` configuration file.
