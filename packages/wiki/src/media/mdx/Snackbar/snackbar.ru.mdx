## Описание

Компонент для отображения временных уведомлений (снекбаров) в различных частях экрана. Снекбары используются для предоставления кратких сообщений о процессах приложения в нижней или верхней части экрана.

Snackbar управляет очередью сообщений, автоматически скрывая их по истечении заданного времени или при взаимодействии пользователя.

**Основные возможности:**

- Гибкое позиционирование (сверху/снизу, слева/справа/по центру)
- Настраиваемая задержка автоматического скрытия (по умолчанию 8 секунд)
- Поддержка различных анимаций появления в зависимости от позиции
- Управление списком активных уведомлений

**Типичные сценарии использования:**

- Подтверждение действий пользователя (например, "Файл успешно сохранен")
- Информирование об ошибках или предупреждениях
- Системные уведомления в реальном времени

```html
<script setup>
import { ref } from 'vue'

const snackbar = ref(null)

const addMessage = () => {
  snackbar.value?.data.add({
    label: 'Заголовок уведомления',
    description: 'Описание события, которое произошло в приложении.',
    button: 'Ок'
  })
}
</script>

<template>
  <Button label="Показать уведомление" @click="addMessage" />

  <Snackbar
    ref="snackbar"
    vertical="bottom"
    horizontal="right"
  />
</template>
```

> Рекомендуется использовать Snackbar для второстепенной информации, которая не требует немедленного обязательного действия со стороны пользователя.
