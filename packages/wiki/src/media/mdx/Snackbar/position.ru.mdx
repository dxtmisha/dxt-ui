## Позиционирование

Группа свойств `vertical`, `horizontal` и `origin` позволяет определить точное местоположение уведомлений на экране и направление их появления.

**Особенности:**
- `vertical` — определяет положение по вертикали (`top`, `bottom`).
- `horizontal` — определяет положение по горизонтали (`right`, `left`, `block`).
- `origin` — задает направление анимации появления (`topToBottom`, `bottomToTop`, `rightToLeft`, `leftToRight`).

> Данные свойства предназначены исключительно для настройки поведения и адаптации под различные экраны. Рекомендуется задавать эти значения в конфигурационном файле `properties.json`.
