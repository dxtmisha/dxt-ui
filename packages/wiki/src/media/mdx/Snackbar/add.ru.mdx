## Добавление уведомлений

Основной метод взаимодействия с компонентом `Snackbar` — добавление новых сообщений в очередь.

### Метод add()

Через инстанс компонента (`ref`) доступен объект `data`, предоставляющий метод `add(item: SnackbarValue)`. Параметр `item` определяет конфигурацию и поведение добавляемого уведомления.

```typescript
snackbar.value?.data.add({
  data: {
    label: 'Заголовок уведомления',
    description: 'Описание происходящего события'
  }
})
```

### Структура SnackbarValue

Объект конфигурации включает следующие свойства:

- **`data`** (`SnackbarItemProps`) — параметры для отображаемого компонента (по умолчанию `SnackbarItem`). Здесь передаются заголовок (`label`), описание (`description`), иконка (`icon`), кнопки действий и прочие визуальные данные.
- **`component`** (`any`) — позволяет передать пользовательский Vue-компонент для рендеринга вместо стандартного `SnackbarItem`.
- **`highPriority`** (`boolean`) — при значении `true` уведомление отображается в отдельном приоритетном контейнере (обычно закрепленном сверху или в начале списка).
- **`delay`** (`number`) — индивидуальное время жизни уведомления в миллисекундах. Если не указано, используется глобальное значение `delay` из настроек `Snackbar`.
- **`value`** (`string`) — уникальный ключ уведомления. Если не передан, генерируется автоматически. Необходим для ручного удаления сообщения через метод `remove(value)`.

### Использование кастомных компонентов

Для вывода уведомлений с нестандартной версткой можно передать свой компонент через свойство `component`. При этом все свойства, указанные в объекте `data`, будут переданы в ваш компонент как `props`.

```typescript
import MyAlert from './MyAlert.vue'

snackbar.value?.data.add({
  component: MyAlert,
  data: {
    title: 'Внимание!',
    message: 'Это сообщение в кастомном стиле'
  }
})
```

Свойство `highPriority` рекомендуется использовать для критических системных уведомлений или ошибок авторизации, чтобы они гарантированно привлекали внимание пользователя и отображались отдельно от общего потока сообщений.
