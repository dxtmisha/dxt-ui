- `back(): void` — Transition to the previous slide.
- `next(): void` — Transition to the next slide.
- `to(selected: string | number): void` — Transition to the specified slide.
- `top(selected: string | number): void` — Transition to the slide with animation from the top.
- `right(selected: string | number): void` — Transition to the slide with animation from the right.
- `bottom(selected: string | number): void` — Transition to the slide with animation from the bottom.
- `left(selected: string | number): void` — Transition to the slide with animation from the left.
- `down(selected: string | number): void` — Transition to the slide with animation downwards.
- `up(selected: string | number): void` — Transition to the slide with animation upwards.
