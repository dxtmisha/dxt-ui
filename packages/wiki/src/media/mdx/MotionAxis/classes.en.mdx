## CSS Classes for Behavior Control

- `*--axis--x` — animation along the X axis
- `*--axis--y` — animation along the Y axis
- `*--axis--z` — animation along the Z axis
- `*--direction--next` — forward animation
- `*--direction--back` — backward animation
- `*--start` — animation start state (when `animationHeight` is enabled)
- `*--move` — moving state (when `animationHeight` is enabled)

### Slide Classes

- `*__slide` — base slide class
- `*__slide--previous` — previously active slide
- `*__slide--preparation` — slide in preparation state for animation
- `*__slide--active` — currently active slide

Where `*` is the component class name (e.g., `d1-motionAxis`, `m3-motionAxis`).
