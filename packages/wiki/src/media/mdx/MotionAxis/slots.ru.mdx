### `[name]`

Слот для содержимого слайда. Имя слота соответствует значению, передаваемому в свойство `selected`.

**Параметры:**
- `props: any` — параметры слота (в данный момент не используются)

```html
<MotionAxis selected="slide1">
  <template #slide1>
    <div>Контент первого слайда</div>
  </template>
  <template #slide2>
    <div>Контент второго слайда</div>
  </template>
</MotionAxis>
```
