## CSS классы для управления поведением

- `*--axis--x` — анимация по оси X
- `*--axis--y` — анимация по оси Y
- `*--axis--z` — анимация по оси Z
- `*--direction--next` — анимация вперед
- `*--direction--back` — анимация назад
- `*--start` — состояние начала анимации (при включенном `animationHeight`)
- `*--move` — состояние движения (при включенном `animationHeight`)

### Классы слайдов

- `*__slide` — базовый класс слайда
- `*__slide--previous` — предыдущий активный слайд
- `*__slide--preparation` — слайд в состоянии подготовки к анимации
- `*__slide--active` — текущий активный слайд

Где `*` — название класса компонента (например, `d1-motionAxis`, `m3-motionAxis`).
