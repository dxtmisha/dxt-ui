## Управление скроллом при анимации

Компонент автоматически управляет видимостью скролла во время анимации перехода, чтобы предотвратить "скачки" контента.

### Использование

Для активации этой функции необходимо добавить класс `*__scroll` родительскому контейнеру, который имеет прокрутку.

1.  Оберните компонент в контейнер с прокруткой.
2.  Добавьте этому контейнеру класс `*__scroll` (где `*` — имя класса компонента, например `d1-tabs__scroll`).

```html
<div class="d1-tabs__scroll">
  <Tabs ... />
</div>
```

Во время анимации компонент автоматически добавит класс `*__scroll--hidden` (который применяет `overflow: hidden`) к этому контейнеру, а после завершения анимации удалит его.
