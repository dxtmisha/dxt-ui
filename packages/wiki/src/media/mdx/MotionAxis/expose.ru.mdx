### `back`

Переход к предыдущему слайду.

**Тип:** `() => void`

### `next`

Переход к следующему слайду.

**Тип:** `() => void`

### `to`

Переход к указанному слайду.

**Параметры:**
- `selected: string | number` — идентификатор слайда

**Тип:** `(selected: string | number) => void`
