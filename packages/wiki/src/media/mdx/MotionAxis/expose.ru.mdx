- `back(): void` — Переход к предыдущему слайду.
- `next(): void` — Переход к следующему слайду.
- `to(selected: string | number): void` — Переход к указанному слайду.
- `top(selected: string | number): void` — Переход к слайду с анимацией сверху.
- `right(selected: string | number): void` — Переход к слайду с анимацией справа.
- `bottom(selected: string | number): void` — Переход к слайду с анимацией снизу.
- `left(selected: string | number): void` — Переход к слайду с анимацией слева.
- `down(selected: string | number): void` — Переход к слайду с анимацией вниз.
- `up(selected: string | number): void` — Переход к слайду с анимацией вверх.
