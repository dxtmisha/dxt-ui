- `[column: string](props: TableRecordItemSlotProps): any` — Динамический слот для кастомного рендеринга ячеек столбца с указанным именем (применяется как для ячеек данных, так и для ячеек заголовка шапки).
- `body(props: { columns: string[] }): any` — Слот для пользовательского содержимого тела таблицы.

> **TableRecordItemSlotProps:**
> - `item: any` — Объект данных строки для обычных ячеек или объект конфигурации заголовка для ячеек шапки.
> - `value: any` — Значение ячейки (или значение/объект заголовка).
> - `classes: { label: string, description: string }` — Вспомогательные имена классов элементов ячейки.
> - `key: string` — Уникальный ключ рендеринга.
> - `index: string` — Индекс колонки.
> - `slotsName: string` — Имя слота.
