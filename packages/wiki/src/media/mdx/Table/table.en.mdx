Table component orchestrating grid-like column indexes and list row records for tabular data visualization.

The `Table` component handles rendering structured datasets. It integrates column configurations (`columns`), header setups (`header`), dataset rows (`list`), sticky header positioning (`headerTop`), sticky left columns (`stickyLeft`), and cell formatting to deliver high-performance interactive tables.

**Key Features:**

- Structured representation of datasets in tabular form
- Column-based data extraction via the `columns` array
- Multi-row hierarchical header support via the `header` prop
- Sticky headers (`headerTop`) and sticky left columns (`stickyLeft`) for large datasets
- Column-level cell alignment, row selection, and attribute binding (`tableItemColumnAttrs`)
- Custom slot support per column index for custom cell rendering

    code={`
<script setup>
const columns = ['id', 'name', 'role', 'status']
const header = { id: 'ID', name: 'Name', role: 'Role', status: 'Status' }
