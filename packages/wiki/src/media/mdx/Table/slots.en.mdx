- `[column: string](props: TableRecordItemSlotProps): any` — Dynamic slot for custom cell rendering of the specified column name (applies to both data cells and header cells).
- `body(props: { columns: string[] }): any` — Slot for custom table body content.

> **TableRecordItemSlotProps:**
> - `item: any` — Row data object for data cells, or header configuration object for header cells.
> - `value: any` — Current cell value (or header value/object).
> - `classes: { label: string, description: string }` — Auxiliary cell element class names.
> - `key: string` — Unique rendering key.
> - `index: string` — Column index.
> - `slotsName: string` — Slot name.
