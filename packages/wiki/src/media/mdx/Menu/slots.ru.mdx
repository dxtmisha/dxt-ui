- `title(props: MenuControlItem): any` — Слот для размещения заголовка окна меню.
- `footer(props: MenuControlItem): any` — Слот для размещения содержимого в нижней части окна меню.
- `contextTop(props: MenuControlItem): any` — Слот для размещения контекстной области над списком меню.
- `contextBottom(props: MenuControlItem): any` — Слот для размещения контекстной области под списком меню.

> **MenuControlItem:**
> - `loading: Ref<boolean>` — состояние загрузки данных меню.
> - `open: Ref<boolean>` — состояние открытия меню.
> - `isSelected: ComputedRef<boolean>` — наличие выбранных элементов.
> - `selectedList: Ref<any[]>` — список выбранных элементов.
> - `selectedNames: Ref<string[]>` — метки выбранных элементов.
> - `selectedValues: Ref<any[]>` — значения выбранных элементов.
> - `onclick: (event) => Promise<void>` — обработчик клика для открытия/закрытия меню.
> - `oncontextmenu: (event) => Promise<void>` — обработчик контекстного меню.
