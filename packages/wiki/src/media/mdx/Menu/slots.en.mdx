- `title(props: MenuControlItem): any` — Slot for placing the menu window title.
- `footer(props: MenuControlItem): any` — Slot for placing content in the bottom part of the menu window.
- `contextTop(props: MenuControlItem): any` — Slot for placing contextual area above the menu list.
- `contextBottom(props: MenuControlItem): any` — Slot for placing contextual area below the menu list.

> **MenuControlItem:**
> - `loading: Ref<boolean>` — reactive state of menu data loading.
> - `open: Ref<boolean>` — reactive state of menu open/close.
> - `isSelected: ComputedRef<boolean>` — presence of selected items.
> - `selectedList: Ref<any[]>` — list of selected items.
> - `selectedNames: Ref<string[]>` — labels of selected items.
> - `selectedValues: Ref<any[]>` — values of selected items.
> - `onclick: (event) => Promise<void>` — click handler for opening/closing menu.
> - `oncontextmenu: (event) => Promise<void>` — context menu handler.
