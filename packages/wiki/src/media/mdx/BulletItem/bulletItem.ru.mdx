## Описание

Отдельный элемент внутри маркированного списка.

Компонент BulletItem представляет собой отдельную строку или пункт маркированного списка внутри компонента Bullet. Он поддерживает вывод описания в виде текста или числа, имеет кастомный слот для описания, а также различные настройки размеров для адаптации под разный текстовый контекст.

**Основные возможности:**

- Рендерит семантический тег `<li>` с настраиваемыми отступами и позиционированием
- Интегрирует кастомный визуальный маркер (символ списка "•") с помощью CSS-псевдоэлементов
- Гибкая настройка содержимого через свойство или слот `description`
- Поддержка необязательного идентификатора `descriptionId` для доступности и связывания
