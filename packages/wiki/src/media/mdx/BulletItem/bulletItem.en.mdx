## Description

An individual item within a bulleted list.

The BulletItem component represents a single line or list point inside a structured Bullet list. It supports receiving description text or number directly, custom description slots, and different size options to adapt to various textual contexts.

**Key Features:**

- Renders a semantic `<li>` tag with custom spacing and positioning
- Integrates a custom visual marker (a bullet point symbol "•") via CSS pseudo-elements
- Flexible content customization via the `description` property or slot
- Support for an optional `descriptionId` for accessibility and linking
