## Properties for text and number handling

The `label`, `labelMax`, and `formatting` properties are designed to control the display of textual and numerical content in components.

### Properties

- **label** — Text label or numeric value of the component
- **labelMax** — Maximum length of a label text before truncation
- **formatting** — Determines whether values should be formatted before display

### Functionality

These properties provide flexible content display management:

#### Label processing
- `label` can contain both textual and numeric values
- When a numeric value is passed to `label`, the `toNumberByMax` function is automatically applied
- Conversion occurs with consideration of maximum values and localization

#### Length limitation
- `labelMax` sets the maximum length of displayed text
- When the limit is exceeded, a text is truncated with an overflow indicator
- Ensures consistent display across different contexts

#### Value formatting
- `formatting` controls whether formatting should be applied to values
- When formatting is enabled, numeric values are converted according to user locale
- Supports automatic interface language detection
