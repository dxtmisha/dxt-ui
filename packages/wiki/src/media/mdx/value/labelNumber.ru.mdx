## Свойства для работы с текстом и числами

Свойства `label`, `labelMax` и `formatting` предназначены для управления отображением текстового и числового контента в компонентах.

### Свойства

- **label** — Текстовая метка или числовое значение компонента
- **labelMax** — Максимальная длина текста метки перед обрезкой
- **formatting** — Определяет необходимость форматирования значений перед отображением

### Функциональность

Эти свойства обеспечивают гибкое управление отображением контента:

#### Обработка меток
- `label` может содержать как текстовые, так и числовые значения
- При передаче числового значения в `label` автоматически применяется функция `toNumberByMax`
- Происходит преобразование с учетом максимальных значений и локализации

#### Ограничение длины
- `labelMax` устанавливает максимальную длину отображаемого текста
- При превышении лимита текст обрезается с добавлением индикатора переполнения
- Обеспечивает консистентность отображения в различных контекстах

#### Форматирование значений
- `formatting` контролирует, нужно ли применять форматирование к значениям
- При включенном форматировании числовые значения преобразуются согласно локали пользователя
- Поддерживается автоматическое определение языка интерфейса
