Компонент ячейки заголовка таблицы, представляющий тег `<th>`. Расширяет все возможности `TableItem` и включает встроенную поддержку отображения всплывающих подсказок для предоставления дополнительной информации или руководства прямо из ячейки заголовка.

**Ключевые особенности:**

- Расширяет свойства и функции `TableItem` (выравнивание, настройка тегов, липкость)
- Поддержка стандартных HTML-тегов (свойство `tag`, по умолчанию `th`)
- Встроенная поддержка всплывающих подсказок, запускаемая иконкой (`iconTooltip` или `help-circle` по умолчанию)
- Слот для размещения сложного HTML или пользовательского содержимого внутри всплывающей подсказки (слот `tooltip`)
- Настройка подсказки через свойства `tooltipLabel` и `tooltipDescription`

<Source
    code={`
<script setup>
</script>

<template>
  <table>
    <thead>
      <tr>
        <!-- Простая ячейка заголовка -->
        <TableHeaderItem value="Заголовок 1" />

        <!-- Заголовок со встроенной подсказкой -->
        <TableHeaderItem
          label="Заголовок с подсказкой"
          tooltipLabel="Пояснение"
          tooltipDescription="Это объясняет назначение данной колонки."
        />
      </tr>
    </thead>
  </table>
</template>
  `}
    language="html"
/>

> TableHeaderItem идеально подходит для заголовков, требующих пояснительного текста, помогая пользователям понять столбцы данных без загромождения основного вида таблицы.
