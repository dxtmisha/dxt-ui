### `barsBack`

Срабатывает при клике по кнопке «Назад» в панелях (Bars). Передаёт только полезную нагрузку о клике.

```ts
function onBarsBack (value: { type: string; value: any; detail?: Record<string, any> }) {
  // обработка клика по кнопке «Назад»
}
```

Где:
- `type` — тип клика/кнопки (значение будет `"back"`)
- `value` — произвольное значение кнопки
- `detail` — дополнительные данные (если есть)

Используйте событие для реализации возврата/закрытия текущего контекста. Кнопку можно скрыть свойством `barsBackHide`.
