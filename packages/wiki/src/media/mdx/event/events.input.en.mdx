### `input`

Event fires when the value in the input field changes, passing the original DOM event and validation data.

**Parameters:**
- `event: InputEvent` ‚Äî original DOM input event
- `value: FieldValidationItem` ‚Äî object with validation data

**FieldValidationItem structure:**
- `group: string | undefined` ‚Äî mask group key
- `input: HTMLInputElement | HTMLTextAreaElement | undefined` ‚Äî related input element
- `pattern: FieldPatternItemOrFunction | undefined` ‚Äî validation pattern used
- `type: string | undefined` ‚Äî custom type
- `status: boolean | undefined` ‚Äî validation success status
- `required: boolean | undefined` ‚Äî field required flag
- `isFull: boolean | undefined` ‚Äî value completion flag
- `validationMessage: string | undefined` ‚Äî error message text
- `validity: ValidityState | undefined` ‚Äî native validity state
- `validityMessage: string | undefined` ‚Äî custom validity message
- `value: any` ‚Äî original value
- `valueInput: any | undefined` ‚Äî entered value
- `detail: Record<string, any> | undefined` ‚Äî additional data

<Source
    code={`
<script setup>
const handleInput = (event, value) => {
  console.log('DOM event:', event)
  console.log('Value:', value.value)
  console.log('Validation:', value.status)
  console.log('Complete:', value.isFull)

  if (!value.status) {
    console.log('Error:', value.validationMessage)
  }
}
</script>

<template>

  `}

### `inputLite`

Event fires when the value in the input field changes, passing only validation data without the original DOM event (lightweight version).

**Parameters:**
- `value: FieldValidationItem` ‚Äî object with validation data

### `change`

Event fires when input is completed and focus is lost, passing the original DOM event and validation data.

**Parameters:**
- `event: InputEvent` ‚Äî original DOM input event
- `value: FieldValidationItem` ‚Äî object with validation data

**FieldValidationItem structure:** same as for `input` event

<Source
    code={`
<script setup>
const handleChange = (event, value) => {
  console.log('DOM event:', event)
  console.log('Final value:', value.value)

  if (value.status && value.isFull) {
    // Submit data to server
    submitForm(value.value)
  }
}
</script>

<template>
  <Mask
    mask="+1 (***) ***-****"
    @change="handleChange"
  />
</template>
  `}
    language="html"
/>
