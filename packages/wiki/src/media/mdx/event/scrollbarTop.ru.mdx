### `scrollbarTop`

Событие, которое срабатывает при изменении состояния достижения верхней границы скролла.

```ts
function onScrollbarTop (isTop: boolean) {
    // обработка события достижения/покидания верхней границы
}
```

Параметры:
- `isTop` - `boolean` - true, если скролл находится в верхней позиции, false — если покинул верхнюю позицию
