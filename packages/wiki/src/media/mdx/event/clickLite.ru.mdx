### `clickLite`

Облегченное событие клика компонента, передающее только данные без оригинального DOM события.

```ts
function onClickLite (value: EventClickValue) {
    // обработка облегченного события клика
}

type EventClickValue = {
  /** Идентификатор места нажатия, задается через атрибут data-event-type у элемента */
  type: string
  /** Значение компонента */
  value: any
  /** Дополнительные данные события */
  detail: Record<string, any> | undefined
}
```
