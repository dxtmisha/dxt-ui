### `barsLite`

Срабатывает при клике по элементу панелей (Bars) в облегчённом режиме. Передаёт только полезную нагрузку о клике.

```ts
function onBarsLite (value: { type: string; value: any; detail?: Record<string, any> }) {
  // обработка клика по кнопке панели (lite)
}
```

Где:
- `type` — тип клика/кнопки (например, `"back"`)
- `value` — произвольное значение кнопки
- `detail` — дополнительные данные (если есть)
