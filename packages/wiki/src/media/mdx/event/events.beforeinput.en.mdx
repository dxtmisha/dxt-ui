### `beforeinput`

Event fires before the value in the input field changes, allowing you to prevent or modify data input.

**Parameters:**
- `event: InputEvent` — original DOM input event

```html
<script setup>
const handleBeforeInput = (event) => {
  console.log('Input type:', event.inputType)
  console.log('Data:', event.data)

  // Prevent special characters input
  if (event.data && /[^0-9]/.test(event.data)) {
    event.preventDefault()
    console.log('Input blocked: numbers only')
  }
}
</script>

<template>
  <Mask
    mask="+1 (***) ***-****"
    @beforeinput="handleBeforeInput"
  />
</template>
```

**Input type examples (inputType):**
- `insertText` — text input
- `insertFromPaste` — paste from clipboard
- `deleteContentBackward` — delete character backward (Backspace)
- `deleteContentForward` — delete character forward (Delete)
- `insertLineBreak` — insert line break

```html
<script setup>
import { ref } from 'vue'

const blockedAttempts = ref(0)

const onBeforeInput = (event) => {
  // Block paste from clipboard
  if (event.inputType === 'insertFromPaste') {
    event.preventDefault()
    blockedAttempts.value++
    console.log('Paste blocked')
  }
}
</script>

<template>
  <div>
    <Mask
      mask="+1 (***) ***-****"
      @beforeinput="onBeforeInput"
    />
    <p v-if="blockedAttempts > 0">
      Paste attempts blocked: {{ blockedAttempts }}
    </p>
  </div>
</template>
```
