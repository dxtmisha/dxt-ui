### `bars`

Срабатывает при клике по элементу панелей (Bars). Передаёт нативное событие мыши и полезную нагрузку о клике.

```ts
function onBars (event: MouseEvent, value: { type: string; value: any; detail?: Record<string, any> }) {
  // обработка клика по кнопке панели
}
```

Где:
- `type` — тип клика/кнопки (например, `"back"` для кнопки назад)
- `value` — произвольное значение кнопки
- `detail` — дополнительные данные (если есть)
