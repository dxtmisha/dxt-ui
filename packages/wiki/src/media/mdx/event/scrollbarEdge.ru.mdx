### `scrollbarEdge`

Событие, которое срабатывает при изменении состояния границ скролла (верхней или нижней).

```ts
function onScrollbarEdge (isTop: boolean, isBottom: boolean, edge: ScrollbarEdgeType) {
    // обработка изменения состояния границ скролла
}
```

Параметры:
- `isTop` — `boolean` — true, если скролл в верхней позиции
- `isBottom` — `boolean` — true, если скролл в нижней позиции
- `edge` — `ScrollbarEdgeType` — текущая активная граница скролла
