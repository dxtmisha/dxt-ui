### `barsLite`

Fires when a Bars item is clicked in lite mode. Passes only the structured click payload.

```ts
function onBarsLite (value: { type: string; value: any; detail?: Record<string, any> }) {
  // handle toolbar item click (lite)
}
```

Where:
- `type` — click/button type (e.g., "back")
- `value` — arbitrary button value
- `detail` — extra data if provided
