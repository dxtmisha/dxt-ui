Компонент многострочного ввода текста с поддержкой автоматического изменения размера, валидации и расширенной стилизации.

Textarea предоставляет удобный интерфейс для ввода больших объемов текста. Компонент автоматически подстраивает высоту под содержимое, поддерживает ограничение количества символов, отображение иконок, меток и вспомогательных сообщений. Интегрируется с системой валидации форм.

**Основные возможности:**

- Автоматическое изменение высоты (autosize)
- Поддержка валидации и отображение ошибок
- Встроенный счетчик символов
- Поддержка иконок, префиксов и суффиксов
- Плавающая или статическая метка (label)
- Состояния: активное, отключенное, только для чтения, загрузка
- Настройка темы и размеров

**Типичные сценарии использования:**

- Поля для комментариев и отзывов
- Формы обратной связи
- Редакторы описания или биографии
- Ввод адреса или заметок

```html
<script setup>
import { ref } from 'vue'

const comment = ref('')
</script>

<template>
  <Textarea
    v-model="comment"
    label="Ваш комментарий"
    placeholder="Введите текст..."
    autosize
    counter
    :maxlength="500"
  />
</template>
```
