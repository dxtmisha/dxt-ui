## Эффект волны при взаимодействии

Компонент `Ripple` создает анимированный эффект волны при клике по элементу, аналогичный эффекту Material Design. Это улучшает пользовательский опыт, предоставляя визуальную обратную связь при взаимодействии.

### Принцип работы

Компонент автоматически отслеживает события клика и создает анимированную волну, которая расширяется от точки касания. Эффект работает путем:

- Определения координат клика относительно элемента
- Создания временного элемента с анимацией расширения
- Автоматического удаления элемента после завершения анимации

### Интеграция

Компонент должен быть размещен внутри интерактивного элемента, к которому нужно добавить эффект волны. Родительский элемент должен иметь `position: relative` для корректного позиционирования эффекта.

### Свойства

- **disabled** — Отключает эффект волны при установке в `true`

### Рекомендации

- Используйте на кнопках, карточках и других интерактивных элементах
- Убедитесь, что родительский элемент имеет `position: relative`
- Не используйте на элементах, которые часто изменяют размер

### Пример использования

```html
<!-- Кнопка с эффектом волны -->
<button style="position: relative;">
  Нажми меня
  <Ripple />
</button>

<!-- Отключенный эффект -->
<button style="position: relative;">
  Без эффекта
  <Ripple :disabled="true" />
</button>

<!-- Карточка с эффектом -->
<div class="card" style="position: relative; cursor: pointer;">
  <h3>Интерактивная карточка</h3>
  <p>Содержимое карточки</p>
  <Ripple />
</div>
```
