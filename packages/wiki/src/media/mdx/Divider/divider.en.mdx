## Description

A simple horizontal line used to visually separate content, sections, or list items.

Divider helps to organize information and create a clear visual hierarchy in the interface. It is designed to be minimal and unobtrusive, providing just enough separation without cluttering the layout.

**Key Features:**

- Simple horizontal separation
- Customizable thickness and color via design tokens
- Minimalist design for any interface
- Preservation of vertical rhythm

**Typical Usage Scenarios:**

- Separating paragraphs of text
- Dividing sections in a list or menu
- Breaking up long forms into logical parts
- Visual boundaries between different content blocks

> Divider is an essential layout tool that provides clean and effective separation between interface elements, enhancing readability and information structure.
