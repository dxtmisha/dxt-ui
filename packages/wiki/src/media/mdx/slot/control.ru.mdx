### `control`

Слот для рендеринга элемента управления (триггера) компонента.

Этот слот позволяет:
- Разместить кнопку, иконку или любой триггер для взаимодействия
- Получать данные контрола через scoped‑slot (если компонент их предоставляет)
- Настроить логику открытия/закрытия и дополнительные действия
