### `description`

Слот для передачи содержимого описания компонента. Является альтернативой props `description` и позволяет передавать элементы вместо простого текста.

Этот слот позволяет:
- Использовать альтернативу свойству `description`
- Передавать элементы вместо простого текста
- Использовать HTML или компоненты в описании

Слот `description` работает как замена props `description` и позволяет передавать любые элементы для более гибкого задания описания.
