### `default`

Основной слот для размещения контента внутри компонента.

Этот слот позволяет:
- Размещать любой пользовательский контент внутри компонента
- Передавать текст, HTML-элементы или другие Vue-компоненты
- Настраивать внутреннее содержимое компонента

Слот `default` является основным способом добавления контента в компонент и поддерживает все стандартные возможности Vue слотов.
