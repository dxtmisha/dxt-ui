## Текстовое содержимое (label, length)

Управляет отображением текста и генерацией заполнителя. По умолчанию: `length = 3`.

> При наличии слота по умолчанию (`default`) свойства `label` и `length` игнорируются.

**Свойства:**
- `label` — статический текст или число для вывода вместо случайного текста
- `length` — ожидаемая длина текста (число) или диапазон длин (`[min, max]`)

### Поведение

- Свойства работают по принципу приоритета: слот `default` отменяет `label` и `length`
- Заданное свойство `label` выводится напрямую и отменяет генерацию через `length`
- При передаче числа в `length` (например, `:length="5"`) генерируется фиксированное количество слов
- При передаче массива в `length` (например, `:length="[4, 10]"`) длина текста определяется случайно в указанных пределах
- Диапазон длин позволяет имитировать естественную разницу длин текстовых строк при отображении списков
