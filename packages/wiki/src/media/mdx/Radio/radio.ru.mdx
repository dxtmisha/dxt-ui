Компонент для создания интерактивных радиокнопок, предназначенных для выбора одного значения из группы предложенных вариантов.

Radio управляет состоянием выбора элементов, поддерживает валидацию, интеграцию с формами, настраиваемые иконки и анимацию переходов. Компонент автоматически обрабатывает изменение состояний, управление фокусом и генерацию событий при взаимодействии пользователя. В отличие от чекбокса, радиокнопка обычно используется в группе, где возможен выбор только одного элемента.

**Основные возможности:**

- Выбор одного значения из группы (эксклюзивный выбор)
- Встроенная валидация с пользовательскими сообщениями об ошибках
- Поддержка меток (label) и описания (description)
- Настраиваемые иконки для разных состояний
- Интеграция с формами через name и value
- Состояния disabled, loading и skeleton
- Эффект ripple при взаимодействии
- Адаптивная раскладка и стилизация

**Типичные сценарии использования:**

- Выбор пола или семейного положения
- Настройка способа доставки или оплаты
- Переключение между режимами работы
- Опросы с единственным вариантом ответа
- Формы настроек, где параметры исключают друг друга
