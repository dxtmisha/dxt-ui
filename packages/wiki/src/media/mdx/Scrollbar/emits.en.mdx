### `top`

Event fires when the top scroll boundary state changes.

**Parameters:**
- `isTop: boolean` — `true` if scroll is at the top position

```html
<script setup>
const handleTop = (isTop) => {
  console.log('Top boundary:', isTop)
}
</script>

<template>
  <Scrollbar @top="handleTop" />
</template>
```

### `reachTop`

Event fires when the top scroll boundary is reached.

```html
<script setup>
const handleReachTop = () => {
  console.log('Reached top boundary')
}
</script>

<template>
  <Scrollbar @reach-top="handleReachTop" />
</template>
```

### `leaveTop`

Event fires when leaving the top scroll boundary.

```html
<script setup>
const handleLeaveTop = () => {
  console.log('Left top boundary')
}
</script>

<template>
  <Scrollbar @leave-top="handleLeaveTop" />
</template>
```

### `edge`

Event fires when any scroll boundary state changes.

**Parameters:**
- `isTop: boolean` — `true` if scroll is at the top position
- `isBottom: boolean` — `true` if scroll is at the bottom position
- `edge: 'top' | 'bottom'` — type of boundary that changed

```html
<script setup>
const handleEdge = (isTop, isBottom, edge) => {
  console.log('Boundary state:', { isTop, isBottom, edge })
}
</script>

<template>
  <Scrollbar @edge="handleEdge" />
</template>
```
