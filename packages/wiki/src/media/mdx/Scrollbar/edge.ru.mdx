### `edge`

Событие, которое срабатывает при изменении состояния границ скролла (верхней или нижней).

```ts
function onEdge (isTop: boolean, isBottom: boolean, edge: ScrollbarEdgeType) {
    // обработка события изменения границ скролла
}
```

Параметры:
- `isTop` - `boolean` - true, если скролл находится в верхней позиции
- `isBottom` - `boolean` - true, если скролл находится в нижней позиции
- `edge` - `ScrollbarEdgeType` - тип текущей границы скролла
