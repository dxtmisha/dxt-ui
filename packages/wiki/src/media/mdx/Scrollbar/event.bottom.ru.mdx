### `bottom`

Событие, которое срабатывает при изменении состояния достижения нижней границы скролла.

```ts
function onBottom (isBottom: boolean) {
    // обработка события достижения/покидания нижней границы
}
```

Параметры:
- `isBottom` - `boolean` - true, если скролл находится в нижней позиции, false - если покинул нижнюю позицию
