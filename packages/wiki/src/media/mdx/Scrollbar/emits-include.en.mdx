### `scrollbarTop`

Event fires when the top scroll boundary state changes (for use in parent components).

**Parameters:**
- `isTop: boolean` — `true` if scroll is at the top position

```html
<script setup>
const handleScrollbarTop = (isTop) => {
  console.log('Scrollbar - top boundary:', isTop)
}
</script>

<template>
  <Component @scrollbar-top="handleScrollbarTop" />
</template>
```

### `scrollbarReachTop`

Event fires when the top scroll boundary is reached (for use in parent components).

```html
<script setup>

  console.log('Scrollbar reached top boundary')
}
</script>

  <Component @scrollbar-reach-top="handleScrollbarReachTop" />
</template>
```

### `scrollbarEdge`

Event fires when any scroll boundary state changes (for use in parent components).

**Parameters:**
- `isTop: boolean` — `true` if scroll is at the top position
- `isBottom: boolean` — `true` if scroll is at the bottom position
- `edge: 'top' | 'bottom'` — type of boundary that changed

```html
<script setup>
const handleScrollbarEdge = (isTop, isBottom, edge) => {
  console.log('Scrollbar - boundary state:', { isTop, isBottom, edge })
}
</script>

<template>
  <Component @scrollbar-edge="handleScrollbarEdge" />
</template>
```
