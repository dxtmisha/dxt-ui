### `top`

Событие срабатывает при изменении состояния достижения верхней границы скролла.

**Параметры:**
- `isTop: boolean` — `true`, если скролл находится в верхней позиции

```html
<script setup>
const handleTop = (isTop) => {
  console.log('Верхняя граница:', isTop)
}
</script>

<template>
  <Scrollbar @top="handleTop" />
</template>
```

### `reachTop`

Событие срабатывает при достижении верхней границы скролла.

```html
<script setup>
const handleReachTop = () => {
  console.log('Достигнута верхняя граница')
}
</script>

<template>
  <Scrollbar @reach-top="handleReachTop" />
</template>
```

### `leaveTop`

Событие срабатывает при покидании верхней границы скролла.

```html
<script setup>

  console.log('Покинута верхняя граница')
```

### `edge`

Событие срабатывает при изменении состояния любой из границ скролла.

**Параметры:**
- `isTop: boolean` — `true`, если скролл находится в верхней позиции
- `isBottom: boolean` — `true`, если скролл находится в нижней позиции
- `edge: 'top' | 'bottom'` — тип границы, которая изменилась

```html
<script setup>
const handleEdge = (isTop, isBottom, edge) => {
  console.log('Состояние границ:', { isTop, isBottom, edge })
}
</script>

<template>
  <Scrollbar @edge="handleEdge" />
</template>
```
