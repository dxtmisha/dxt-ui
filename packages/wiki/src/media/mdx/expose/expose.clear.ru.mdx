### `clear`

Очищает значение компонента до значения по умолчанию и устанавливает фокус на поле ввода.

```html
<script setup>
const componentRef = ref()

const handleClear = () => {
  componentRef.value.clear()
}
</script>

<template>
  <Component ref="componentRef" />
  <button @click="handleClear">Очистить</button>
</template>
```
