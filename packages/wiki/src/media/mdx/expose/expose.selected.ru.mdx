- `isSelected: ComputedRef<boolean>` — Вычисляемое свойство, указывающее наличие выбранных элементов в компоненте.
- `selectedList: Ref<any[]>` — Реактивный список выбранных элементов компонента.
- `selectedNames: Ref<string[]>` — Реактивный список меток (имён) выбранных элементов.
- `selectedValues: Ref<any[]>` — Реактивный массив значений выбранных элементов.
