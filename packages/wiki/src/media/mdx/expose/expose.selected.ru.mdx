- `isSelected(): boolean` — Возвращает информацию о наличии выбранных элементов в компоненте.
- `getSelectedList(): ListList` — Возвращает список выбранных элементов компонента.
- `getSelectedNames(): ListNames` — Возвращает список меток (имён) выбранных элементов.
- `getSelectedValues(): any[]` — Возвращает массив значений выбранных элементов.
