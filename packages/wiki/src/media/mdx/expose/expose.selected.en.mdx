### `isSelected`

Computed property indicating whether there are selected items in the component.

**Type:** `ComputedRef<boolean>`

### `selectedList`

Reactive list of selected component items.

**Type:** `Ref<any[]>`

### `selectedNames`

Reactive list of labels (names) of selected items.

**Type:** `Ref<string[]>`

### `selectedValues`

Reactive array of values of selected items.

**Type:** `Ref<any[]>`
