- `isSelected: ComputedRef<boolean>` — Computed property indicating whether there are selected items in the component.
- `selectedList: Ref<any[]>` — Reactive list of selected component items.
- `selectedNames: Ref<string[]>` — Reactive list of labels (names) of selected items.
- `selectedValues: Ref<any[]>` — Reactive array of values of selected items.
