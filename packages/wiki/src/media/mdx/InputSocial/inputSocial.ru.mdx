Компонент ввода для социальных сетей с автоматическим форматированием и иконками.

InputSocial — это специализированное поле ввода, предназначенное для профилей социальных сетей. Оно автоматически настраивает маску, префикс, суффикс и метку на основе переданного свойства `socialType`.

**Основные возможности:**

- Автоматическое форматирование и маскирование на основе выбранного типа социальной сети
- Поддержка различных социальных сетей (например, Telegram, WhatsApp, Instagram и др.)
- Настраиваемые иконки социальных сетей через свойство `socialIcons`
- Встроенная обработка префиксов и суффиксов, специфичных для профиля социальной сети
- Поддержка двусторонней привязки данных (`v-model`)
- Интеграция с `Field` для единообразной стилизации и статусов валидации

**Типичные сценарии использования:**

- Ссылки на социальные сети в профилях пользователей
- Поля контактной информации в формах регистрации
