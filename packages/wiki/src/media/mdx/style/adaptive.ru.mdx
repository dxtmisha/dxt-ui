## Адаптивная раскладка и контейнерные ограничения

Свойства `adaptive` и `container` управляют поведением компонентов при различных размерах экрана и контейнерных ограничениях. Эти свойства позволяют создавать гибкие адаптивные интерфейсы, которые корректно отображаются на всех устройствах.

### Свойства

- **adaptive** — Управляет адаптивным поведением отображения для различных размеров экрана и медиа-запросов. Позволяет компоненту автоматически подстраиваться под разные условия отображения.
- **container** — Управляет поведением контейнерной раскладки с адаптивными ограничениями ширины и контекстом сдерживания. Включает специальный режим контейнера с ограничениями по ширине.

### Описание

- Свойство `adaptive` обеспечивает автоматическую адаптацию под различные размеры экрана с поддержкой медиа-запросов для точного контроля отображения.
- `container` создает контейнерную раскладку с автоматическими ограничениями ширины, адаптивным поведением и центрированием содержимого.
- Свойства могут использоваться совместно для создания сложных адаптивных раскладок, где `container` обеспечивает структурную основу, а `adaptive` добавляет гибкое поведение.

### Рекомендации

- Используйте `adaptive` для компонентов, которые должны изменять свое поведение в зависимости от размера экрана.
- Применяйте `container` для создания основных блоков с ограниченной шириной и центрированием.
- Комбинируйте оба свойства для достижения максимальной гибкости в адаптивной верстке.
