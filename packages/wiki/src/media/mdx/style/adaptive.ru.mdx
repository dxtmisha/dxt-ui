## Адаптивное поведение

Свойства `adaptive` и `container` предназначены для управления поведением компонентов при различных размерах экрана и контейнерных ограничениях.

**Свойства:**

- `adaptive` — управляет адаптивным поведением для различных размеров экрана и медиа-запросов
- `container` — управляет контейнерной раскладкой с адаптивными ограничениями ширины

Свойства работают совместно: `adaptive` обеспечивает автоматическую адаптацию компонента под различные размеры экрана с поддержкой медиа-запросов для точного контроля отображения. `container` создает контейнерную раскладку с автоматическими ограничениями ширины, адаптивным поведением и центрированием содержимого. Оба свойства могут использоваться вместе для создания сложных адаптивных раскладок, где `container` обеспечивает структурную основу, а `adaptive` добавляет гибкое поведение для разных устройств.
