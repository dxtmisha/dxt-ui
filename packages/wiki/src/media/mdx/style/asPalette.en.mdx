## Color Palette

Property `asPalette` enables color scheme inheritance from the parent element while maintaining base saturation.

**Usage:**

Allows automatic color adaptation to placement context. The element adopts the parent's color palette without losing its saturation characteristics.

**Values:**

- `false` (default) — uses its own color scheme
- `true` — inherits palette from parent element

```html
<!-- Inherit palette from parent -->
<Component asPalette />
```
