## Managing Component Palette

The `asPalette` property applies the parent element's palette color to the component while preserving its base saturation.

### Properties

- **asPalette** â€” Boolean property. When `true`, the component inherits the color defined by its parent's palette class.

### Description

- When `asPalette=false`, the component uses its default color.
- When `asPalette=true`, the component is colored according to the parent's palette class, maintaining its original saturation.

### Recommendations

- Use this property to automatically adapt components to the application's theme.
- Ensure the parent element has a valid palette class.
