## Управление палитрой компонента

Свойство `asPalette` отвечает за изменение цвета компонента в соответствии с классом палитры его родителя, сохраняя базовую насыщенность цвета.

### Свойства

- **asPalette** — Булево свойство. Если `true`, компонент наследует цвет родительского элемента с палитрой.

### Описание

- При `asPalette=false` используется стандартный цвет компонента.
- При `asPalette=true` компонент окрашивается в цвет, заданный классом палитры родителя, при этом сохраняется исходная насыщенность цвета.

### Рекомендации

- Применяйте для автоматической адаптации компонентов к теме приложения.
- Убедитесь, что у родительского элемента задан корректный класс палитры.
