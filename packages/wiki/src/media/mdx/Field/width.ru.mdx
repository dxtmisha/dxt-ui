## Ширина (width)

Свойство для управления горизонтальным размером Field.

### Свойство
- **width** (`string`) — Передавайте ЛЮБОЕ валидное CSS‑значение: `"320px"`, `"48rem"`, `"50%"`, `"min(100%,320px)"`, `"clamp(240px,40vw,480px)"`, либо дизайн‑токен (например `"sm"`, если поддержан системой). Значение `"custom"` — внутренняя тех. метка (обычно не использовать).

### Поведение
- Не задано: ширина «резиновая» (зависит от контейнера).
- С `block`: всегда 100%.
- Любое конкретное значение: применяется напрямую без доп. обёрток.

### Примеры
```vue
<DesignField label="auto" />
<DesignField width="320px" label="320px" />
<DesignField width="50%" label="50%" />
<DesignField width="min(100%,320px)" label="min(...)" />
```

### Заметки
- Предпочтительнее управлять адаптивностью через родительский layout (flex / grid) и использовать width точечно.
- Сложные выражения (calc / min / clamp) поддерживаются, если валидны для CSS.
- `block` может переопределить индивидуально заданную ширину при конфликте приоритетов стилей.
