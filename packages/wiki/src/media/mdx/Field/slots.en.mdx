- `default(props: FieldControl): any` — Default slot for the main input element.
- `leading(props: FieldControl): any` — Slot for displaying content before the input area.
- `trailing(props: FieldControl): any` — Slot for displaying content after the input area.

> **FieldControl:**
> - `id: string` — unique identifier for binding label and input.
> - `className: string` — CSS classes for the visible input area.
> - `classHidden: string` — CSS classes for the hidden native input element.
> - `classForFocus: string` — CSS classes applied when the field is focused.
> - `binds: Record<string, any>` — properties bound to the input element.
