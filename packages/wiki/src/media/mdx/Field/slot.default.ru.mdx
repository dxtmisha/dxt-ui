### `default`

Основной слот ввода Field — сюда помещается реальный интерактивный контрол.

- Props: `{ id, className, classHidden }`
- Всегда привязывайте `:id="id"` и `:class="className"` (доступность + стили)
- `classHidden` можно использовать для скрытого зеркального / измерительного input
- Избегайте лишних обёрток, чтобы не ломать размеры и выравнивание
- Подходит для `<input>`, `<textarea>`, маскированных или кастомных компонентов
