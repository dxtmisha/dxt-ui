- `default(props: FieldControl): any` — Слот по умолчанию для основного элемента ввода (input, select, textarea).
- `leading(props: FieldControl): any` — Слот для отображения контента перед областью ввода.
- `trailing(props: FieldControl): any` — Слот для отображения контента после области ввода.

> **FieldControl:**
> - `id: string` — уникальный идентификатор элемента для связывания label и input.
> - `className: string` — CSS-классы для видимой зоны ввода.
> - `classHidden: string` — CSS-классы для скрытого нативного элемента ввода.
> - `classForFocus: string` — CSS-классы, применяемые при фокусировке на поле.
> - `binds: Record<string, any>` — свойства, привязанные к элементу ввода.
