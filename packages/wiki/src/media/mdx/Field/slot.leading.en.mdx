### `leading`

Slot for placing custom content at the start of the interactive field area (before the main input content flow).

- Positioned at the logical beginning (left in LTR, right in RTL)
- Suitable for small icons, state indicators, lightweight buttons or badges
- Receives the same control props object (`{ id, className, classHidden }`) as the default slot
- Should remain compact; avoid breaking vertical rhythm or padding
