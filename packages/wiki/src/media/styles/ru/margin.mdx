# Margin

Набор SCSS миксинов для управления внешними отступами с поддержкой CSS Custom Properties.

## Миксины

### `margin`

Устанавливает отступы со всех сторон.

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss
.element {
  @include margin(16px);
}
```

### `marginX`

Устанавливает горизонтальные отступы (слева и справа).

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss
.element {
  @include marginX(16px);
}
```

### `marginY`

Устанавливает вертикальные отступы (сверху и снизу).

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss
.element {
  @include marginY(16px);
}
```

### `marginLeft`

Устанавливает левый отступ с поддержкой RTL через margin-inline-start.

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss

}
```

### `marginRight`

Устанавливает правый отступ с поддержкой RTL через margin-inline-end.

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss

  @include marginRight(16px);
}
```

### `marginTop`

Устанавливает верхний отступ.

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

