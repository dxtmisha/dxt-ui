# Padding

Набор SCSS миксинов для управления внутренними отступами с поддержкой CSS Custom Properties.

## Миксины

### `padding`

Устанавливает отступы со всех сторон.

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss
.element {
  @include padding(16px);
}
```

### `paddingX`

Устанавливает горизонтальные отступы (слева и справа).

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss
.element {
  @include paddingX(16px);
}
```

### `paddingY`

Устанавливает вертикальные отступы (сверху и снизу).

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`

```scss

  @include paddingY(16px);
}
```

### `paddingLeft`

Устанавливает левый отступ с поддержкой RTL через padding-inline-start.

**Параметры:**
- `$value` — значение отступа

**Возвращает:** `void`


### `paddingRight`

- `$value` — значение отступа
