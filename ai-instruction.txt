### Обзор проекта dxt-ui для ИИ

**Суть проекта:** Модульная UI-библиотека и конструктор дизайн-систем на Vue 3. Проект ориентирован на автоматизацию (Figma, AI-генерация документации) и высокую типизацию.

#### Архитектура и Технологии:
- **Monorepo**: Управляется через npm workspaces. Основные части находятся в `packages/*`.
- **Стек**: Vue 3 (`<script setup lang="ts">`), TypeScript, Vite, SCSS.
- **Ключевые пакеты**:
    - `ui`, `d1`: Библиотеки компонентов.
    - `functional`, `functional-basic`: Ядро логики и API.
    - `figma*`: Пакеты для синхронизации с Figma (стили, код, переводы).
    - `scripts`: Скрипты автоматизации и генерации промптов для ИИ.
    - `wiki`: Документация в формате MDX.

#### Правила разработки для ИИ:
1.  **Чистота и Атомарность**: Компоненты должны быть минимальными. Вся бизнес-логика выносится в **Composables**.
2.  **Типизация**: Строгий TypeScript, никаких `any`. Обязательное использование интерфейсов для Props и Emits.
3.  **Документация**: Используй TSDoc на языке `[wikiLanguage]`. Комментарии должны объяснять "зачем", а не "что".
4.  **Стили**: Используй SCSS. Изменение базовых стилей библиотеки запрещено, если не указано иное.
5.  **Зависимости**: Используй только те пакеты, которые есть в `package.json`. Никаких галлюцинаций.

Генерируй код так, чтобы его можно было вставить и запустить сразу (**Copy-Paste Ready**).
